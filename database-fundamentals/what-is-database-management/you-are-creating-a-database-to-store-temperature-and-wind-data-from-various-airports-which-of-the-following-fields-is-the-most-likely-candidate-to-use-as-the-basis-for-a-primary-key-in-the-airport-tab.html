WHAT IS DATABASE MANAGEMENT

You are creating a database to store temperature and wind data from various airports. Which of the following fields is the most likely candidate to use as the basis for a primary key in the Airport table?

(A) Address

(B) City

(C) ** Airport code

(D) Day

EXPLANATIONS BELOW

Concept note-1: -You are creating a database to store temperature and wind data from various airports. Which of the following fields is the most likely candidate to use as the basis for a primary key in the Airport table? information policy.

Concept note-2: -A relational database management system (RDBMS) is used to store, manage, query and retrieve data in a relational database. Typically, the RDBMS gives users the ability to control read/write access, specify report generation and analyze use.

Concept note-3: -Which of the following database types would be best suited for storing multimedia? NoSQL DBMS.

Concept note-4: -The data definition capability enables a user to be able to specify the structure of the content of the database. This capability is used to create database tables and to define the characteristics of the fields in each table.

Concept note-5: -A database management system (DBMS) is system software for creating and managing databases. A DBMS makes it possible for end users to create, protect, read, update and delete data in a database.
