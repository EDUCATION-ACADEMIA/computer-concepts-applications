What is Cloud database?

(A) ** Created and maintained using cloud data services that provide defined performance measures for the database

(B) Stores data in a format optimized for decision support

(C) Created and maintained using different data services that provide defined performance measures for the database

(D) Stores historical data and business metrics used exclusively for tactical or strategic decision making

EXPLANATIONS BELOW

Concept note-1: -A cloud database is a database built to run in a public or hybrid cloud environment to help organize, store, and manage data within an organization. Cloud databases can be offered as a managed database-as-a-service (DBaaS) or deployed on a cloud-based virtual machine (VM) and self-managed by an in-house IT team.

Concept note-2: -There are two cloud database environment models – the traditional cloud model and Database-as-a-Service (DBaaS). In the traditional cloud model, the content database runs on the particular company’s infrastructure and any oversight falls into the hands of the IT manager and team.

Concept note-3: -A cloud database is a database service built and accessed through a cloud platform. It serves many of the same functions as a traditional database with the added flexibility of cloud computing. Users install software on a cloud infrastructure to implement the database.
