DATABASE FUNDAMENTALS

WHAT IS DATABASE MANAGEMENT

[SOURCES]
Removes empty space from a database.

(A) ** compact

(B) debug

(C) despace

(D) repair

EXPLANATIONS BELOW

Concept note-1: -The TRIM() function removes the space character OR other specified characters from the start or end of a string. By default, the TRIM() function removes leading and trailing spaces from a string.

Concept note-2: -To perform this operation per database file, use sp clean db file free space (Transact-SQL). The length of time required to run sp clean db free space depends on the size of the file, the available free space, and the capacity of the disk.

Concept note-3: -However, you need to dismount the database before using the EseUtil command. To get rid of the White Space, you can move the mailboxes to a new database and then delete the database. Avoid defragmentation to reclaim the white space as it may cause database corruption and other issues.

Concept note-4: -In PostgreSQL, the TRIM() function is used to remove the longest string consisting of spaces or any specified character from a string. By default, the TRIM() function removes all spaces (’ ‘) if not specified explicitly.
