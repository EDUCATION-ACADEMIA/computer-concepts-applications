USING THE UPPER AND LOWER FUNCTIONS IN EXCEL

[SOURCES]
Fungsi OR

(A) Returns FALSE if every argument is TRUE

(B) ** Returns TRUE if every argument is TRUE

(C) Returns TRUE if any argument is FALSE

(D) Returns TRUE

EXPLANATIONS BELOW

Concept note-1: -Technical Details. The AND function returns TRUE if all its arguments evaluate to TRUE, and returns FALSE if one or more arguments evaluate to FALSE. One common use for the AND function is to expand the usefulness of other functions that perform logical tests.

Concept note-2: -The OR function in Excel checks if ANY argument in it evaluates to TRUE. If anything evaluates to TRUE, then the function will return TRUE; to return FALSE, every argument in the function must evaluate to FALSE. Even if only one argument in this function evaluates to TRUE, the function will return TRUE.

Concept note-3: -The AND function in Excel is a logical function that checks whether all arguments fed into the function are TRUE and returns “TRUE” if all arguments are TRUE.

Concept note-4: -Use the IF function, one of the logical functions, to return one value if a condition is true and another value if it’s false. For example: =IF(A2>B2, "Over Budget", "OK") =IF(A2=B2, B4-A4, "")

Concept note-5: -Using the OR function in Excel The difference is that the OR function returns TRUE if at least one if the arguments evaluates to TRUE, and returns FALSE if all arguments are FALSE.
