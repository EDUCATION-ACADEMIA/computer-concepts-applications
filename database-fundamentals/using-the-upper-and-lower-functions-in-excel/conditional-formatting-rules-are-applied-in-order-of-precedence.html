Conditional formatting rules are applied in order of precedence.

(A) ** True

(B) False

(C) X

(D) X

EXPLANATIONS BELOW

Concept note-1: -While applying the formatting, excel checks the precedence order and then it applies the rules accordingly. To check the precedence order, go to conditional formatting and select the option of Manage Rules.

Concept note-2: -Each cell has several conditional formatting rules. The order is critical since the format stops at the first rule thatâ€™s true. For example is less than 0, red, if cell A <>B, bold, etc.

Concept note-3: -we can apply Font, border, and pattern formats that meet the specified conditions. we can delete any condition from the Conditional Formatting dialog box if it is not required. The correct answer is option B i.e. You can set conditions to look for Bold and apply Italics on them.

Concept note-4: -Solution(By Examveda Team) In Conditional Formatting we cannot set condition to look for Bold and apply Italics on them.
