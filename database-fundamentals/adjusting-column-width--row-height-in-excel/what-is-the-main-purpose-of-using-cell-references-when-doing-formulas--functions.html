What is the main purpose of using cell references when doing formulas & functions?

(A) ** So you don’t have to redo the math if something changes

(B) It feels fancier therefore you’re smarter

(C) It’s the ONLY way to get the right math answer

(D) Confuse students in Info Tech class

EXPLANATIONS BELOW

Concept note-1: -A cell reference refers to a cell or a range of cells on a worksheet and can be used in a formula so that Microsoft Office Excel can find the values or data that you want that formula to calculate.

Concept note-2: -Users insert cell references into formulas, allowing the function to use a cell’s value as part of its calculation. For instance, if you wanted to add the values of A1 and B1, you could use A1 and B1 as cell references to create the following formula: =SUM(A1+B1).

Concept note-3: -When you use cell references in a formula, Excel calculates the answer using the numbers in the referenced cells. When you change the value in a cell, the formula calculates the new result automatically.

Concept note-4: -It tells Microsoft Excel where to look for the value you want to use in the formula. But if you want to move or copy the formula across the worksheet, it’s very important that you choose the right reference type for the formula to get copied correctly to other cells.
