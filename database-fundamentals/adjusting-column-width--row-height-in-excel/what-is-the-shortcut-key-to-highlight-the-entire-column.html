ADJUSTING COLUMN WIDTH ROW HEIGHT IN EXCEL

[SOURCES]
What is the shortcut key to highlight the entire column?

(A) Ctrl+C

(B) Ctrl+Enter

(C) Ctrl+Page Up

(D) ** Ctrl+Space Bar

EXPLANATIONS BELOW

Concept note-1: -Luckily, there are a few shortcuts that can help with this. To select an entire row, click the row number or press Shift+spacebar on your keyboard. To select an entire column, click the column letter or press Ctrl+spacebar.

Concept note-2: -Close a task pane Press Ctrl+Spacebar. Use the arrow keys to select Close, and then press Enter.

Concept note-3: -Ctrl+space selects an entire column, and Shift+space selects an entire row. To choose the row or column, arrow over to it before pressing Ctrl (or Shift) + space.

Concept note-4: -If you want to select an entire column of data, you can use the Ctrl+Space shortcut. This shortcut will select the entire column of the active cell. If you want to select the entire worksheet, including headers and footers, you can use the Ctrl+Shift+Space shortcut.

Concept note-5: -CTRL+Spacebar is the shortcut key used to select column. In Excel, hit Ctrl plus the Space Bar to select the entire column of your active or selected cells.
