ADJUSTING COLUMN WIDTH ROW HEIGHT IN EXCEL

[SOURCES]
Write cell references for cell formed by intersection of row 18 and column Z.

(A) 18Z

(B) ** Z18

(C) Y18

(D) None

EXPLANATIONS BELOW

Concept note-1: -3. Write cell reference for the following: (a) Cell formed by intersection of row 18, and column Z. Answer : Z18 (b) The right most cell in row 32 in a worksheet. Answer : AMJ32 (c) If you select on entire worksheet, which range of cells gets selected.

Concept note-2: -(b) Absolute reference formed by 45 and column Z is referenced as R45. (c) Relative Reference formed by row 19 and column AB is referenced as R19.

Concept note-3: -To refer to a cell, type the column letter followed by the row number. For example, D50 refers to the cell at the intersection of column D and row 50.

Concept note-4: -Expert-Verified Answer (d) Reference (Fixed Column and relative row) formed by row 134 and column BD is $BD134.
