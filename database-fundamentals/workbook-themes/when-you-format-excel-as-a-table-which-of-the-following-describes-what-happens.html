When you format Excel as a table, which of the following describes what happens?

(A) It changes color

(B) ** It changes color and the filters (drop down arrows for each column) are activated

(C) The title row has color

(D) Nothing happens

EXPLANATIONS BELOW

Concept note-1: -When you use Format as Table, Excel automatically converts your data range to a table. If you don’t want to work with your data in a table, you can convert the table back to a regular range while keeping the table style formatting that you applied. For more information, see Convert an Excel table to a range of data.

Concept note-2: -When someone selects a cell, the drop-down list’s down-arrow appears, and they can click it and make a selection. You can make a worksheet more efficient by providing drop-down lists. Someone using your worksheet clicks an arrow, and then clicks an entry in the list. Select the cells that you want to contain the lists.

Concept note-3: -The correct answer is Conditional formatting. Conditional formatting allows users to automatically apply formattings such as colours, icons, and data bars to one or more cells based on the cell value.

Concept note-4: -Formatting in Excel is a neat trick used to change the appearance of the data represented in the worksheet. We can do formatting in multiple ways, such as we can format the font of the cells or format the table by using the “Styles” and “Format” tabs available in the “Home” tab.
