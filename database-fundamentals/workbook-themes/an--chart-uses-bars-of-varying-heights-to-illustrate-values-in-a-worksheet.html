A(n) ____ chart uses bars of varying heights to illustrate values in a worksheet.

(A) bar

(B) pie

(C) ** column

(D) line

EXPLANATIONS BELOW

Concept note-1: -Column charts are mostly used to compare discrete values in excel. As shown in the above figure, each data point of the Column chart is plotted as a vertical rectangular bar where the height of the column represents the value of a data point.

Concept note-2: -A bar chart may be either horizontal or vertical. The important point to note about bar charts is their bar length or height-the greater their length or height, the greater their value.

Concept note-3: -Column charts are useful for showing data changes over a period of time or for illustrating comparisons among items. In column charts, categories are typically organized along the horizontal axis and values along the vertical axis.

Concept note-4: -A Bar Graph (also called Bar Chart) is a graphical display of data using bars of different heights.

Concept note-5: -Line graphs are useful for illustrating trends such as temperature changes during certain dates.
