USING THE RIGHT LEFT AND MID FUNCTIONS IN EXCEL

[SOURCES]
Extracts one or more characters from the right side of a text string

(A) LEFT

(B) ** RIGHT

(C) MID

(D) CONCATENATE

EXPLANATIONS BELOW

Concept note-1: -The RIGHT function is used to extract text from the right side of a text string. Use the LEFT function to extract text starting from the left side of the text, and the MID function to extract from the middle of the text. The LEN function returns the length of text as a count of characters.

Concept note-2: -Excel RIGHT function syntax The RIGHT function in Excel returns the specified number of characters from the end of a text string. Where: Text (required)-the text string from which you want to extract characters. Num chars (optional)-the number of characters to extract, starting from the rightmost character.

Concept note-3: -RIGHT returns the last character or characters in a text string, based on the number of characters you specify. RIGHTB returns the last character or characters in a text string, based on the number of bytes you specify.
