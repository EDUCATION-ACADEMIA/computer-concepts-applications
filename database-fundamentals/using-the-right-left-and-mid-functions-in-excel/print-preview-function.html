USING THE RIGHT LEFT AND MID FUNCTIONS IN EXCEL

Print Preview Function____

(A) ** See how the page looks before printing

(B) Print worksheets

(C) Prints a specific portion of the worksheet

(D) All wrong answers

EXPLANATIONS BELOW

Concept note-1: -Click File, and then click Print to display the Preview window and printing options. Keyboard shortcut You can also press Ctrl+F2.

Concept note-2: -Print Preview is a functionality that lets users see the pages that are about to print, allowing the users to see exactly how the pages will look when they are printed.

Concept note-3: -This option is used to view the page or make adjustments before any document gets printed. By using print preview, you can discover any errors that may exist in the document or fix the layout before printing, which can save ink/toner and paper.

Concept note-4: -Print preview is a feature that displays on the screen what a hard copy would look like when printed. Using print preview, you can find any errors that may exist or fix the layout before printing, which can save ink or toner and paper by not having to print more than once.

Concept note-5: -Print preview is a useful function to users as it helps them to see how the final printed material will appear. It also gives the user the opportunity to check or adjust the layout or resolve any issues before printing the material to achieve the intended final form.
