TYPES OF DATA

[SOURCES]
The heights of the students in a classroom

(A) Discrete

(B) ** Continuous

EXPLANATIONS BELOW

Concept note-1: -(a) the heights of the students in a class: Heights of students are measured in either feet or centimeters. These measurement units have a decimal or fractional values also. Hence, the variable can take any real number so, it is a continuous data.

Concept note-2: -Continuous variables For instance, consider the height of a student. The height can’t take any values. It can’t be negative and it can’t be higher than three metres. But between 0 and 3, the number of possible values is theoretically infinite.

Concept note-3: -Continuous data is data that can take any value. Height, weight, temperature and length are all examples of continuous data.

Concept note-4: -A continuous random variable is one which takes an infinite number of possible values. Continuous random variables are usually measurements. Examples include height, weight, the amount of sugar in an orange, the time required to run a mile. A continuous random variable is not defined at specific values.

Concept note-5: -Examples of continuous variables are body mass, height, blood pressure and cholesterol. A discrete quantitative variable is one that can only take specific numeric values (rather than any value in an interval), but those numeric values have a clear quantitative interpretation.
