Which of the following values can be indicated using conditional formatting?

(A) ** High, Low or Duplicate values

(B) Average values

(C) Projected values based on data

(D) No values

(E) None

EXPLANATIONS BELOW

Concept note-1: -More formatting options to show highest and lowest values On the Home tab, click Conditional Formatting > New Rule. In the New Formatting Rule dialog box, choose Format only top or bottom ranked values. In the drop-down list, select Top or Bottom, and type how many values to highlight in the box next to it.

Concept note-2: -They are grouped into three categories: Data Bars are horizontal bars added to each cell, much like a bar graph. Color Scales change the color of each cell based on its value. Each color scale uses a two-or three-color gradient.

Concept note-3: -Conditional formatting can help make patterns and trends in your data more apparent. To use it, you create rules that determine the format of cells based on their values, such as the following monthly temperature data with cell colors tied to cell values.

Concept note-4: -Conditional formatting makes it easy to highlight certain values or make particular cells easy to identify. This changes the appearance of a cell range based on a condition (or criteria). You can use conditional formatting to highlight cells that contain values which meet a certain condition.
