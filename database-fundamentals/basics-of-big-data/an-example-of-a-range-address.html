An example of a range address.

(A) ** A1:B3

(B) A1;B3

(C) A1-B3

(D) A1=B3

EXPLANATIONS BELOW

Concept note-1: -A group of cells is known as a cell range. Rather than a single cell address, you will refer to a cell range using the cell addresses of the first and last cells in the cell range, separated by a colon. For example, a cell range that included cells A1, A2, A3, A4, and A5 would be written as A1:A5.

Concept note-2: -The Range is the difference between the lowest and highest values. Example: In 4, 6, 9, 3, 7 the lowest value is 3, and the highest is 9. So the range is 9 − 3 = 6.

Concept note-3: -In Microsoft Excel, a range is a collection of cells. A range can be 2 or more cells and those cells don’t necessarily have to be adjacent to each other. For eg. A1: A10.

Concept note-4: -The thick border gridlines around the cell indicates that it is the Active cell inside Excel Worksheet. The Active cell is where the focus is on and where the data will be entered when a key is typed on keyboard.
