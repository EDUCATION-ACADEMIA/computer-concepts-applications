When stored, which of the following could create a large data set?

(A) List of songs

(B) Group of Images

(C) Data from search results

(D) Customer Shopping Trends

(E) ** All of the above

EXPLANATIONS BELOW

Concept note-1: -Big data is stored in three main places: in the cloud, on-site, and a hybrid model.

Concept note-2: -Big data is often stored in a data lake. While data warehouses are commonly built on relational databases and contain structured data only, data lakes can support various data types and typically are based on Hadoop clusters, cloud object storage services, NoSQL databases or other big data platforms.

Concept note-3: -Big data technologies can be categorized into four main types: data storage, data mining, data analytics, and data visualization [2].
