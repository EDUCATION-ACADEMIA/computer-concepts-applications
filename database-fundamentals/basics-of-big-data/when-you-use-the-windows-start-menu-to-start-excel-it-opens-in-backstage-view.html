When you use the Windows Start menu to start Excel, it opens in Backstage view.

(A) ** True

(B) False

EXPLANATIONS BELOW

Concept note-1: -When you use the Windows Start menu to start Excel, it opens in Backstage view. When you print in portrait orientation, the page is wider than it is tall. You can edit cell contents in the formula bar or in the cell itself. Changing a worksheet window to Normal view changes the contents of the worksheet.

Concept note-2: -You can start Excel using the Start button on the Windows taskbar of a shortcut on your desktop. You can use the DELETE key to clear cell contents. You can use the I-beam pointer to copy cell contents into adjacent cells.

Concept note-3: -When you open a template from the Templates folder in Backstage view, a new copy of the template opens as a workbook. Templates can only be used on the computer on which they are created.

Concept note-4: -In MS-Excel, the address of any cell starts with “A1‟. Hence statement 2 is false.

Concept note-5: -It is located directly above the spreadsheet. Excel formula bar is a special toolbar at the top of the Excel worksheet window. You can use it to enter a new formula or copy an existing one. The formula bar displays information entered in the current or active cell.
