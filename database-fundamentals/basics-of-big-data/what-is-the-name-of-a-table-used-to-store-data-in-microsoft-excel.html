What is the name of a table used to store data in Microsoft Excel?

(A) Sheet Tab

(B) Name Box

(C) ** Title Bar

(D) Worksheet

EXPLANATIONS BELOW

Concept note-1: -The title bar displays both the name of the application and the name of the spreadsheet.

Concept note-2: -A table header row is the top row of a table that acts as a title for the type of information they will find in each column. It’s common to manually bold the top row to signal this information visually, but it’s important to mark table headers at the code level so the change is also structural.
