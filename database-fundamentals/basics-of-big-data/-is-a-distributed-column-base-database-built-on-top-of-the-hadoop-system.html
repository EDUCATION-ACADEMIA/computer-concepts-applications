____ is a distributed column base database built on top of the Hadoop system

(A) ** Hadoop

(B) HBase

(C) Hive

(D) HDFS

EXPLANATIONS BELOW

Concept note-1: -HBase is a column-oriented non-relational database management system that runs on top of Hadoop Distributed File System (HDFS). HBase provides a fault-tolerant way of storing sparse data sets, which are common in many big data use cases.

Concept note-2: -What is HBase? HBase is a distributed column-oriented database built on top of the Hadoop file system. It is an open-source project and is horizontally scalable. HBase is a data model that is similar to Googleâ€™s big table designed to provide quick random access to huge amounts of structured data.

Concept note-3: -HBase is an open-source non-relational, scalable, distributed database written in Java. It is developed as a part of the Hadoop ecosystem and runs on top of HDFS. It provides random real-time read and write access to the given data. It is possible to write NoSQL queries to get the results using APIs.

Concept note-4: -Yes, Hbase is known to be a column oriented database (where the column data stay together), the data in HBase for a particular row stay together and the column data is spread and not together.

Concept note-5: -Explanation: HBase is a distributed column-oriented database built on top of the Hadoop file system.
