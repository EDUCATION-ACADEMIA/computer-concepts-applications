BASICS OF BIG DATA

[SOURCES]
The “boxes” going from left to right are called ____

(A) ** Rows

(B) Columns

(C) Letters

(D) Titles

EXPLANATIONS BELOW

Concept note-1: -It depends on what you are talking about exactly. In the case of graph paper, a ledger or spreadsheet, they are called ‘cells’.

Concept note-2: -The cell. An Excel worksheet is made up of columns and rows. Where these columns and rows intersect, they form little boxes called cells.

Concept note-3: -The cell or range of cells for which you want the row number. If reference is omitted, it is assumed to be the reference of the cell in which the ROW function appears. If reference is a range of cells, and if ROW is entered as a vertical array, ROW returns the row numbers of reference as a vertical array.

Concept note-4: -MS Excel is in tabular format consisting of rows and columns. Row runs horizontally while Column runs vertically. Each row is identified by row number, which runs vertically at the left side of the sheet. Each column is identified by column header, which runs horizontally at the top of the sheet.

Concept note-5: -In an Excel worksheet, each small rectangle or box is known as a cell. The active cell is the selected cell in which data is entered when you begin typing.
