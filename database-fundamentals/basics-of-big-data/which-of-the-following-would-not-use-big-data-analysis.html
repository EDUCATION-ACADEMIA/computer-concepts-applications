BASICS OF BIG DATA

[SOURCES]
Which of the following would NOT use big data analysis?

(A) Healthcare companies

(B) Public agencies

(C) Retail companies

(D) ** High school students doing homework

EXPLANATIONS BELOW

Concept note-1: -1 Answer. It would be high school students doing homework, who would not use big data analytics to get the job done.

Concept note-2: -Data management For instance, if you have graph data, it might be useless for campaign data and yet very useful for network analysis. Machines won’t be able to analyze this type of data so you will have to manage and process it manually, changing it so that machines can work with it.

Concept note-3: -Drawbacks or disadvantages of Big Data ➨Big data analysis violates principles of privacy. ➨It can be used for manipulation of customer records. ➨It may increase social stratification. ➨Big data analysis is not useful in short run.

Concept note-4: -Examples of big data applications are : Advertising and Marketing. Banking and Financial Services. Government. Media and Entertainment.

Concept note-5: -Big Data Analytics Tools Hadoop-helps in storing and analyzing data. MongoDB-used on datasets that change frequently. Talend-used for data integration and management. Cassandra-a distributed database used to handle chunks of data.
