HOW TO MODIFY THE WORKSHEET TAB IN EXCEL

[SOURCES]
Print Titles are rows or columns that are repeated on every page when you print a worksheet. How do you set Print Titles?

(A) ** Select Page Layout and go to Print Titles on the Page Setup group. Select if you want to repeat rows or columns and then highlight the rows or columns that you want as titles

(B) Go to the Backstage View by clicking on the File tab and changing the Print Titles property

(C) Go to the Insert tab and select Object in the Text group. Enter the rows or columns that you want to use as titles

(D) Highlight the rows or columns that you want to set as the titles then right-click and select Set as Print Titles

EXPLANATIONS BELOW

Concept note-1: -If a worksheet spans more than one printed page, you can label data by adding row and column headings that will appear on each print page. These labels are also known as print titles.

Concept note-2: -On the Ribbon, click the Page Layout tab. In the Sheet Options group, under Headings, select the Print check box., and then under Print, select the Row and column headings check box . To print the worksheet, press CTRL+P to open the Print dialog box, and then click OK.
