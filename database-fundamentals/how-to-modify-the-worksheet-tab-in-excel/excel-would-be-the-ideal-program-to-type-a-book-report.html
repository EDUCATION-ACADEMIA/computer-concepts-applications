Excel would be the ideal program to type a book report?

(A) ** False

(B) True

EXPLANATIONS BELOW

Concept note-1: -The answer to this question is false because Microsoft Excel is not the most suited program for documentation. It is because Microsoft Excel is one of the spreadsheet programs which are not a documentation program.

Concept note-2: -Excel is an incredibly powerful tool for getting meaning out of vast amounts of data. But it also works really well for simple calculations and tracking almost any kind of information. The key for unlocking all that potential is the grid of cells. Cells can contain numbers, text, or formulas.
