HOW TO MODIFY THE WORKSHEET TAB IN EXCEL

[SOURCES]
Which group on the Data tab includes commands that enable you to connect to external data sources and create new queries in the workbook?

(A) ** Get & Transform

(B) Look and Find

(C) View and Print

(D) Source externals

EXPLANATIONS BELOW

Concept note-1: -On the Data tab, in the Get External Data group, click From Other Sources, and then click From Microsoft Query. Do one of the following: To specify a data source for a database, text file, or Excel workbook, click the Databases tab.

Concept note-2: -Click Options > Trust Center > Trust Center Settings, and then click External Content. Click the option that you want under Security settings for Linked Data Types: Enable all Linked Data Types (not recommended) Click this option if you want to create linked data types without receiving a security warning.

Concept note-3: -The Workbook Connections dialog box (Select Data > Connections) helps you manage one or more connections to external data sources in your workbook. You can use this dialog box to do the following: Create, edit, refresh, and delete connections that are in use in the workbook.
