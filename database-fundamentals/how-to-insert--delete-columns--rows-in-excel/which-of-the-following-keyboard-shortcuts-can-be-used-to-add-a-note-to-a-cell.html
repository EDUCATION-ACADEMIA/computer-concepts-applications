HOW TO INSERT DELETE COLUMNS ROWS IN EXCEL

[SOURCES]
Which of the following keyboard shortcuts can be used to add a note to a cell?

(A) ** Shift+F2

(B) Ctrl+N

(C) Alt+Shift+C

(D) Ctrl+F9

EXPLANATIONS BELOW

Concept note-1: -This shortcut will enter the comment in the selected cell. For example, if you select cell A1 and use the Shift + F2 keyboard shortcut, the comment “This is a comment” will be entered in the cell. 12. To enter a value in a cell and move to the next cell: Use the Tab key.

Concept note-2: -Shift+F2 adds or edits a cell comment.

Concept note-3: -Right-click the cell and then click Insert Comment (or press Shift+F2). If you’re using Excel for Office 365, right-click the cell and choose New Note.

Concept note-4: -Everybody (well, almost everybody) knows that pressing the F2 key in Excel activates the “editing” mode for the active cell – the cursor goes into the cell so that you can change the contents and the various cell references in that formula turn different colours.

Concept note-5: -Whether you want to force-quit a hanging application, quickly launch your favorite program, or perform any other task that you can think of, Alt + F2 is there to ease the task for you. This keyboard shortcut works by default on all Linux distributions that use GNOME, KDE, or Xfce desktop environments.
