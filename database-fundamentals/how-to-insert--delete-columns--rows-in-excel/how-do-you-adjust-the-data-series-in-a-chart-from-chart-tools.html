How do you adjust the data series in a chart from Chart Tools?

(A) Go to the Format tab and Format Selection.

(B) Go to the Design tab and Add Chart Elements.

(C) Go to the Design tab and Change Chart Type.

(D) ** Go to the Design tab and Select Data.

EXPLANATIONS BELOW

Concept note-1: -On the ribbon, click Chart Design and then click Select Data. This selects the data range of the chart and displays the Select Data Source dialog box. To edit a legend series, in the Legend entries (series) box, click the series you want to change. Then, edit the Name and Y values boxes to make any changes.

Concept note-2: -On the Design tab, in the Type group, click Change Chart Type. In the Change Chart Type dialog box, click a chart type that you want to use. The first box shows a list of chart type categories, and the second box shows the available chart types for each chart type category.

Concept note-3: -To edit the data selection of a chart, right click the chart and select the Select Data option-or select the chart, go to the Design tab, and click the Select Data button from the left side of the ribbon menu.

Concept note-4: -In the chart, select a data series. For example, in a column chart, click a column, and all the columns of that data series become selected. Select Chart Design > Select Data. In the Select Data Source dialog box, next to Legend entries (Series), use the up and down arrows to move the series up or down in the list.

Concept note-5: -On the Design tab, you can activate/deactivate both the Title and Subtitle. Once activated, you can add text and set the text alignment: The Chart Title and Subtitle are limited to 150 characters including spaces.
