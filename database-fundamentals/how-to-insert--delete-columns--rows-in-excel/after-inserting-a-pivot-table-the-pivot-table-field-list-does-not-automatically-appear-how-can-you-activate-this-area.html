After inserting a Pivot Table, the Pivot Table Field List does not automatically appear. How can you activate this area?

(A) Click on the Insert Pivot Table button and select PivotTable Fields List.

(B) Go to the Backstage View and in the Options dialogue box, click on the PivotTable tab. Check the box for PivotTable Fields List.

(C) ** Click on the Field List button in the Show group, under Options in the PivotTable Tools contextual tab.

(D) All of the options listed above are correct.

EXPLANATIONS BELOW

Concept note-1: -The Field List should appear when you click anywhere in the PivotTable. If you click inside the PivotTable but don’t see the Field List, open it by clicking anywhere in the PivotTable. Then, show the PivotTable Tools on the ribbon and click Analyze> Field List.

Concept note-2: -If the PivotTable Field List task pane does not appear automatically, follow these steps to enable it: Click any cell in the pivot table layout. On the Excel Ribbon, click the PivotTable Analyze tab. Then in the Show group (at the far right), click the Field List command.

Concept note-3: -To Unhide tables and fields, return to Power Pivot: Power Pivot > Manage, right-click the item to unhide, and select Unhide from Client Tools. Back in Power View, in your Fields list, you can see the fields that were Unhidden.

Concept note-4: -Right-click any cell in the pivot table and select Show Field List from the menu. This will make the field list visible again and restore it’s normal behavior. The field list will disappear when a cell outside the pivot table is selected, and it will reappear again when a cell inside the pivot table is selected.
