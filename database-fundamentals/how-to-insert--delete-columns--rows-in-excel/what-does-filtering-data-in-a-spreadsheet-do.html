What does filtering data in a spreadsheet do?

(A) Shows data that did not meet the criteria

(B) ** Hides data that you do not want to see

(C) Shows all the data in the spreadsheet

(D) Hides the data you are looking for

EXPLANATIONS BELOW

Concept note-1: -Filtered data displays only the rows that meet criteria that you specify and hides rows that you do not want displayed. After you filter data, you can copy, find, edit, format, chart, and print the subset of filtered data without rearranging or moving it. You can also filter by more than one column.

Concept note-2: -Filtering Data When data is filtered, only rows that meet the filter criteria will display and other rows will be hidden. With filtered data, you can then copy, format, print, etc., your data, without having to sort or move it first.

Concept note-3: -Select the range of cells whose formulas you want to hide. You can also select nonadjacent ranges or the entire sheet. Click Home > Format > Format Cells. On the Protection tab, select the Hidden check box.

Concept note-4: -Data filtering is the process of choosing a smaller part of your data set and using that subset for viewing or analysis. Filtering is generally (but not always) temporary â€“ the complete data set is kept, but only part of it is used for the calculation.

Concept note-5: -Sorting: It is a feature that lets you arrange data in ascending or descending order. Filtering: It is a feature that allows you to view only selective data that satisfies a given condition.
