Which command would you use to group a set of cells together?

(A) ** Merge

(B) Cut

(C) Format as Table

(D) Cell Group

EXPLANATIONS BELOW

Concept note-1: -Merge cells Click the first cell and press Shift while you click the last cell in the range you want to merge. Important: Make sure only one of the cells in the range has data. Click Home > Merge & Center.

Concept note-2: -Merge cells Select the cells that you want to merge. Under Table Tools, on the Layout tab, in the Merge group, click Merge Cells.

Concept note-3: -Merging cells is often used when a title is to be centered over a particular section of a spreadsheet. When a group of cells is merged, only the text in the upper-leftmost box is preserved.
