HOW TO INSERT DELETE COLUMNS ROWS IN EXCEL

[SOURCES]
The Fill Handle copies both the content AND the formatting to an adjacent cell or range.

(A) ** True

(B) False

EXPLANATIONS BELOW

Concept note-1: -You can quickly copy formulas into adjacent cells by using the fill handle. When you drag the handle to other cells, Excel will show different results for each cell.

Concept note-2: -The fill handle allows you to copy and paste formulas to adjacent cells. You can hide or delete multiple rows or columns by using the Ctrl or Shift key and then selecting the rows or columns you want to hide or delete. In Excel, columns are identified by numbers and rows by letters.

Concept note-3: -Copy cells by using Copy and Paste Select the cell or range of cells. Select Copy or press Ctrl + C. Select Paste or press Ctrl + V.

Concept note-4: -Fill Handle: The fill handle is the small bold square in the bottom right corner of a cell that can be used to copy (fill) data to adjacent cells in the same row or column.
