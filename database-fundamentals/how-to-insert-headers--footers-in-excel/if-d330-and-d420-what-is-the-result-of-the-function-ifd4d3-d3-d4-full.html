HOW TO INSERT HEADERS FOOTERS IN EXCEL

[SOURCES]
If D3=30 and D4=20, what is the result of the function =IF(D4<D3, D3-D4, “FULL")?

(A) Full

(B) Unknown

(C) ** 10

(D) -10

EXPLANATIONS BELOW

Concept note-1: -If you want to maintain the original cell reference when you copy it, you “lock” it by putting a dollar sign ($) before the cell and column references. For example, when you copy the formula =$A$2+$B$2 from C2 to D2, the formula stays exactly the same. This is an absolute reference.

Concept note-2: -The SUM function adds values. You can add individual values, cell references or ranges or a mix of all three. For example: =SUM(A2:A10) Adds the values in cells A2:10.

Concept note-3: -Microsoft Excel might show ##### in cells when a column isn’t wide enough to show all of the cell contents. Formulas that return dates and times as negative values can also show as #####.

Concept note-4: -Combine data with the Ampersand symbol (&) Type = and select the first cell you want to combine. Type & and use quotation marks with a space enclosed. Select the next cell you want to combine and press enter. An example formula might be =A2&” “&B2.
