In the Vlookup function the TRUE and FALSE values ​​can be replaced with the ?

(A) ** 1 and 0

(B) 0 and 1

(C) 1 and 3

(D) 2 and 3

EXPLANATIONS BELOW

Concept note-1: -True is the same as 1, False is the same as 0 . The lookup range should normally be in ascending order from top to bottom. Vlookup will return the last row where the range value is <= the lookup value. If no rows meet this criteria then #N/A is returned.

Concept note-2: -About the VLOOKUP function The number of the column within your defined range, that contains the return value; 0 or FALSE for an exact match with the value your are looking for; 1 or TRUE for an approximate match.

Concept note-3: -Identify a column of cells you’d like to fill with new data. Select ‘Function’ (Fx) > VLOOKUP and insert this formula into your highlighted cell. Enter the lookup value for which you want to retrieve new data. Enter the table array of the spreadsheet where your desired data is located.

Concept note-4: -TRUE or 1: If we provide TRUE, it will look for an approximate match. FALSE or 0: If we provide FALSE, it will look for an exact match. Since [Range Lookup] is an optional argument, it will take TRUE as the default parameter. Now, look at the below data tables in excel.
