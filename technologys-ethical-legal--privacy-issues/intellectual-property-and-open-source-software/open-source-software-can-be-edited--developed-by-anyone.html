INTELLECTUAL PROPERTY AND OPEN SOURCE SOFTWARE

[SOURCES]
Open Source software can be edited / developed by anyone.

(A) ** False

(B) True

EXPLANATIONS BELOW

Concept note-1: -Generally, Open Source software is software that can be freely accessed, used, changed, and shared (in modified or unmodified form) by anyone. Open source software is made by many people, and distributed under licenses that comply with the Open Source Definition.

Concept note-2: -Who Owns It? Typically, for open-source projects, the creator owns the software but provides a license to other “contributors” to reproduce, modify and redistribute the software. The licenses offered to contributors and users of the software are often standardized (see MIT, GNU LPGL, and Apache as examples).

Concept note-3: -Open source software is software with source code that anyone can inspect, modify, and enhance. “Source code” is the part of software that most computer users don’t ever see; it’s the code computer programmers can manipulate to change how a piece of software-a “program” or “application"-works.

Concept note-4: -Overview. Open source is a term that originally referred to open source software (OSS). Open source software is code that is designed to be publicly accessible-anyone can see, modify, and distribute the code as they see fit.

Concept note-5: -A developer working on a limited but working codebase, releases it to the public as the first version of an open-source program. The source code of a mature project is released to the public. A well-established open-source project can be forked by an interested outside party.
