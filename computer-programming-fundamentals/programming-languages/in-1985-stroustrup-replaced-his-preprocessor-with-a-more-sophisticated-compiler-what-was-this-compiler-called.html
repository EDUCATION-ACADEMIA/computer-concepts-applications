PROGRAMMING LANGUAGES

[SOURCES]
In 1985, Stroustrup replaced his preprocessor with a more sophisticated compiler. What was this compiler called?

(A) C

(B) C++

(C) Cpre

(D) ** Cfront

EXPLANATIONS BELOW

Concept note-1: -The Simula 67 language-which was the variant that Stroustrup worked with-is regarded as the first language to support the object-oriented programming paradigm. Stroustrup found that this paradigm was very useful for software development, however the Simula language was far too slow for practical use.

Concept note-2: -C++ was developed by Bjarne Stroustrup, as an extension to the C language. C++ gives programmers a high level of control over system resources and memory. The language was updated 4 major times in 2011, 2014, 2017, and 2020 to C++11, C++14, C++17, C++20.

Concept note-3: -C++ is a high-level computer programming language developed by Bjarne Stroustrup of Bell Laboratories in the early 1980s, it is based on the traditional C language but with added object-oriented programming and other capabilities.

Concept note-4: -Since C++ is an attempt to add object-oriented features (plus other improvements) to C, earlier it was called as “C with Objects”. As the language developed, Stroustrup named it as C++ in 1983. The name C++ suggests “C incremented” (recall the ++ is an increment operator of C).
