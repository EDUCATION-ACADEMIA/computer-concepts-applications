Basic stands for beginners’ All-purpose Symbolic instruction computation.

(A) true

(B) ** false

EXPLANATIONS BELOW

Concept note-1: -BASIC stands for “Beginner’s All-purpose Symbolic Instruction Code.” Originally designed as an interactive mainframe timesharing language by John Kemeney and Thomas Kurtz in 1963, it became widely used on personal computers everywhere. On IBM’s first “family” computer, the PCJr, a BASIC cartridge was a popular add-on.

Concept note-2: -BASIC (Beginners’ All-purpose Symbolic Instruction Code) is a family of general-purpose, high-level programming languages designed for ease of use. The original version was created by John G. Kemeny and Thomas E. Kurtz at Dartmouth College in 1963.

Concept note-3: -BASIC, in fullBeginner’s All-purpose Symbolic Instruction Code, computer programming language developed by John G. Kemeny and Thomas E. Kurtz at Dartmouth College in the mid 1960s.

Concept note-4: -Python is always recommended if you’re looking for an easy and even fun programming language to learn first. Rather than having to jump into strict syntax rules, Python reads like English and is simple to understand for someone who’s new to programming.

Concept note-5: -COBOL stands for Common Business-Oriented Language, while FORTRAN stands for Formula Translation.
