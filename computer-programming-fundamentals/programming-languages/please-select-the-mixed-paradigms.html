EXPLANATIONS BELOW

Concept note-1: -Python is a multi-paradigm programming language. Meaning it supports different styles of writing code. One can write Python code in procedural, object oriented, functional or imperative manner. For this reason, Python is considered a “swiss army knife” in the developers toolbox.

Concept note-2: -It is based on Perl, Smalltalk, Eiffel, Ada, and Lisp. Ruby supports multiple programming paradigms, including functional, object oriented, imperative, and reflective. It also has a dynamic type system and automatic memory management.

Concept note-3: -Imperative, procedural, functional, declarative, and object oriented paradigms are some of the most popular and widely used paradigms today. And knowing the basics about them is good for general knowledge and also for better understanding other topics of the coding world.
