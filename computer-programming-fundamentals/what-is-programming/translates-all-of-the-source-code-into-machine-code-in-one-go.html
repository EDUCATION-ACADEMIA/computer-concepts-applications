COMPUTER PROGRAMMING FUNDAMENTALS

WHAT IS PROGRAMMING

[SOURCES]
Translates all of the source code into machine code in one go

(A) Assembler

(B) ** Compiler

(C) Interpreter

EXPLANATIONS BELOW

Concept note-1: -The correct answer is Compiler. Compiler-A compiler is a special program that processes statements written in a particular programming language and turns them into machine language or “code” that a computer’s processor uses. Example-Pascal, C etc.

Concept note-2: -Compilers. A compiler takes the source code as a whole and translates it into machine code all in one go. Once converted, the object code can be run unassisted at any time. This process is called compilation .

Concept note-3: -A compiler is a special program that translates a programming language’s source code into machine code, bytecode or another programming language.

Concept note-4: -Compilers are used to translate a program written in a high-level language into machine code (object code). Once compiled (all in one go), the translated program file can then be directly used by the computer and is independently executable.

Concept note-5: -Compilers often translate source code for a high-level language, such as C++, to object code for the current computer architecture, such as Intel x64. The object modules produced from multiple high-level language files are then linked into an executable file.
