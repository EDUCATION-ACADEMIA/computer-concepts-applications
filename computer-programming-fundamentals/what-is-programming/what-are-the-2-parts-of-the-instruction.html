What are the 2 parts of the instruction?

(A) Opinstruction and Opmemory

(B) Opcommand and Opstate

(C) ** Opcode and Operand

(D) Opcommand and Opmemory

EXPLANATIONS BELOW

Concept note-1: -The opcode is the instruction that is executed by the CPU and the operand is the data or memory location used to execute that instruction.

Concept note-2: -They are saved in the memory along with the information. Each computer has its specific group of instructions. They can be categorized into two elements as Operation codes (Opcodes) and Address.

Concept note-3: -Operands are definite elements of instructions that specify the data on which operation is to be done. In general, each instruction code is 16-bit and consists of three parts called fields, which include the mode, the opcode, and the address.

Concept note-4: -They can be categorized into two elements as Operation codes (Opcodes) and Address. Opcodes specify the operation for specific instructions, and an address determines the registers or the areas used for that operation.

Concept note-5: -The first word of each instruction is called the opcode word. It contains an operation code (opcode) and zero, one, or two effective-address (EA) fields.
