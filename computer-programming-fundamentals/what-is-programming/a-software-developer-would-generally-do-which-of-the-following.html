WHAT IS PROGRAMMING

A software developer would generally do which of the following?

(A) ** Write programs, websites and databases

(B) Design a Website

(C) Manage a team

(D) Fix Computers

EXPLANATIONS BELOW

Concept note-1: -Software developers typically do the following: Analyze users’ needs and then design and develop software to meet those needs. Recommend software upgrades for customers’ existing programs and systems. Design each piece of an application or system and plan how the pieces will work together.

Concept note-2: -As part of their job, database developers work closely with other software developers and stakeholders to ensure the database meets their needs and supports the applications and systems that rely on it. In addition, they ensure the database’s security, scalability, and performance.

Concept note-3: -A web developer specializes in web-based applications such as websites, e-commerce, and mobile development. Whereas a software developer will concentrate on creating software for the underlying operating system, network, or platform.” “The main difference is the type of projects you might get to work on.

Concept note-4: -Most software developers worldwide report the use of Windows operating system as their preferred development environment, as of 2021. Apple’s macOS comes in third with 44 percent, behind the 47 percent of developers preferring Linux.
