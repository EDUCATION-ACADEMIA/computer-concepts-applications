COMPUTER PROGRAMMING FUNDAMENTALS

WHAT IS PROGRAMMING

[SOURCES]
Compilers produce executable files .

(A) ** True

(B) False

EXPLANATIONS BELOW

Concept note-1: -Compiler creates executable file and after that no need of compiler for running the program again and again while in case of interpreter, it reads your program instruction by instruction and executes it that means you need interpreter each time you need to run the program.

Concept note-2: -Compilers generate object (executable) files from source code files. There are several utilities, libraries and objects used by compilers: Compiler languages: C/C++

Concept note-3: -Users can create executable files in a basic text editor or in an integrated development environment (IDE). If using a text editor, users need a separate compiler to turn the file into machine code.

Concept note-4: -Compilers, transpilers, interpreters, and JIT compilers Compilers intended to produce object code for architectures that differ from the one running the compiler are called cross-compilers. It is common to use cross-compilers running on desktop (or larger) computers to produce executables for embedded systems.
