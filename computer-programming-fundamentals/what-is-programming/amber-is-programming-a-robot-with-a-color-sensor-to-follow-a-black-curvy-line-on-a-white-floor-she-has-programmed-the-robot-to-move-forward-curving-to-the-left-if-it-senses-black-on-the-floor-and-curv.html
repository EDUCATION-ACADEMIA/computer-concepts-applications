COMPUTER PROGRAMMING FUNDAMENTALS

WHAT IS PROGRAMMING

[SOURCES]
Amber is programming a robot with a color sensor to follow a black curvy line on a white floor. She has programmed the robot to move forward, curving to the left if it senses black on the floor, and curving right if it does not sense black. She is hoping this will allow the robot to follow the black line. Which type of control structure has Amber written?

(A) ** Selection

(B) Sequence

(C) Loop

(D) Functional

EXPLANATIONS BELOW

Concept note-1: -Sequence is the default control structure; instructions are executed one after another.

Concept note-2: -A boolean expression(named for mathematician George Boole) is an expression that evaluates to either true or false.

Concept note-3: -Code tracing is a method in which the programmer uses paper and pencil to hand trace the execution of a program or code segment in order to track the variable values as they change during execution and to determine the output of the code.

Concept note-4: -In selection control structures, conditional statements are features of a programming language which perform different computations or actions depending on whether a programmer-specified Boolean condition evaluates to true or false.
