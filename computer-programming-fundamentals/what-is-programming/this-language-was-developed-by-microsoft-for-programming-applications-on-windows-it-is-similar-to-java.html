This language was developed by Microsoft for programming applications on windows. It is similar to Java.

(A) ** C#

(B) Javascript

(C) Visual Basic

(D) Python

EXPLANATIONS BELOW

Concept note-1: -C# was created by Microsoft for Microsoft, so it’s easy to see why it’s most popularly used for the development of Windows desktop applications.

Concept note-2: -C++: C++ is the workhorse language at Microsoft, which uses C++ to build many of its core applications.

Concept note-3: -The C# programming language was designed by Anders Hejlsberg from Microsoft in 2000 and was later approved as an international standard by Ecma (ECMA-334) in 2002 and ISO/IEC (ISO/IEC 23270) in 2003.

Concept note-4: -1 | Scala. Scala is a statically typed programming language that fuses the object-oriented model and functional programming ideas. That means, in practice, that you can declare classes, create objects, and call methods just like you would typically do in Java.

Concept note-5: -Microsoft Visual Programming Language, or VPL, is a visual programming and dataflow programming language developed by Microsoft for the Microsoft Robotics Studio. VPL is based on the event-driven and data-driven approach.
