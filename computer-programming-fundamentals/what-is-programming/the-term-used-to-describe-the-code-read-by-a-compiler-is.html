COMPUTER PROGRAMMING FUNDAMENTALS

WHAT IS PROGRAMMING

[SOURCES]
The term used to describe the code READ by a compiler is..

(A) Object Code

(B) Machine Code

(C) ** Source Code

(D) Binary Code

EXPLANATIONS BELOW

Concept note-1: -The first type of source code is compiler source code, whereby a programmer writes the code that a compiler directly converts into a machine or binary code that a computer can execute. This direct translation leads to more efficient execution of the code.

Concept note-2: -Source code is generally understood to mean programming statements that are created by a programmer with a text editor or a visual programming tool and then saved in a file. Object code generally refers to the output, a compiled file, which is produced when the Source Code is compiled with a C compiler.

Concept note-3: -Source code and object code are sometimes referred to as the before and after versions of a compiled computer program.

Concept note-4: -Compilers take source code, such as C or Basic, and compile it into machine code. Interpreted languages differ from compiled languages; for example, interpreted code, such as shell code, is compiled on the fly each time the program is run. Bytecode, such as Java bytecode, is also interpreted code.

Concept note-5: -Source code is the source of a computer program. It contains declarations, instructions, functions, loops and other statements, which act as instructions for the program on how to function.
