Which term describes a set of instructions that lists steps that must be followed in a particular order, starting at the beginning and continuing to the end?

(A) Executed

(B) Compiled

(C) Assembly

(D) ** Sequential

Concept note-1: -An algorithm is a procedure used for solving a problem or performing a computation. Algorithms act as an exact list of instructions that conduct specified actions step by step in either hardware-or software-based routines.
