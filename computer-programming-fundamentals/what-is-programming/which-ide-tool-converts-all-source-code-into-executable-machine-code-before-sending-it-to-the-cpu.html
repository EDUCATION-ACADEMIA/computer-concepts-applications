Which IDE tool converts all source code into executable machine code before sending it to the CPU?

(A) Loader

(B) ** Compiler

(C) Debugger

(D) Linker

EXPLANATIONS BELOW

Concept note-1: -The correct answer is Compiler. Compiler-A compiler is a special program that processes statements written in a particular programming language and turns them into machine language or “code” that a computer’s processor uses. Example-Pascal, C etc.

Concept note-2: -A compiler is a special program that translates a programming language’s source code into machine code, bytecode or another programming language.

Concept note-3: -Compilers are a type of translator that support digital devices, primarily computers. The name compiler is primarily used for programs that translate source code from a high-level programming language to a lower level language (e.g., assembly language, object code, or machine code) to create an executable program.

Concept note-4: -Compilers are utility programs that take your code and transform it into executable machine code files. When you run a compiler on your code, first, the preprocessor reads the source code (the C++ file you just wrote).

Concept note-5: -A compiler takes the source code as a whole and translates it into object code all in one go. Once converted, the object code can be run at any time. This process is called compilation .
