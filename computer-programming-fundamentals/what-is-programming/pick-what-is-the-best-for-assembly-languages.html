EXPLANATIONS BELOW

Concept note-1: -In this generation, assembly language and high-level programming languages like FORTRAN, COBOL were used. The computers used batch processing and multiprogramming operating system.

Concept note-2: -Assembly or assembler language was the second generation of computer language. By the late 1950s, this language had become popular. Assembly language consists of letters of the alphabet. This makes programming much easier than trying to program a series of zeros and ones.

Concept note-3: -Assembly languages or asm is a low-level language that is used to implement the code in machine code which directly interacts with the system hardware. It is a second-generation language based on the machine code which is a first-generation language.
