COMPUTER PROGRAMMING FUNDAMENTALS

WHAT IS PROGRAMMING

[SOURCES]
A language that is translated all at once into a form the machine can understand, and then executed by the machine.

(A) ** compiled

(B) reduced

(C) interpreted

(D) translated

EXPLANATIONS BELOW

Concept note-1: -A program that converts a high-level language source file into a machine-language file is called a compiler.

Concept note-2: -Coding. Coding is the act of translating the design into an actual program, written in some form of programming language. This is the step where you actually have to sit down at the computer and type!

Concept note-3: -Compilers. Compilers translate an entire computer program into machine language before execution.

Concept note-4: -A high-level language is a programming language that uses English and mathematical symbols in its instructions. To execute a program in a high-level language, it can be compiled or interpreted. A compiler translates the entire program written in a high-level language to machine language prior to execution.
