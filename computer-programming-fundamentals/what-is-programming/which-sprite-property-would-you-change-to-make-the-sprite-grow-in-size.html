Which sprite property would you change to make the sprite grow in size?

(A) sprite.x

(B) sprite.y

(C) ** sprite.scale

(D) sprite.rotation

EXPLANATIONS BELOW

Concept note-1: -You can make a Sprite bigger or smaller with the SET SIZE command. ACTIONS Toolkit-Scroll down to SPRITE SETTINGS-Set Size Block. Use a number less than 1 to make your sprite smaller and a number bigger than 1 to make the sprite larger. This example program makes a sprite half of it’s original size.

Concept note-2: -scale. Shrink or grow a sprite keeping the height to width ratio the same. For example, a value of 2 will be make the sprite twice as big and a value of 0.5 will make the sprite half as big.

Concept note-3: -You can increase or decrease a sprite from it’s current size to a new size by setting a scale value. If a sprite’s scale is set to 2, then the sprite’s new size is two times larger than it’s original size. The sprite’s size will shrink if you give a negative value.

Concept note-4: -use sprite. scale to resize a sprite and keep the height to width ratio constant. Sprites all have the same properties and you use the dot notation (combining the name of the sprite, followed by a dot, with the label of the property) to both access and update the property for that sprite.
