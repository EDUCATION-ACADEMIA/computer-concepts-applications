What is the power source for the Raspberry Pi 3?

(A) USB-C

(B) ** Micro USB

(C) USB-A

EXPLANATIONS BELOW

Concept note-1: -The official Raspberry Pi 12.5W Micro USB Power Supply is the latest recommended power supply for all micro USB-powered Raspberry Pi computers.

Concept note-2: -The first, recommended and easiest way to power the Raspberry Pi is via the Micro USB port on the side of the unit. The recommended input voltage is 5V, and the recommended input current is 2A. At The Pi Hut, our standard power supply for the Raspberry Pi is 5.1V 2.5A.

Concept note-3: -Power-up! The Raspberry Pi 3 officially uses a 5.1 V micro USB power supply.

Concept note-4: -Powering using a micro-usb cable Another way to power the Raspberry Pi is with a micro-usb cable connected to a usb port on a laptop. Note that there are different types of usb connectors. The Raspberry Pi uses a micro-usb connector. You can also directly power it with an AC adapter.
