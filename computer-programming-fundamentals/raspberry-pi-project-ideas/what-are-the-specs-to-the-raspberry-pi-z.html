What are the specs to the Raspberry Pi Z?

(A) Dual Core 1GHz Processer

(B) 1GB RAM

(C) ** 40GPIO Pins

(D) ** OTG USB-OUT

(E) DSI port

EXPLANATIONS BELOW

Concept note-1: -If you’re adding USB drives or other USB devices, please note that the maximum current the Raspberry Pi 4** USB ports (either combined or a single USB port) can pass through is 1.2A.

Concept note-2: -Official 5.1V 3A Raspberry Pi 4 Power Supply (US) White. Input: 100-240V 50/60Hz 0.5A, Output: 5.1-3.0A.

Concept note-3: -The Raspberry Pi Zero and Pi Zero W feature a USB OTG port, allowing users to configure the device as (amongst other things) an Ethernet device. In this mode, it is possible to control the Pi Zero’s GPIO pins over USB from another computer using the remote GPIO feature.
