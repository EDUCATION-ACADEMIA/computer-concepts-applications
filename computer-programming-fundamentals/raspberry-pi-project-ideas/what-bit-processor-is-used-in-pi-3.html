What bit processor is used in Pi 3?

(A) ** 64-bit

(B) 32-bit

(C) 128-bit

(D) Both 64 and 32 bit

EXPLANATIONS BELOW

Concept note-1: -The Raspberry Pi 3 Model B was released in February 2016 with a 1.2 GHz 64-bit quad core ARM Cortex-A53 processor, on-board 802.11n Wi-Fi, Bluetooth and USB boot capabilities.

Concept note-2: -The Raspberry Pi 3 is equipped with a quad-core 64-bit Broadcom BCM2837 ARM Cortex-A53 SoC processor running at 1.2 GHz, making it about 50% more powerful than the Pi 2.

Concept note-3: -Awesome Extras. The latest Raspberry Pi 3 Model B+ has a faster 64-bit 1.4GHz quad core processor, 1GB of RAM, faster dual-band 802.11 b/g/n/ac wireless LAN, Bluetooth 4.2, and significantly faster 300Mbit/s ethernet.

Concept note-4: -How to check if my Raspberry Pi OS is 64 bit? This prints out the machine hardware name and if it says aarch64, then you have a 64 bit version. If it says armv7l then you have the 32 bit OS.

Concept note-5: -Your Raspberry Pi will now be using the 64-bit Raspberry Pi OS and can run 64-bit Linux applications that have the ARM64 architecture (aka AArch64). Since Raspberry Pi OS is based on Debian, this means you should be able to install and run any standard ARM64 Debian package on the 64-bit version of the OS.
