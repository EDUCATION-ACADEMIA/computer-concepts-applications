USING MICROSOFT POWERPOINT

FORMATTING YOUR POWERPOINT PRESENTATION USING SLIDE MASTERS AND LAYOUTS

[SOURCES]
____ defines the hue of the characters.

(A) Font

(B) ** Color

(C) Size

(D) Style

EXPLANATIONS BELOW

Concept note-1: -Style defines the hue of text characters. The mini toolbar contains the Italic button. The Font Color gallery is located in the Color Group on the DESIGN tab. The layout gallery displays 10 slide layouts with a variety of placeholders to define text and content positioning and formatting.

Concept note-2: -Design Theme: A design theme is a predefined set of formatting options that control the look and feel of your presentation. Theme settings include backgrounds, layout, colors, fonts, sizes and bullets.

Concept note-3: -Italic is a format that makes the appearance of text a little slanted. Italic is a style of font that slants the letters evenly to the right.
