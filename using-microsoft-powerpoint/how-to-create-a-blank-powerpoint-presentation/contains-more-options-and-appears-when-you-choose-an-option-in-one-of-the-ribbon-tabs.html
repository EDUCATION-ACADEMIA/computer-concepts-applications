EXPLANATIONS BELOW

Concept note-1: -The Task Pane contains more options and appears when you choose an option in one of the Ribbon tabs. For example if you click the Format Background button within the Design tab of the Ribbon, the Format Background task pane opens (refer to Figure 1, shown earlier on this page).

Concept note-2: -Clicking the icon will allow you to choose from three options: hide the Ribbon completely, show only the ribbon tabs, or show the tabs and commands. If Auto-hide Ribbon is selected, the ribbon will be completely hidden. Clicking the three dots in the upper right corner of your document will restore the Ribbon view.

Concept note-3: -The Ribbon is organized into a set of task-oriented tabs, and each tab on the Ribbon contains groups of commands. The Home Tab contains the most frequently used commands in Word.

Concept note-4: -Using the Ribbon Display Options Click the Ribbon Display Options button in the lower-right corner of the ribbon. In the menu that opens, click Show Tabs and Commands to show the Ribbon with all tabs and full commands. This option is the default view.

Concept note-5: -The Page Layout Tab holds all the options that allow you to arrange your document pages just the way you want them. You can set margins, apply themes, control of page orientation and size, add sections and line breaks, display line numbers, and set paragraph indentation and lines.
