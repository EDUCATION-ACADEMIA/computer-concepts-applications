USING MICROSOFT POWERPOINT

BEGINNERS GUIDE TO USING POWERPOINT

[SOURCES]
The ____ option will copy a PowerPoint presentation and linked files onto a CD or DVD.

(A) Package for Remote Users

(B) ** Package for CD

(C) Prepare CD or DVD

(D) Save as DVD

EXPLANATIONS BELOW

Concept note-1: -In PowerPoint, click File, click Export, click Package Presentation for CD, and then click Package for CD. In the Package for CD dialog box, type a name for your CD in the Name the CD box. To add one or more presentations to package together, click Add, select the presentation, and then click Add.

Concept note-2: -When you want to run a slide show on another computer you can use the Package for CD wizard. This wizard packages together on a disk all the supporting files including any linked files. self contained CD with all the necessary files. Select (File > Package for CD).

Concept note-3: -Click File > Export > Create a Video. (Or, on the Recording tab of the ribbon, click Export to Video.) In the first drop-down box under the Create a Video heading, select the video quality you want, which pertains to the resolution of the finished video. The higher the video quality, the larger the file size.

Concept note-4: -On the File tab of the Ribbon, select Save a Copy. On the Save a Copy panel that opens, select Save a Copy. The Save a Copy dialog box opens. The default “save” location is the current folder where the original presentation is stored.
