BEGINNERS GUIDE TO USING POWERPOINT

The checkboxes next to the Ruler and Notes Pane options are selected in the Show group on the View tab. Which of the following best describes what is being shown in the program window?

(A) only the gridlines are visible in the window

(B) only the ruler is visible in the window

(C) only the Notes Pane is visible in the window

(D) ** Both the ruler and Notes Pane are visible in the window

EXPLANATIONS BELOW

Concept note-1: -The checkbox next to the ruler and Navigation Pane options are selected in the show group on the View Tab. Which of the following best describes what is being shown in the program window? It helps you quickly find commands that you need to complete a task.

Concept note-2: -If you are using the default ribbon in Word: The Navigation Pane is opened by clicking the View tab and then selecting the Navigation Pane checkbox in the Show group.

Concept note-3: -Select the File tab to access Backstage view. Select New, then click Blank document.

Concept note-4: -The View tab enables you to switch between Normal or Master Page, and Single Page or Two-Page Spread views. This tab also gives you control over showing boundaries, guides, rulers, and other layout tools, zooming the size of your view of the publication, and managing Publisher windows you have open.
