What are Hyperlinks? (Power Point)

(A) ** A tool that you use to navigate from one slide to another

(B) A tool that you use to play in Power Point

(C) A tool that you use to SAVE a file and slide

(D) A tool that you points to another image in the ppt

EXPLANATIONS BELOW

Concept note-1: -You can add text or an object as a hyperlink that opens a specific slide in a PowerPoint presentation or a custom slide show. You can also link to a Word document, an Excel workbook, a file, or an email address. Select the text or object that you want to make into a hyperlink.

Concept note-2: -Go to the Insert Ribbon and click on the Hyperlink icon in the Links Group.

Concept note-3: -In PowerPoint, you can use hyperlinks to link to resources that are not online. To create a quick way to refer to another slide in your presentation, you can create a hyperlink to that slide. If you need to access a file stored on your computer, you can create a hyperlink to it.

Concept note-4: -In PowerPoint just right-click the slide you want assistance with and choose Link to this Slide.

Concept note-5: -Go to the next slide Press N, Right arrow, Down arrow, or Spacebar. Mouse: Click once anywhere on a slide to move to the next one.
