Which of the following statements is true about the header and footer formatting that is done on the Slide tab in the Header and Footer dialog box?

(A) ** They are applied automatically to every slide

(B) Only the header is automatically applied to every slide

(C) Only the footer is automatically applied to every slide

(D) None are automatically applied to every slide

EXPLANATIONS BELOW

Concept note-1: -Detailed Solution The correct answer is You can set different header and footer for the last page of a section. A header is the top margin of each page, and a footer is the bottom margin of each page.

To do so: Insert section breaks; see ARCHIVED: In Microsoft Word, what are sections?

Concept note-3: -You choose which headers and footers to show, and what the text footer says. To insert headers and footers, you click INSERT, Header & Footer, make sure Slide is selected, and choose your options. These include Date and time, a date and time format, and whether to show the current date or a Fixed one.
