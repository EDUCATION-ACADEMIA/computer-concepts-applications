USING MICROSOFT POWERPOINT

BEGINNERS GUIDE TO USING POWERPOINT

[SOURCES]
Action buttons cannot do the same things as hyperlinks.

(A) ** FALSE

(B) TRUE

EXPLANATIONS BELOW

Concept note-1: -Answer: Answer: true, Action buttons can do many of the same things as hyperlinks. Explanation: Action buttons are built-in button shapes you can add to a presentation and set to link to another slide, play a sound, or perform a similar action.

Concept note-2: -Action buttons cannot do the same things as hyperlinks.

Concept note-3: -Action buttons are built-in button shapes you can add to a presentation and use as hyperlinks. When someone clicks or moves over the button, the action can occur. Hyperlinks and action buttons are closely connected and can do many of the same things.

Concept note-4: -Highlight the text or object you would like to hyperlink. Click the [Insert] tab > From the “Links” group, click [Action]. Select the “Hyperlink to:” radio button > From the drop-down menu, choose where you would like the action button to link to when activated. Click [OK].

Concept note-5: -Use an Action button to do an action in your presentation, such as going to the next or a specific slide, running an app, or playing a video clip.
