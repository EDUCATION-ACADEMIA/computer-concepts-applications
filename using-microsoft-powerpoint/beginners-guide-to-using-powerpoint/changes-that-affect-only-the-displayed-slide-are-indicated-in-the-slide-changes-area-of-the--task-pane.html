USING MICROSOFT POWERPOINT

BEGINNERS GUIDE TO USING POWERPOINT

[SOURCES]
Changes that affect only the displayed slide are indicated in the Slide changes area of the ____ task pane.

(A) Notes

(B) Reviewers

(C) ** Revisions

(D) Replacements

EXPLANATIONS BELOW

Concept note-1: -A transition is applied to an entire slide. Only one transition effect can be applied to a slide. To use a transition effect, select the slide you want to transition to, then apply an effect. In your slide show, the effect will appear when the previous slide gives way to the slide that has the transition effect.

Concept note-2: -The Notes pane is located beneath the slide window.

Concept note-3: -This view allows easy navigation through slides. Slide pane contains the current slide in your presentation. You can use the vertical scroll bar to view other slides in the presentation. Notes pane is located below the slide pane and is used to type reference notes.

Concept note-4: -Any changes you make to the slide master automatically affect all the slides in the presentation; for example, changing the background color of the slide master changes the background for every single one of your slides.
