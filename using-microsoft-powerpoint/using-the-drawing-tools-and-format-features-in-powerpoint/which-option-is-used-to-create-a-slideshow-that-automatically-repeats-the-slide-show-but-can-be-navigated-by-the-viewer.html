USING MICROSOFT POWERPOINT

USING THE DRAWING TOOLS AND FORMAT FEATURES IN POWERPOINT

[SOURCES]
Which option is used to create a slideshow that automatically repeats the slide show but can be navigated by the viewer?

(A) Presented by a speaker (full screen)

(B) Browsed by an individual (window)

(C) ** Browsed at a kiosk (full screen)

EXPLANATIONS BELOW

Concept note-1: -Click the “Set Up Slide Show” button. 3. Check the “Loop continuously until ‘Esc”’ option .

Concept note-2: -Open your PowerPoint presentation. Click the [Slide Show] tab > From the “Set Up” group, click “Set Up Slide Show". From the resulting dialogue box, check “Loop continuously until ‘Esc”’ under the “Show options” section > Click [OK].

Concept note-3: -PowerPoint Kiosk mode. At its basic level, PowerPoint’s Kiosk mode is merely a different way to display a presentation. Instead of clicking to move between slides, PowerPoint will automatically display the slides going in an infinite loop … until someone presses Escape.

Concept note-4: -The Kiosk Mode automatically steps through all slides in your presentation at a fixed interval. This view is useful when you want to display an infinitely looping deck on digital signage or on a kiosk display.

Concept note-5: -To start a slide show in PowerPoint for the web, open the file, and on the Slide Show tab of the ribbon, select Play from Beginning.
