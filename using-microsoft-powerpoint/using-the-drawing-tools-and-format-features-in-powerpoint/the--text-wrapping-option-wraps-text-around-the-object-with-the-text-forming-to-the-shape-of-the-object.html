USING MICROSOFT POWERPOINT

USING THE DRAWING TOOLS AND FORMAT FEATURES IN POWERPOINT

[SOURCES]
The ____ text wrapping option wraps text around the object, with the text forming to the shape of the object.

(A) shape

(B) cover

(C) ** tight

(D) contour

EXPLANATIONS BELOW

Concept note-1: -"Wrapping text” means displaying the cell contents on multiple lines, rather than one long line. This will allow you to avoid the “truncated column” effect, make the text easier to read and better fit for printing.

Concept note-2: -Go to Picture Format or Shape Format and select Arrange > Wrap Text.

Concept note-3: -Select the image you want to wrap text around. The Format tab will appear on the right side of the Ribbon. On the Format tab, click the Wrap Text command in the Arrange group, then select the desired text wrapping option. In our example, we’ll select In Front of Text so we can freely move it without affecting the text.

Concept note-4: -Tight. Wrapping text in Word using the Tight option is also similar to the Square option. While using the Tight option, the image becomes a square and the text is wrapped evenly around it. If you use the Tight option, the text around the image is tightly packed and unevenly wrapped around the image.
