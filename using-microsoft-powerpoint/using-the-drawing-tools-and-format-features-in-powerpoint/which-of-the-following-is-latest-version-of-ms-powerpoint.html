Which of the following is latest version of MS-PowerPoint?

(A) PowerPoint 2000

(B) PowerPoint 2002

(C) PowerPoint ME

(D) ** PowerPoint Xp

EXPLANATIONS BELOW

Concept note-1: -2012 PowerPoint 2013 (version 15) (Office 2013) 2015 PowerPoint 2016 (version 16) (Office 2016) 2018 PowerPoint 2019 (version 16) (Office 2019) 2021 PowerPoint 2021 (version 16) (Office 2021)

Concept note-2: -July 1995: PowerPoint 95 (new version naming) To align PowerPoint with all other Office applications, Microsoft decided to skip versions 5 and 6 and instead call it PowerPoint 95.

Concept note-3: -Answer. Answer: Correct answer: MS PowerPoint 1920 is not a valid edition of Microsoft PowerPoint.

Concept note-4: -Check for PowerPoint Updates In Microsoft PowerPoint you can easily check for updates using the option under File-> Account. Clicking on Update Options button, you can see different Update options available.
