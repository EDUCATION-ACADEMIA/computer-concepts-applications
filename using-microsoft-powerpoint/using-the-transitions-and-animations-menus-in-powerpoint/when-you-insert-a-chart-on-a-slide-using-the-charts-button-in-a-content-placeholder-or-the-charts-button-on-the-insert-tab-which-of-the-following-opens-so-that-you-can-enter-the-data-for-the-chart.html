USING MICROSOFT POWERPOINT

USING THE TRANSITIONS AND ANIMATIONS MENUS IN POWERPOINT

[SOURCES]
When you insert a chart on a slide using the Charts button in a content placeholder or the Charts button on the Insert tab, which of the following opens so that you can enter the data for the chart?

(A) ** An Excel worksheet

(B) A PowerPoint table

(C) A hyperlink to an Excel worksheet

(D) A legend

EXPLANATIONS BELOW

Concept note-1: -PowerPoint uses an Excel worksheet as a placeholder for entering chart data. Therefore, when you insert or edit a chart in PowerPoint, an Excel window will automatically open.

Concept note-2: -On the Insert tab, in the Illustrations group, click Chart. In the Insert Chart dialog box, click a chart, and then click OK.

Concept note-3: -How are charts inserted into a PowerPoint slide? Use the Copy tool to copy the chart from Excel and the Paste tool to insert it into the slide.

Concept note-4: -In the Insert Chart window, select the type of chart you want to add, and then click OK. The selected type of chart is added to your slide, including sample data. The data used to build the chart is displayed in the Chart in Microsoft PowerPoint window, which looks like an Excel spreadsheet.
