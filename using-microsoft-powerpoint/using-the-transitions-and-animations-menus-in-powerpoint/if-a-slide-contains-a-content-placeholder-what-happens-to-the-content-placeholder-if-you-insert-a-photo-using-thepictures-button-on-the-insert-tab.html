USING MICROSOFT POWERPOINT

USING THE TRANSITIONS AND ANIMATIONS MENUS IN POWERPOINT

[SOURCES]
If a slide contains a content placeholder, what happens to the content placeholder if you insert a photo using thePictures button on the Insert tab?

(A) ** The content placeholder is replaced by the picture.

(B) The content placeholder is moved so that it is next to the picture.

(C) The content placeholder disappears while the picture is selected, but reappears when the picture is no longer selected.

(D) The content placeholder stays on the slide behind the picture.

EXPLANATIONS BELOW

Concept note-1: -If a slide contains a content placeholder, what happens to the content placeholder if you insert a photo using the Pictures button on the Insert tab? The content placeholder is replaced by the picture.

Concept note-2: -On the Slide Master tab, click Insert Placeholder, and then click the type of placeholder that you want to add. Click a location on the slide layout, and then drag to draw the placeholder. You can add as many placeholders as you like.

Concept note-3: -Click on the Insert Placeholder button and a new popup will appear. Now, you can insert a picture placeholder but also some other content types, for example a table, a smartart graphic, a media content, clipart or even text or content.

Concept note-4: -A placeholder is a container that is used to display content, such as text, table, picture, movies, sound, clip art, chart, SmartArt etc. A placeholder can be resized, moved and edited. In a PowerPoint placeholders are displayed in the form of a dotted rectangular box and are found in all the built-in slide layouts.

Concept note-5: -To insert a picture Placeholder in a Slide layout, open the View tab. Then, select the “Slide Master view” from the “Master Views” group. Now, click on the “Slide Layout” to insert the picture placeholder. From the menu, select the “Picture” option and draw the same into the slide.
