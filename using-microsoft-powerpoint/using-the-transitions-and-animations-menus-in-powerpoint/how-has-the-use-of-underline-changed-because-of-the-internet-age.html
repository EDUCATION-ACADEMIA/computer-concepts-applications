How has the use of underline changed because of the Internet age?

(A) People are used to seeing underlined words all the time.

(B) ** Most people assume an underlined word is a hyperlink to a website.

(C) All underlined words are hyperlinks.

(D) There is no difference at all. People see underlined words as placing extra emphasis on the word.

EXPLANATIONS BELOW

Concept note-1: -How has the use of underline changed because of the Internet age? People are used to seeing underlined words all the time. Most people assume an underlined word is a hyperlink to a website. All underlined words are hyperlinks.

Concept note-2: -Websites use hyperlinks as a way to navigate online content. Hyperlinks can point to web content that is part of that website, or it can point to web content that is part of external websites. Both images and text can be used to create a hyperlink.
