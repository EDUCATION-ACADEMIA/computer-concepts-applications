What is NOT a font effect?

(A) Shadow

(B) WordArt

(C) ** Regular Black Font

(D) Underline

EXPLANATIONS BELOW

Concept note-1: -Superscript is not a font style.

Concept note-2: -Detailed Solution Regular, Bold, and Italic are examples of font styles.

Concept note-3: -Font styles are predefined formatting options that are used to emphasize text. They include: Bold, Italic, and Underline. To add these to text: Select the text and click the Font Styles included on the Font group of the Home tab or. Select the text and right-click to display the font tools.
