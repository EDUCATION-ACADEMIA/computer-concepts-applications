USING MICROSOFT POWERPOINT

USING THE TRANSITIONS AND ANIMATIONS MENUS IN POWERPOINT

[SOURCES]
You can apply more than one animation to an object.

(A) ** True

(B) False

EXPLANATIONS BELOW

Concept note-1: -You can have two or more animation effects occur at the same time for one object. For example, a picture could fade in and float in. To accomplish this layering of two (or more) effects at once, do as follows: Use Add Animation to add the second effect.

Concept note-2: -Answer. so the answer is true !!!

Concept note-3: -An object can have only one animation applied to it. When you type a web or an email address on a slide, you must format it as a hyperlink manually. Text links are usually underlined and the same color as the rest of the text on a slide.

Concept note-4: -Repeat or rewind You can set an animation effect (or sequence of effects) to return to its original state, or repeat. Click the Timing tab, and do one or both of the following: To play an animation effect more than once, select an option in the Repeat list.

Concept note-5: -You can add an animation to grouped objects, text, and more. Press Ctrl and select the objects you want. Select Format > Group > Group to group the objects together. Select Animations and choose an animation.
