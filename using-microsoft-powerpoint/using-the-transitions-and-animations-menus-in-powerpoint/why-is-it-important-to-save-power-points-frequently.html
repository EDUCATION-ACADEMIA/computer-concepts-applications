USING THE TRANSITIONS AND ANIMATIONS MENUS IN POWERPOINT

[SOURCES]
why is it important to save power points frequently?

(A) so you won’t lose your work

(B) so you can come back to your work

(C) so you can work on it anywhere

(D) ** all of the above

EXPLANATIONS BELOW

Concept note-1: -It’s important to save your presentation whenever you start a new project or make changes to an existing one. Saving early and often can prevent your work from being lost. You’ll also need to pay close attention to where you save the presentation so it will be easy to find later.

Concept note-2: -After you save your presentation, you can keep working on it, if you want. But, remember to save it often, so you don’t lose any new work. If you want to print your presentation, click the FILE tab to go to the backstage, and click Print.

Concept note-3: -It is used to make presentations for personal and professional purposes.

Concept note-4: -Making your presentation more interesting through the use of multimedia can help to improve the audience’s focus. PowerPoint allows you to use images, audio and video to have a greater visual impact. These visual and audio cues may also help a presenter be more improvisational and interactive with the audience.
