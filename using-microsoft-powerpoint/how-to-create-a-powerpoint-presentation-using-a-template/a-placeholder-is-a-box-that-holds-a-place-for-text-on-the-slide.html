USING MICROSOFT POWERPOINT

HOW TO CREATE A POWERPOINT PRESENTATION USING A TEMPLATE

[SOURCES]
A placeholder is a box that holds a place for text on the slide.

(A) ** True

(B) False

EXPLANATIONS BELOW

Concept note-1: -A placeholder is a box that holds a place for text on the slide. Once you select a slide layout, it cannot be changed.

Concept note-2: -A placeholder is a container that is used to display content, such as text, table, picture, movies, sound, clip art, chart, SmartArt etc. A placeholder can be resized, moved and edited. In a PowerPoint placeholders are displayed in the form of a dotted rectangular box and are found in all the built-in slide layouts.

Concept note-3: -Expert-Verified Answer A placeholder is box wherein you can add content like text, table, picture, clipart, smart art etc. It is present as a built-in feature in every PowerPoint slide. In order to insert text in the placeholder, you just need to click inside it and then begin writing.

Concept note-4: -In PowerPoint, a placeholder is a pre-formatted container on a slide for content (text, graphics, or video). The pre-set formatting makes it easier to format slides consistently. You format a placeholder in Slide Master view.

Concept note-5: -Answer. Explanation: All slide layouts (except for the Blank layout) contain placeholders to help control the position of objects on a slide. Placeholders can hold clip art, pictures, charts, text and more.
