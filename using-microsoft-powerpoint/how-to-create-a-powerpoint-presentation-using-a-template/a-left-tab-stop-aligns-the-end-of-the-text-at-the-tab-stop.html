HOW TO CREATE A POWERPOINT PRESENTATION USING A TEMPLATE

[SOURCES]
a left tab stop aligns the end of the text at the tab stop.

(A) True

(B) ** False

EXPLANATIONS BELOW

Concept note-1: -A left tab stop sets the start position of text that will run to the right as you type. A center tab stop sets the position at the middle of the text. The text centers on this position as you type. A right tab stop sets the right end of the text.

Concept note-2: -Types of tab stops: Left Tab left-aligns the text at the tab stop. Center Tab centers the text around the tab stop. Right Tab right-aligns the text at the tab stop.

Concept note-3: -Select Left Tab A Left tab stop sets the left end of the text line. As you type, the text fills toward the right. A Center tab stop sets the position at the middle of the text line. As you type, the text centers on this position.
