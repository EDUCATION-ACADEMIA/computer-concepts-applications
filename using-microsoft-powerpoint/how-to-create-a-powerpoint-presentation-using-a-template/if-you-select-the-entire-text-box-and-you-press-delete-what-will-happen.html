HOW TO CREATE A POWERPOINT PRESENTATION USING A TEMPLATE

[SOURCES]
If you select the entire text box and you press Delete, what will happen?

(A) It will delete the text only

(B) it will delete the single character to the right of the insertion point

(C) ** It will delete the entire text box and everything in it

EXPLANATIONS BELOW

Concept note-1: -Normally, you cannot delete all the text within text boxes. If you select all the text boxes in a document and press the Del key, the text boxes themselves will be deleted, along with the text inside. If you press Ctrl+A (or choose Select All from the Edit menu) and press Del, everything in your document disappears.

Concept note-2: -When the delete option is selected, PowerPoint shapes remain on the slide and will retain their original appearance, but without any text. However Text boxes disappear if the delete option is selected where there is not additional formatting on the textbox (for example color fill or margin).

Concept note-3: -Click the border of the text box that you want to delete, and then press DELETE. Make sure that the pointer is not inside the text box, but rather on the border of the text box. If the pointer is not on the border, pressing DELETE will delete the text inside the text box and not the text box.

Concept note-4: -Select the border of the text box that you want to delete, and then press Delete. Note: Make sure the pointer is on the border of the text box and not inside it. If the pointer is inside the box, pressing Delete will delete the text inside the text box, not the box.
