USING MICROSOFT POWERPOINT

HOW TO MODIFY YOUR POWERPOINT PRESENTATIONS THEME AND BACKGROUND

[SOURCES]
To add another slide to my presentation, we have to use

(A) ** New Slide

(B) Themes

(C) Insert picture

EXPLANATIONS BELOW

Concept note-1: -Click on the New Slide button located on the Home tab in the Slides group. The new slide should appear below the selected slide. If you want to add a new slide between two, select the first of the two slides and click New Slide. Notice the New Slide button contains a New Slide button arrow.

Concept note-2: -Select Slides from Outline. Find and select the Word document in the Insert Outline dialog box, and click Insert. A new slide will be created for each Heading 1 in you document.

Concept note-3: -While you rehearse, use the Slide Timing feature to record the time that you need to present each slide, and then use the recorded times to advance the slides automatically when you give your presentation to your actual audience. The Slide Timing feature is ideal for creating a self-running presentation.
