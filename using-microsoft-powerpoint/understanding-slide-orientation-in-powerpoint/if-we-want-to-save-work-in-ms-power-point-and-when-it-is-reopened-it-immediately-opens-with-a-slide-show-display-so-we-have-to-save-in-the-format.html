USING MICROSOFT POWERPOINT

UNDERSTANDING SLIDE ORIENTATION IN POWERPOINT

[SOURCES]
If we want to save work in MS. Power Point and when it is reopened it immediately opens with a slide show display, so we have to save in the format____

(A) Other Formats

(B) ** Power Point Show

(C) Power Point Slide Show

(D) Powerpoint Presentation

(E) Power Point 97-2003 Presentation

EXPLANATIONS BELOW

Concept note-1: -In PowerPoint, click File > Save as. Browse to the folder where you want to save your presentation. In the File name box, type a name for your presentation. Under Save as type, select PowerPoint Show.

Concept note-2: -Select File > Options. Select Save. Under Save presentations, in the Save files in this format list, choose the file format you want to use by default.

Concept note-3: -The commands for saving, opening, and closing presentations are on the File menu. To save a new presentation: Choose File > Save As (Ctrl+S), or click the Save button on the toolbar. With a new, untitled presentation, Save and Save As function the same.
