USING MICROSOFT POWERPOINT

UNDERSTANDING SLIDE ORIENTATION IN POWERPOINT

[SOURCES]
Name the sub-tool, which you can use to fill bricks in the picture

(A) haze tool

(B) grass tool

(C) ** bricks tool

EXPLANATIONS BELOW

Concept note-1: -Name the sub-tool using which you can fill bricks in the picture. Explanation : Tux paint is a drawing software program in which many attractive designs can be made. This program is used to draw and fill color etc.

Concept note-2: -Brick trowel: or mason’s trowel is a point-nosed trowel for spreading mortar on bricks or concrete blocks with a technique called “buttering".

Concept note-3: -The ‘Fill’ tool ‘flood-fills’ a contiguous area of your drawing with a color of your choice. Three fill options are offered: Solid-click once to fill an area with a solid color. Brush-click and drag to fill an area with a solid color using freehand painting.

Concept note-4: -What does the Magic Wand tool do? The Magic Wand tool automatically selects an object or area of your image, using machine learning to discern what you might be trying to select. If you click on the sky, it selects the sky; click on a person’s face, and you have the entire face selected in just seconds.

Concept note-5: -Magic Tool Ans: It is used to add special effects to a drawing.
