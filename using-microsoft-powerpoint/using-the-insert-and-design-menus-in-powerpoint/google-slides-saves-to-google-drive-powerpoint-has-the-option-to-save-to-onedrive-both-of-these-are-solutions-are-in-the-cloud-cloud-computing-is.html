Google Slides saves to Google Drive, Powerpoint has the option to save to OneDrive. Both of these are solutions are in the cloud. Cloud Computing is:

(A) A remote server that is for the use of students and staff of your district

(B) ** Services or programs located on the internet

(C) A network of computers in the same room, connected with cords to each other.

(D) A network of computers in the same company, connected by software.

EXPLANATIONS BELOW

Concept note-1: -Cloud computing are services or programs located on the Internet, they are usually remote resources making use of cloud storage. For example, cloud computing includes; -Google Slides saves your work to Google Drive.-PowerPoint has the option to save to OneDrive.
