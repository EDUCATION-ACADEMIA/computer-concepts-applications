USING MICROSOFT POWERPOINT

USING THE INSERT AND DESIGN MENUS IN POWERPOINT

[SOURCES]
PowerPoint provides you with a gallery of preset animations, which are divided into different categories. Choosing ____ will automatically apply different styles.

(A) Dissolve

(B) Chance

(C) Fade

(D) ** Random

EXPLANATIONS BELOW

Concept note-1: -The preset animation is applied to the text or object and a preview of the effect is displayed. A number appears next to the object showing the order in which the animation will be displayed.

Concept note-2: -There are four types of animation effects in PowerPoint – entrance, emphasis, exit and motion paths. These reflect the point at which you want the animation to occur.

Concept note-3: -Preset Animations Schemes are available in the Animation group of the Animations Tab.

Concept note-4: -Select chart. “Animations” tab and select the desired animation. Effect options. Select the desired option (by series, by category, by element in series or by element in category.
