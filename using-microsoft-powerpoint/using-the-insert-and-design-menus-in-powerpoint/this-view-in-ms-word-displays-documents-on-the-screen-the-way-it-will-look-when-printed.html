USING MICROSOFT POWERPOINT

USING THE INSERT AND DESIGN MENUS IN POWERPOINT

[SOURCES]
This view in MS Word displays documents on the screen the way it will look when printed.

(A) Read Mode View

(B) ** Print View

(C) Outline View

(D) Draft View

EXPLANATIONS BELOW

Concept note-1: -Print Layout view. This view displays a document on the screen the way it will look when printed. You can see elements such as margins, page breaks, headers and footers, and watermarks.

Concept note-2: -Print Layout Your document opens showing the entire page, just the way it will appear when it is printed on a sheet of paper.

Concept note-3: -Print preview is a feature that displays on the screen what a hard copy would look like when printed. Using print preview, you can find any errors that may exist or fix the layout before printing, which can save ink or toner and paper by not having to print more than once.

Concept note-4: -In Reading view, Word for the web displays a document as it looks in Print Layout view. Other views available in the Word desktop app (Outline, Draft, Web Layout, and Full Screen Reading) are not available in Word for the web. Likewise, side-by-side viewing and split windows are not available in Word for the web.

Concept note-5: -Full Screen Reading view is optimized for reading a document on the computer screen. In Full Screen Reading view, you also have the option of seeing the document as it would appear on a printed page.
