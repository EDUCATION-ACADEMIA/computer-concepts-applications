USING MICROSOFT EXCEL

HOW TO MODIFY THE WORKSHEET TAB IN EXCEL

[SOURCES]
In order to change the printer to which the workbook should be set, which tab should you go to?

(A) ** File

(B) Home

(C) Review

(D) View

EXPLANATIONS BELOW

Concept note-1: -Select File > Print, or press CTRL+P. Select the Print button or adjust Settings before you select the Print button.

Concept note-2: -You also learned that there are seven major settings that can be modified to help change the layout of your printout, which included the page orientation, margins, and scaling. It may take a bit of practice, but next time you need to print from Excel, remember to preview and adjust your settings.

Concept note-3: -To print the entire workbook, before you click Print, under Settings, click the arrow next to Print Active Sheets, click Print Entire Workbook.

Concept note-4: -Click the File tab and then the Save As button in the left side of the Backstage view window. This will open the Save As dialog box. Determine a location for saving on your computer by clicking Browse on the left side to open the Save As dialog box.
