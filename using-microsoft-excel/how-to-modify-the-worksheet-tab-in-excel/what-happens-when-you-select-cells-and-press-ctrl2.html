What happens when you select cells and press Ctrl+2?

(A) Text in the selected cells are underlined.

(B) ** Bold formatting is applied to the text in the selected cells.

(C) Italic formatting is applied to the text in the selected cells.

(D) A duplicate of the selected cells will be inserted directly below the selection.

EXPLANATIONS BELOW

Concept note-1: -4) The correct answer is b. Using Ctrl+2 you can quickly apply bold formatting to a selected cell, row or column. It is also possible to use the more traditional Office shortcut of Ctrl+B to achieve the same result.

Concept note-2: -In Microsoft Excel and all other spreadsheet programs, pressing Ctrl + 2 toggles the bold text feature on and off for a cell.

Concept note-3: -Click Bold. in the Font group on the Home tab. Type the keyboard shortcut: CTRL+B.
