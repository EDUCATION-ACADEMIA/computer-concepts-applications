USING MICROSOFT EXCEL

HOW TO MODIFY THE WORKSHEET TAB IN EXCEL

[SOURCES]
Which command can you use to hide column or row data that doesn’t match the defined criteria?

(A) Sort

(B) ** Filter

(C) Conditional Formatting

(D) Custom Sort

EXPLANATIONS BELOW

Concept note-1: -Press Ctrl + 0 to hide the selected columns. Done!

Concept note-2: -Right click on the column you want to hide and then click “Hide.” You can hide multiple columns this way if you have them all selected. One last look at the data set. If you want to see the hidden information again, simply right click on the space the column should be and click “Unhide.”

Concept note-3: -One way is to use the built-in filter feature. To do this, first select the data that you want to filter. Then, click the Data tab on the ribbon and click the Filter button. In the drop-down menu that appears, click the column that you want to filter by and then uncheck the box next to the value that you want to hide.

Concept note-4: -On the Home tab, in the Cells group, click Format. Do one of the following: Under Visibility, click Hide & Unhide, and then click Unhide Rows or Unhide Columns.

Concept note-5: -If you want to hide rows based on cell values, you can do so by using a conditional formatting rule. To do this, select the rows you want to hide and then go to the Home tab > Styles group > Conditional Formatting > Highlight Cell Rules > A Date Occurring.
