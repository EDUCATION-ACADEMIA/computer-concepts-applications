(A) Column Styles

(B) ** cell styles

EXPLANATIONS BELOW

Concept note-1: -A cell style is a defined set of formatting characteristics, such as fonts and font sizes, number formats, cell borders, and cell shading. To prevent anyone from making changes to specific cells, you can also use a cell style that locks cells.

Concept note-2: -A style is a set of formatting characteristics, such as font name, size, color, alignment, and spacing. When you want to apply more than one format at a time to a cell or range of cells, it is easiest to apply a style. Styles also help you keep formatting consistent throughout a workbook.
