What version of Excel are we using in class?

(A) ** Excel 2016

(B) Excel 2017

(C) Excel 2018

(D) Excel 2015

EXPLANATIONS BELOW

Concept note-1: -The version number will be “12.0” for Excel 2007, “14.0” for Excel 2010, “15.0” for Excel 2013, or “16.0” for Excel 2016 and later versions.

Concept note-2: -Excel 2013, 2016, 2019, 2021 (Microsoft 365) Start by clicking on the File button, on the top left corner of Excel. Click on Account, on the left-hand side of the screen, then About Excel. The version is visible in the first paragraph of the dialog box that appears.

Concept note-3: -Excel 2016 and 2019 differences You will discover various differences in Excel 2016 and Excel 2019. For example, in Excel 2019 it is possible to use various new or improved functions such as MAXIFS, TEXTJOIN, CONCAT and SWITCH.

Concept note-4: -Aside from superficial color and labeling, Microsoft Excel 2016 isn’t much different from Excel 2013 or 2010. All the menu items and often-used features are the same.
