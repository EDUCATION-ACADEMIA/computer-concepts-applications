USING MICROSOFT EXCEL

HOW TO ADD COPY AND MOVE WORKSHEETS WITHIN EXCEL WORKBOOKS

[SOURCES]
returns the cell pointer to cell A1

(A) ** ctrl+home

(B) shift+home

(C) ctrl+end

(D) alt+home

EXPLANATIONS BELOW

Concept note-1: -If you want to quickly jump to cell A1 from anywhere of the sheet, you just need to hold Ctrl key, and press Home, then the cursor jumps to the cell A1 at once.

Concept note-2: -Move and scroll within worksheets: HOME Move to the beginning of the row. CTRL+HOME Move to the beginning of the worksheet. CTRL+END Move to the last cell on the worksheet, in the bottom-most used row of the rightmost used column. PAGE DOWN Move down one screen.

Concept note-3: -Detailed Solution. The correct answer is Ctrl + Home. Ctrl + Home is used to move the active cell to the cell A1 in MS-Excel. Alternatively referred to as Control+Home and C-Home, Ctrl+Home is a keyboard shortcut that moves the cursor to the beginning of a document.

Concept note-4: -The easiest way to go back to the A1 cell may the shortcut keys, please do as this: Hold down the Ctrl + Home keys together as following screenshot shown, the cursor will be jump to the cell A1 from anywhere of the worksheet.
