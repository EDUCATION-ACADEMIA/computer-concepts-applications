What is it called when you view some records and hide others?

(A) Report

(B) Sort

(C) Search

(D) ** Filter

EXPLANATIONS BELOW

Concept note-1: -A primary key field from one table appears in a second table. What is the field in the second table called? Filter. What is it called when you view some records and hide others? Search.

Concept note-2: -Filtering is a useful way to see only the data that you want displayed in Access databases. You can use filters to display specific records in a form, report, query, or datasheet, or to print only certain records from a report, table, or query.

Concept note-3: -You sort data to quickly organize your data and to find the data that you want. You filter data to display only the rows that meet criteria that you specify and hide rows that you do not want displayed, for one or more columns of data.

Concept note-4: -To filter data in a query, open it in Datasheet View, click the down-arrow at the top of a column, and select a filter option.
