The cell reference for the sixth row and fourth column is:Select the BEST possible answer.

(A) F4

(B) ** D6

(C) F6

(D) D4

EXPLANATIONS BELOW

Concept note-1: -Expert-Verified Answer By taking the row number and column letter, the cell ADDRESS will provide a cell reference (its “address"). The cell reference will be given as a text string. The cell produced by the intersection of the 4th row and 6th column has the Cell Address F4.

Concept note-2: -"C6, the address of the cell at the interaction of 3rd column & 6th in MS Excel". Explanation : In MS Excel, the row is named in numerical order 1, 2, 3, 4.. and the column is named in alphabetical order A, B, C, D, and so on.

Concept note-3: -A cell range is specified in a spreadsheet by the reference of the upper left cell (minimum value) and the reference of the lower right cell (highest value). The excel range notation A1:D6 denotes from upper left cell A1 to bottom right cell D6.

Concept note-4: -In each cell there is an address of the cell the address of cell is started from A1 the address of first row and first column . That’s why the address of f 4th row and 4th column is D4.
