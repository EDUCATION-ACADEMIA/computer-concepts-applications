USING MICROSOFT EXCEL

HOW TO ADD COPY AND MOVE WORKSHEETS WITHIN EXCEL WORKBOOKS

[SOURCES]
Text entered in a worksheet that is used to identify numeric data.

(A) ** Label

(B) Keywords

(C) Excel Words

(D) Cell Names

EXPLANATIONS BELOW

Concept note-1: -By default, Excel uses the A1 reference style, which refers to columns as letters (A through IV, for a total of 256 columns), and refers to rows as numbers (1 through 65, 536). These letters and numbers are called row and column headings.

Concept note-2: -Entering data into a spreadsheet is just like typing in a word processing program, but you have to first click the cell in which you want the data to be placed before typing the data. All words describing the values (numbers) are called labels. The numbers, which can later be used in formulas, are called values.

Concept note-3: -In Microsoft Excel, you can enter numbers as labels or as values. Labels are alphabetic, alphanumeric, or numeric text on which you do not perform mathematical calculations. Values are numeric text on which you perform mathematical calculations.

Concept note-4: -A label identifies the purpose of a cell or text box, displays brief instructions, or provides a title or caption.
