A formatting tool that appears above or below the shortcut menu when you right-click a cell and that displays the most commonly used formatting commands

(A) rules manager

(B) wrap

(C) style

(D) ** mini toolbar

EXPLANATIONS BELOW

Concept note-1: -The mini toolbar is a context toolbar that is often displayed when you use the right mouse button. It provides access to some of the most common formatting commands and is displayed above the shortcut drop-down menu. An example of a mini toolbar can been seen by selecting any cell and clicking the right mouse button.

Concept note-2: -Mini Toolbar in Excel It is a floating toolbar that appears above the shortcut menu when you right click a cell or other objects like a shape or chart. It displays commonly used formatting commands like Bold, Italics, Font Size and Font Color.

Concept note-3: -It is a small version of the full toolbar that displays commonly used formatting commands like Bold, Italics, Font Size and Font Color. The mini toolbar was introduced as a quick-access version so the Excel users can easily access the Font group, border, and few extra buttons on the Home tab.

Concept note-4: -The formatting toolbar provides access to several text formatting functions such as font size and color, text alignment, lists, and the like. Applies the selected font to the text. Applies the selected font size to the text.
