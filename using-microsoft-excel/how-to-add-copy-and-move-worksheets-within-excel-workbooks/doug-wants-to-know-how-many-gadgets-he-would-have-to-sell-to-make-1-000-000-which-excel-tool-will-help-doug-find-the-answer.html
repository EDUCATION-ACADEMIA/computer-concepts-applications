Doug wants to know how many gadgets he would have to sell to make $1, 000, 000. Which Excel tool will help Doug find the answer?

(A) Data Validation

(B) Existing Connections

(C) ** Goal Seek

(D) Scenario Manager

EXPLANATIONS BELOW

Concept note-1: -Goal Seek runs and produces a result, as shown in the following illustration. Cells B1, B2, and B3 are the values for the loan amount, term length, and interest rate. Cell B4 displays the result of the formula =PMT(B3/12, B2, B1). Finally, format the target cell (B3) so that it displays the result as a percentage.

Concept note-2: -1) Solver can solve formulas (or equations) which use several variables whereas Goal Seek can only be used with a single variable. 2) Solver will allow you to vary the values in up to 200 cells whereas Goal Seek only allows you to vary the value in one cell.

Concept note-3: -Go to the Data tab > Forecast group, click the What if Analysis button, and select Goal Seek… In the Goal Seek dialog box, define the cells/values to test and click OK: Set cell-the reference to the cell containing the formula (B5). To value-the formula result you are trying to achieve (1000).

Concept note-4: -The Goal Seek function in Excel allows you to adjust an input value in a formula to determine a desired outcome. This What-If Analysis tool is ideal for situations where you know the outcome you want, but aren’t sure of the values needed to reach that outcome.
