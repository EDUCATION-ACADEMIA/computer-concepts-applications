USING MICROSOFT EXCEL

HOW TO ADD COPY AND MOVE WORKSHEETS WITHIN EXCEL WORKBOOKS

[SOURCES]
A(n) ____ is a collection of coordinated fonts, colors, and effects that can be quickly applied to all sheets in a workbook.

(A) ** THEME

(B) NON-THEME

(C) LESSON

(D) FILL

EXPLANATIONS BELOW

Concept note-1: -Themes in Excel are inbuilt and customizable functions that help change the look or formatting of the entire workbook. Themes are created with a collection of colors, fonts, and shape effects.

Concept note-2: -In MS-Excel, a collection of worksheets is called a workbook.

Concept note-3: -Themes are a combination of colors, fonts and effects that can be applied to worksheets. You can select from the predefined, built-in themes, or you can get creative and modify existing theme elements.

Concept note-4: -To switch to another theme, click Page Layout > Themes, and pick the one you want. To customize that theme, you can change its colors, fonts, and effects as needed, save them with the current theme, and make it the default theme for all new workbooks if you want.
