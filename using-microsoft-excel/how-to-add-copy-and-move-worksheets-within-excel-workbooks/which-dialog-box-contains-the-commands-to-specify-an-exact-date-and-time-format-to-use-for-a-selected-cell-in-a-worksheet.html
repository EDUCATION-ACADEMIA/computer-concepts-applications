HOW TO ADD COPY AND MOVE WORKSHEETS WITHIN EXCEL WORKBOOKS

Which dialog box contains the commands to specify an exact date and time format to use for a selected cell in a worksheet?

(A) ** Format cells

(B) Number

EXPLANATIONS BELOW

Concept note-1: -To change the date or time format, right-click on a cell, and select Format Cells. Then, on the Format Cells dialog box, in the Number tab, under Category, click Date or Time and in the Type list, select a type, and click OK.

Concept note-2: -On the Home tab, in the Number group, click the Dialog Box Launcher next to Number. You can also press CTRL+1 to open the Format Cells dialog box. In the Category list, click Date or Time. In the Type list, click the date or time format that you want to use.

Concept note-3: -Ctrl +1 or ! is a shortcut key in the Microsoft Excel 2007 to display the format cell dialog box. The Format Cells Dialog Box is where you set the formatting options for your report objects. One can format the number, font, alignment, border, and pattern for a row or column header and the values.

Concept note-4: -The answer to the given question is as follow: On the Home tab of Format dialogue, one can format dates. Click the Dialog Box Launcher next to Number on the Home tab, in the Number group. We can also open the Format Cells dialogue box by pressing CTRL+1.
