USING MICROSOFT EXCEL

HOW TO ADD COPY AND MOVE WORKSHEETS WITHIN EXCEL WORKBOOKS

[SOURCES]
You can select ____ using the arrow keys on your keyboard.

(A) ** Cells

(B) Rows

(C) Columns

(D) Worksheets

EXPLANATIONS BELOW

Concept note-1: -To select a range of cells, press and hold the Shift key while using the arrow keys on your keyboard. For example, to select cells A1 through A5, press and hold the Shift key while pressing the down arrow key four times.

Concept note-2: -To use the arrow keys to move between cells, you must turn SCROLL LOCK off. To do that, press the Scroll Lock key (labeled as ScrLk) on your keyboard. If your keyboard doesnâ€™t include this key, you can turn off SCROLL LOCK by using the On-Screen Keyboard.

Concept note-3: -Select one or more cells Or use the Shift + arrow keys to select the range. To select non-adjacent cells and cell ranges, hold Ctrl and select the cells.

Concept note-4: -To select a single cell, just use the Ctrl+Shift+I shortcut. To select a range of cells, use the Ctrl+Shift+I shortcut, then use the arrow keys to select the range of cells you want to select. To select a column or row, use the Ctrl+Shift+I shortcut, then press the Spacebar.
