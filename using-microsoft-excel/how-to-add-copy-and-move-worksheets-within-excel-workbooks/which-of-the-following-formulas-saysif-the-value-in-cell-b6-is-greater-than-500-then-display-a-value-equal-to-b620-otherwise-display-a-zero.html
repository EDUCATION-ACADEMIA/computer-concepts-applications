Which of the following formulas says:if the value in cell B6 is greater than 500, then display a value equal to B6*20; otherwise, display a zero?

(A) =(B6>500, B6*20, 0)

(B) =SUM(B6>500, B6*20, 0)

(C) =MULT(B6>500, B6*20, 0)

(D) ** =IF(B6>500, B6*20, 0)
