Returns the largest value in a set of values

(A) AVERAGE

(B) MIN

(C) ** MAX

(D) SUM

EXPLANATIONS BELOW

Concept note-1: -MAX will return the largest value in a given list of arguments. From a given set of numeric values, it will return the highest value. Unlike MAXA function, the MAX function will count numbers but ignore empty cells, text, the logical values TRUE and FALSE, and text values.

Concept note-2: -Answer. The Excel MAX function returns the largest numeric value in a range of values.

Concept note-3: -The MAX() function returns the largest value of the selected column.

Concept note-4: -Based on the returned values, the max() function will return the key having the largest value.
