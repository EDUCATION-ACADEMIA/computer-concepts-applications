The MAX function returns the total of all cells in the selected range of cells.

(A) True

(B) ** False

EXPLANATIONS BELOW

Concept note-1: -The MAX function returns the total of all cells in the selected range. The MIN function returns the smallest value in a range of cells. You must use the AutoSum button to enter a SUM function-you cannot type it manually in a cell.

Concept note-2: -MAX will return the largest value in a given list of arguments. From a given set of numeric values, it will return the highest value. Unlike MAXA function, the MAX function will count numbers but ignore empty cells, text, the logical values TRUE and FALSE, and text values.

Concept note-3: -Returns the largest value in a set of values.

Concept note-4: -Use the COUNT function to get the number of entries in a number field that is in a range or array of numbers. For example, you can enter the following formula to count the numbers in the range A1:A20: =COUNT(A1:A20). In this example, if five of the cells in the range contain numbers, the result is 5.

Concept note-5: -The MAX function returns the largest numeric value in the data provided. The MAX function can be used to return the largest value from any type of numeric data.
