Specific keyboard keys that press together or in a sequence to execute commands or apply settings.

(A) ** keyboard shortcuts

(B) function keys

(C) control keys

(D) formatting keys

EXPLANATIONS BELOW

Concept note-1: -In computing, a keyboard shortcut is a sequence or combination of keystrokes on a computer keyboard which invokes commands in software. Most keyboard shortcuts require the user to press a single key or a sequence of keys one after the other.

Concept note-2: -In addition to standard Windows menu commands, many commands have a key combination associated with it that lets you execute the command with keystrokes. Typically, you press the Ctrl or Alt key and simultaneously press the underlined letter from the appropriate command menu.

Concept note-3: -To undo an action press Ctrl+Z.

Concept note-4: -Still, there are also some keyboard shortcuts for using it. One is on the Quick Access Toolbar, or we can press F12 and display the save as option or press the keyboard shortcut CTRL+S, which opens the “Save As” dialog box to save the file in the desired format path.
