HOW TO ADD COPY AND MOVE WORKSHEETS WITHIN EXCEL WORKBOOKS

[SOURCES]
An individual box within a worksheet is known as?

(A) square

(B) range

(C) reference

(D) ** cell

EXPLANATIONS BELOW

Concept note-1: -ACTIVE CELL. In an Excel worksheet, each small rectangle or box is known as a cell. The active cell is the selected cell in which data is entered when you begin typing.

Concept note-2: -Columns are identified by letters (A, B, C), while rows are identified by numbers (1, 2, 3). Each cell has its own name-or cell address-based on its column and row. In the example below, the selected cell intersects column C and row 5, so the cell address is C5.

Concept note-3: -Cells are the boxes you see in the grid of an Excel worksheet, like this one. Each cell is identified on a worksheet by its reference, the column letter and row number that intersect at the cellâ€™s location. This cell is in column D and row 5, so it is cell D5. The column always comes first in a cell reference.

Concept note-4: -The box located to the left side of the formula bar which addresses the selected cell or group of cells in the spreadsheet is called Name box.

Concept note-5: -A group of cells is known as a cell range. Rather than a single cell address, you will refer to a cell range using the cell addresses of the first and last cells in the cell range, separated by a colon. For example, a cell range that included cells A1, A2, A3, A4, and A5 would be written as A1:A5.
