A style that displays decimal numbers as a percentage.

(A) decimal fraction macro

(B) ratio style

(C) ** percentage format

(D) format conversion

EXPLANATIONS BELOW

Concept note-1: -On the Home tab, in the Number group, click the icon next to Number to display the Format Cells dialog box. In the Format Cells dialog box, in the Category list, click Percentage.

Concept note-2: -% Style Button It is a button with a % symbol in it. This will format the cell to display as a percentage.

Concept note-3: -To display a number as a percentage of 100, include the percent sign (%) in your number format. For example, to display percentages as integers, use this format: #%. As the result, the number 0.25 entered in a cell will appear as 25%.
