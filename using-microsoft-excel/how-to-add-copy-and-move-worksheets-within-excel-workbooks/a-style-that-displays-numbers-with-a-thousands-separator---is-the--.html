A style that displays numbers with a thousands separator (, ) is the ____ .

(A) ** comma format

(B) comma default

(C) numeric label

(D) cell style

EXPLANATIONS BELOW

Concept note-1: -The comma style is a formatting style used in visualizing numbers with commas when the values are over 1, 000. Such as, if we use this style on data with a value of 100, 000, then the result displayed will be as (100, 000).

Concept note-2: -The character used as the thousands separator In the United States, this character is a comma (, ). In Germany, it is a period (.). Thus one thousand and twenty-five is displayed as 1, 025 in the United States and 1.025 in Germany. In Sweden, the thousands separator is a space.

Concept note-3: -Comma style is a type of number format where it adds commas to large numbers, adds two decimal places (i.e. 1000 becomes 1, 000.00), displays negative values in closed parentheses & represents zeros with a dash (–). The shortcut key for comma style is ALT + HK.

Concept note-4: -To create an Excel custom number format with a thousands separator, include a comma (, ) in the format code. For example: #, ###-display a thousands separator and no decimal places. #, ##0.00-display a thousands separator and 2 decimal places.
