You can create mixed references by placing a dollar sign in front of only the column letter or only the row number

(A) ** True

(B) False

EXPLANATIONS BELOW

Concept note-1: -Answer. Answer: it sentence is false .

Concept note-2: -A mixed reference uses a dollar sign either in front of the row letter or in front of the column number, but not both â€“ for example, A$1 is a mixed reference in which the row adjusts, but the column does not. So if you move a formula containing that reference one cell down and one cell to the right, it becomes B$1.

Concept note-3: -Mixed reference in Excel means only part of the reference is fixed, either the row or the column, and the other part is relative. Unlike absolute references, only one $ (dollar sign) is applied, either in front of the column or row number.

Concept note-4: -Mixed Referencing is a combination of Relative and Absolute reference. In this type of reference, either row or column has to remain fixed.
