Which of these formulas will add three cells and then divide the sum by 4?

(A) =A1+A2+A3^4

(B) ** =(A1+A2+A3)/4

(C) (A1+A2) + A3/4

(D) A1+A2+A3/4

EXPLANATIONS BELOW

Concept note-1: -Answer. Explanation: a. =(A1 +A2 +A3)/4 shows the summation of values of A1, A2 and A3 which is then divided by 4.

Concept note-2: -In Excel, a formula is an expression that operates on values in a range of cells or a cell. For example, =A1+A2+A3, which finds the sum of the range of values from cell A1 to cell A3.

Concept note-3: -=A1+A2+A3 Adds the values in cells A1, A2, and A3.

Concept note-4: -For example: =SUM(A2:A10) Adds the values in cells A2:10.

Concept note-5: -Divide numbers in a cell To do this task, use the / (forward slash) arithmetic operator. For example, if you type =10/5 in a cell, the cell displays 2. Important: Be sure to type an equal sign (=) in the cell before you type the numbers and the / operator; otherwise, Excel will interpret what you type as a date.
