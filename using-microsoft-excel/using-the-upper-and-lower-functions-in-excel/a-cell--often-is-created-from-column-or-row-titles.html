USING MICROSOFT EXCEL

USING THE UPPER AND LOWER FUNCTIONS IN EXCEL

[SOURCES]
A cell ____ often is created from column or row titles.

(A) function

(B) ** name

(C) formula

(D) data table

EXPLANATIONS BELOW

Concept note-1: -A cell name often is created from column and row titles. You cannot use column and row titles within formulas when you want to refer to the related data by name. Assigning names to a cell or range of cells allows you to select them quickly. Spreadsheet specialists usually format a worksheet before they enter any data.

Concept note-2: -When you want column and row headings displayed whether you are at the top, bottom, left, or right of the spreadsheet, select: Freeze Panes. Use this button to combine two or more cells into one cell and center the text over a group of columns.

Concept note-3: -RANGE. A range is a group of two or more cells in a worksheet. The cells in a range can be adjacent or nonadjacent.

Concept note-4: -First, all of the cells must be adjacent to each other. This means that they must be next to each other horizontally or vertically, but not diagonally. Second, the cells must be in the same row or column; you cannot select a range of cells that are in different rows or columns.

Concept note-5: -The most commonly used way to select multiple cells is to click and drag. You need to click on a cell and drag it over the spreadsheet. Press on a cell.
