How do you quickly modify a table style to meet user preferences? → Logical

(A) ** False

(B) True

(C) ** Right-click on style and choose duplicate

EXPLANATIONS BELOW

Concept note-1: -To modify a custom table style, go to the Table Styles gallery, right-click on the style, and click Modify… To delete a custom table style, right-click on it, and select Delete.

Concept note-2: -On the Home tab, in the Styles group, click Format as Table. Click the table style that you want to use. Tips: Auto Preview-Excel will automatically format your data range or table with a preview of any style you select, but will only apply that style if you press Enter or click with the mouse to confirm it.

Concept note-3: -(First make sure your selected cell is anywhere in the data set that you want to convert into a table). The keyboard shortcut for this procedure is Ctrl + T . This will bring up the Create Table box, which is prepopulated with the existing boundaries of the data set you’re in.
