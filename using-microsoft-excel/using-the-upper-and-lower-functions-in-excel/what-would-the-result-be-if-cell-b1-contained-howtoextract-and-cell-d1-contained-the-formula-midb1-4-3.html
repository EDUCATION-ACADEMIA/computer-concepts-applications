What would the result be if cell B1 contained ‘howtoextract’ and cell D1 contained the formula =MID(B1, 4, 3)?

(A) how

(B) wto

(C) ** toe

(D) ext

EXPLANATIONS BELOW

Concept note-1: -For example, if cell A1 had a first name and cell B1 had a last name, you could enter =A1&” “&B1, which combines A1 with B1, with a space between each value. You can also use a concatenate formula to combine cell values.

Concept note-2: -The formula for this function is =CONCATENATE(Cell reference 1, Cell reference 2). So, for example, you can input “=CONCATENATE (A1, B1)” into cell C1 to combine the values of A1 and B1 into that cell.

Concept note-3: -=AVERAGE(number1, [number2], …) The function uses the following arguments: Number1 (required argument) – This is the first number of a cell reference or a range for which we want the average. Number2 (optional argument) – They are the additional numbers, cell references or a range for which we want the average.

Concept note-4: -If cell contains specific text, then return a value Select the output cell, and use the following formula: =IF(cell="text", value to return, “"). For our example, the cell we want to check is A2, the text we’re looking for is “example”, and the return value will be Yes.
