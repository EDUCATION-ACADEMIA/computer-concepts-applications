USING MICROSOFT EXCEL

MODIFYING PAGE SETUP PROPERTIES IN EXCEL

[SOURCES]
The sort feature is responsible for controlling the sequence, or order, of the records.

(A) ** True

(B) False

EXPLANATIONS BELOW

Concept note-1: -The sort feature is responsible for controlling the sequence, or order, of the records. Q. A database and a table is the same thing.

Concept note-2: -Press CTRL+F. The Find and Replace dialog box appears. If you only want to find records that contain wildcard characters, click the Find tab. If you want to find wildcard characters and replace them with other data, click the Replace tab.

Concept note-3: -Search for a record When you enter text in the Search box, the first matching value is highlighted in real time as you enter each character. You can use this feature to quickly search for a record with a matching value. You can also search for a specific record in a table or form by using the Find option.

Concept note-4: -To sort a query in Access in the query design view, click into the “Sort” row of the field in the QBE grid by which to sort the result set. Then use the drop-down that appears to select either “Ascending” or “Descending” order.
