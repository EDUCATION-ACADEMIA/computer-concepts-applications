Which of the following can be done with the Move Chart dialog box?

(A) Make a duplicate copy of a chart

(B) ** Move a chart to a new worksheet or existing worksheet

(C) Move a chart to a Word document

(D) Move a chart to a PowerPoint presentation

(E) None

EXPLANATIONS BELOW

Concept note-1: -To move the chart to a new worksheet, click New sheet, and then in the New sheet box, type a name for the worksheet. To move the chart as an object in another worksheet, click Object in, and then in the Object in box, select the worksheet in which you want to place the chart.
