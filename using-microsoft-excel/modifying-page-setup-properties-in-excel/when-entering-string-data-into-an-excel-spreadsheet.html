USING MICROSOFT EXCEL

MODIFYING PAGE SETUP PROPERTIES IN EXCEL

[SOURCES]
When entering string data into an Excel spreadsheet,

(A) ** default data align left and bottom in cell

(B) Default data centered in cell

(C) default data left aligned in cell

(D) default data top right in cell

EXPLANATIONS BELOW

Concept note-1: -As a default in Microsoft Excel, text typed into a cell is aligned to the left border of the cell while numbers are aligned to the right.

Concept note-2: -As for WHY was this default chosen, it’s because it’s easier (or at least it was) to read adjacent content by considering the end of each cell. This comes from old accounting books, see example below, you’ll see the amounts are at the baseline of the content of left column (which is multiline).

Concept note-3: -Excel actually has its own defaults for alignment. For example, text, such as titles and labels, automatically align horizontally to the left, and numbers, however, are automatically aligned to the right. All data is vertically aligned to the bottom.
