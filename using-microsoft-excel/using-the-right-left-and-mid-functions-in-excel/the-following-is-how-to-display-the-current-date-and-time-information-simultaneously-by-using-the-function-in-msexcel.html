USING MICROSOFT EXCEL

USING THE RIGHT LEFT AND MID FUNCTIONS IN EXCEL

[SOURCES]
The following is how to display the current Date and Time information simultaneously by using the function in MS.Excel____

(A) =Date()

(B) =time()

(C) ** =now()

(D) =Today()

EXPLANATIONS BELOW

Concept note-1: -To insert the current date and time, press Ctrl+; (semi-colon), then press Space, and then press Ctrl+Shift+; (semi-colon).

Concept note-2: -The correct answer is Today(). The TODAY() function takes no arguments and returns the current date, updated whenever a worksheet is changed or opened. You can also use F9 to force the worksheet to recalculate and update the value. If you need the current date and time, use the NOW() function.

Concept note-3: -To enter a date in Excel, use the “/” or “-” characters. To enter a time, use the “:” (colon). You can also enter a date and a time in one cell.

Concept note-4: -Calculates the number of days, months, or years between two dates. This function is useful in formulas where you need to calculate an age. DATEVALUE function. Converts a date in the form of text to a serial number. DAY function.

Concept note-5: -The current timestamp() function is used to get the current date and time. The curtime() function is used to get the current time. The last day() function is used to get the last date of the given month on the date.
