Where is a name box and what does it do?

(A) It is a box to the right of the formula bar that displays the cell that is currently selected in the spreadsheet.

(B) It is a toolbar at the top of MS Excel spreadsheet .window that you can use to enter or copy an existing formula into the cells or charts.

(C) ** It is a box to the left of the formula bar that displays the cell reference of the cell that is currently selected.

(D) It is a single spreadsheet that contains cells organized by rows and columns.

EXPLANATIONS BELOW

Concept note-1: -In Microsoft Excel, the Name Box displays the cell that is currently selected in the spreadsheet. It is located to the left of the formula bar. If a name is defined for a cell that is selected, the Name Box displays the name of the cell. You can use the Name Box to define a name for a selected cell, as well.

Concept note-2: -The box located to the left side of the formula bar which addresses the selected cell or group of cells in the spreadsheet is called Name box.

Concept note-3: -The box located to the left side of the formula bar which addresses the selected cell or group of cells in the spreadsheet is called Name box.

Concept note-4: -Expert-Verified Answer The Name Box displays the cell and Formula bar is used to see the formulas. Explanation: The Name Box indicates the cell actually picked in the spreadsheet. It is situated at the left hand side of the formula sign.

Concept note-5: -The name box is the box at left end of the formula bar that identifies the selected cell, chart item, or drawing object. The Name Box displays the cell reference of the active cell. The Name Box can also be used to assign names to cells or ranges of cells.
