What is special about the +Body and +Heading fonts?

(A) They automatically increase in point size each time to select them.

(B) ** They automatically adjust themselves when a new theme is applied.

(C) They are designed for use in HTML representations of your workbook.

(D) They automatically configure themselves to be one point larger in size than text in surrounding cells.

EXPLANATIONS BELOW

Concept note-1: -Heading fonts and body fonts are part of the templates available to you as part of your brand. Heading fonts are used as larger, higher impact text, such as in a title or section header. Body fonts are usually used with longer text, like a paragraph or secondary text.

Concept note-2: -Go to Format > Font > Font. + D to open the Font dialog box. Select the font and size you want to use.

Concept note-3: -By default, Word applies the Office theme to all new, blank documents. In Word 2016, the Office theme uses a primarily blue palette, the Calibri font for body text, and Calibri Light for headings. If you plan to frequently use a theme other than the Office theme, you can make that the default theme.

Concept note-4: -Upon installation, Microsoft Word uses the Calibri font and 11 point font size as the default settings.
