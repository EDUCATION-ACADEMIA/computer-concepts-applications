The ____ inserts a comma every three positions to the left of the decimal point and causes numbers to be displayed to the nearest hundredths.

(A) Floating dollar sign

(B) Ribbon

(C) Summary

(D) ** Comma style format

EXPLANATIONS BELOW

Concept note-1: -The comma style in Excel is also known as the thousand separator format that converts the large number values with the commas inserted as separators to distinguish the number value length into thousands, hundred thousand, millions, and so on.

Concept note-2: -Use 1000 separator-The 1000 separator will insert a comma every 3 digits, i.e. hundred, thousand etc.

Concept note-3: -The standard comma style format contains up to two decimal points and a thousand separators and locks the dollar sign to the extreme left half of the cell. Negative numbers are shown in enclosures.

Concept note-4: -To display numbers with comma style format in Excel: Select the cells containing numeric sales values for which I want to display numbers with comma style number format. On the Home tab, you can select the comma symbol & click on the Comma Style command in the Number group.
