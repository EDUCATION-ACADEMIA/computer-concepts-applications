Sam is using the functions within Excel to complete questions in a school homework assignment. Sam needs to find the average of a range of numbers. What would be an example of a formula written correctly?

(A) =AVG(E4, E9)

(B) ** =AVERAGE(E4:E9)

(C) =AVG(E4:E9)

(D) +AVERAGE(E4:E9)

EXPLANATIONS BELOW

Concept note-1: -The MIN function checks your data range and returns the smallest value in the set.

Concept note-2: -Use the COUNTIF function to count how many times a particular value appears in a range of cells. For more information, see COUNTIF function.

Concept note-3: -Use the COUNT function to get the number of entries in a number field that is in a range or array of numbers. For example, you can enter the following formula to count the numbers in the range A1:A20: =COUNT(A1:A20). In this example, if five of the cells in the range contain numbers, the result is 5.
