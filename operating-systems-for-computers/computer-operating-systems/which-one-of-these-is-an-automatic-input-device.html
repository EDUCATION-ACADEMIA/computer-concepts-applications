Which one of these is an automatic input device?

(A) ** Optical Mark Recognition

(B) Mouse

(C) Keyboard

(D) Scanner

EXPLANATIONS BELOW

Concept note-1: -OMR is now used as an input device for data entry. Two early forms of OMR are paper tape and punch cards which use actual holes punched into the medium instead of pencil filled circles on the medium. Paper tape was used as early as 1857 as an input device for telegraph.

Concept note-2: -The OMR is an input device of computer system which is used for collecting data and information from other sources and presents it to computer system for further processing and handling. This OMR device uses laser light for detection, they are extensively used in examination to check correct answers.

Concept note-3: -Yes, Optical Mark Reader is an Input Device. Whereas an output devices are such type of computer hardware or devices which converts those piece of information or data into a human-perceptible form. Output devices process the raw input data which are entered in the computer through an input device.

Concept note-4: -The most common use of OMR is in offices, academics and research departments where large numbers of hand-filled documents must be processed such as surveys, questionnaires, exams, reply cards and ballots. OMR can handle hundreds of thousands of physical documents per hour, and its accuracy is up to 99%.
