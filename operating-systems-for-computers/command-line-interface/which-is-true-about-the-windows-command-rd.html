Which is true about the Windows command RD?

(A) It can remove directories and files by default.

(B) It is used for removing files in a directory.

(C) ** By default, it can be used to delete an empty directory.

(D) It is used to rename directories.

EXPLANATIONS BELOW

Concept note-1: -You can’t use the rd command to delete the current directory. If you attempt to delete the current directory, the following error message appears: The process can’t access the file because it is being used by another process.

Concept note-2: -To remove a directory, just use the command rmdir <directory name> . Note: Any directories deleted with the rmdir command cannot be recovered.

Concept note-3: -To remove a directory that is not empty, use the rm command with the-r option for recursive deletion. Be very careful with this command, because using the rm-r command will delete not only everything in the named directory, but also everything in its subdirectories.

Concept note-4: -Use the rm command to remove files you no longer need. The rm command removes the entries for a specified file, group of files, or certain select files from a list within a directory.

Concept note-5: -The RD or RMDIR command deletes an empty directory. The directory must be empty of both files and directories when used without the “/S” switch. Examples of how to use this command. Before you delete a directory, it must be empty of files and folders.
