SYSTEMS DEVELOPMENT LIFE CYCLES

What is done at the analysis stage?

(A) ** Review existing systems for weaknesses or improvements that can be made

(B) Create a new system

(C) Implementing a new system to the company

(D) Decided to use the new system

EXPLANATIONS BELOW

Concept note-1: -In the system analysis phase the system to be constructed is analyzed using an object-oriented method, in order to provide the architecture of the system, and to identify the objects to be implemented to obtain the required functionality (figure 5).

Concept note-2: -Systems Analysis It is a process of collecting and interpreting facts, identifying the problems, and decomposition of a system into its components. System analysis is conducted for the purpose of studying a system or its parts in order to identify its objectives.

Concept note-3: -Analysis Stage Define any prototype system requirements. Evaluate alternatives to existing prototypes. Perform research and analysis to determine the needs of end-users.

Concept note-4: -Systems analysis includes: initial investigation; systems survey; feasibility study; determination of information needs and system requirements; and delivery of systems requirements.
