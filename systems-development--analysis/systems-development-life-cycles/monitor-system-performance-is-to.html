EXPLANATIONS BELOW

Concept note-1: -System performance monitoring helps admins to monitor the required parameters of the network devices closely. This data can be used to identify system performance degradation, deploy proactive countermeasures, and eventually prevent unintended network downtime.

Concept note-2: -A system performance monitor (SPM) is a type of application that identifies, collects, monitors and reports on the overall operational health of a computer system. It is a performance monitoring tool that enables end users, administrators and organizations to gauge and evaluate the performance of a given system.

Concept note-3: -Continuous system performance monitoring can do the following: Sometimes detect underlying problems before they have an adverse effect. Detect problems that affect a user’s productivity. Collect data when a problem occurs for the first time.

Concept note-4: -Histograms show the frequencies with which events are occurring, for example, the number of complaints received per product produced or sold by an organisation. Studies show that using dashboards is the most effective way of measuring the operational performance.
