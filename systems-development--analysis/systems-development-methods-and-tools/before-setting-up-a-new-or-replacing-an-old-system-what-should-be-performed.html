Before setting up a new, or replacing an old system, what should be performed?

(A) Systems investigation

(B) Systems development

(C) ** Systems analysis

EXPLANATIONS BELOW

Concept note-1: -Before setting up a new or replacing an old system, what should be performed?, Systems analysis should be performed before setting up a new, or replacing an old system. Which of these is the correct order to perform things? The order is analysis, design, implementation, evaluation.

Concept note-2: -System analysis allows for better management through changing the software to suit any business changes, this means that the final product will be totally controllable. If changes or enhancements are required, the requirement of rewriting the whole software will be removed, which is normally costly.

Concept note-3: -The first step in the systems development life cycle is Preliminary investigation. 1) Planning/Preliminary Investigation: This is the first phase of the systems development process. It identifies whether or not there is a need for a new system to achieve a business"s strategic objectives.

Concept note-4: -These activities, or phases, usually include planning, analysis, design, implementation, and maintenance/support.
