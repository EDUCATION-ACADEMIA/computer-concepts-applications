Which four items can be related to a code merge review request in Oracle Developer CloudService?

(A) ** An issue in the issue tracking system

(B) ** Code reviewers

(C) ** A build process

(D) ** Branches of your code

(E) A deployment configuration

EXPLANATIONS BELOW

Concept note-1: -A pull request – also referred to as a merge request – is an event that takes place in software development when a contributor/developer is ready to begin the process of merging new code changes with the main project repository.

Concept note-2: -What Is a Merge Request? A merge request is simply a request from a user to merge their code from one branch to another, typically to the master branch.
