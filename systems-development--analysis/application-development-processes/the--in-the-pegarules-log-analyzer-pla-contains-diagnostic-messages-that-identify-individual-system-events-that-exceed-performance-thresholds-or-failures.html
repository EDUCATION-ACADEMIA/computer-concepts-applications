The ____ in the PegaRULES Log Analyzer (PLA) contains diagnostic messages that identify individual system events that exceed performance thresholds or failures.

(A) Garbage Collection (GC) log

(B) PEGA log

(C) ALERTSECURITY log

(D) ** ALERT log

EXPLANATIONS BELOW

Concept note-1: -The ALERT log contains diagnostic messages that identify individual system events that exceed performance thresholds or failures. Alert messages contain a set of field values that identify the alert, the conditions that caused it, and the state of system when it occurred.

Concept note-2: -It is recommended that you use PLA to test a new or reconfigured Process Commander application immediately after deployment into a production environment. This is the phase when most major performance issues are likely to occur due to the effects of real-world scaling and processing demands.

Concept note-3: -The PEGA log contains warnings, errors, and information messages about internal operations. This file, also referred to as the Console log or System log, is used for debugging the application.

Concept note-4: -You access log files from the Log utilities section of the Logs tab in Dev Studio (Configure > System > Operations > Logs) to open the Log utilities.

Concept note-5: -Once you sign into the PDC tenant, you will be greeted by the following screen. The arrow is pointing you to the Event Viewer, which provides access to the alerts logs.
