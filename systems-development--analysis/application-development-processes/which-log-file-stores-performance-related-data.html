Which log file stores performance related data?

(A) PEGA

(B) ** ALERT

(C) ALERTSECURITY

(D) SERVICES-PAL

EXPLANATIONS BELOW

Concept note-1: -The PEGA log contains warnings, errors, and information messages about internal operations. This log is also called the console log or system log and is used for debugging the application. The ALERT log contains performance-related alerts triggered when an operation exceeds the specified performance threshold.

Concept note-2: -Alert logs contain important information about error messages and exceptions that occur during database operations. Each Oracle Database for Windows instance has one alert log; information is appended to the file each time you start the instance. All threads can write to the alert log.

Concept note-3: -A log file is an event that took place at a certain time and might have metadata that contextualizes it. Log files are a historical record of everything and anything that happens within a system, including events such as transactions, errors and intrusions.

Concept note-4: -Types of Log Files An event log is a high-level log that records network traffic and usage data such as incorrect password attempts, login attempts, and application events. A server log is a text document that keeps track of actions on a certain server over some time.
