TYPES OF NETWORK SECURITY

[SOURCES]
A type of threats where events or attacks that remove, corrupt, deny access to, allow access to or steal information.

(A) ** Data

(B) Physical

(C) Internal

(D) External

EXPLANATIONS BELOW

Concept note-1: -Data: Events or attacks that remove, corrupt, deny access to, allow access to, or steal information. Internal: Employees who have access to data, equipment, and the network. Internal attacks can be characterized as follows: Malicious threats are when an employee intends to cause damage.

Concept note-2: -Threats can be classified into four different categories; direct, indirect, veiled, conditional.

Concept note-3: -Distributed denial-of-service (DDoS) attacks In a distributed denial-of-service (DDoS) attack, multiple compromised machines attack a target, such as a server, website or other network resource, making the target totally inoperable.

Concept note-4: -A cyber attack is any attempt to gain unauthorized access to a computer, computing system or computer network with the intent to cause damage. Cyber attacks aim to disable, disrupt, destroy or control computer systems or to alter, block, delete, manipulate or steal the data held within these systems.

Concept note-5: -The main types of information security threats are: Malware attack. Social engineering attacks. Software supply chain attacks.
