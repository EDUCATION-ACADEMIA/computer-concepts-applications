Which of the following ciphers uses asymmetric encryption?

(A) AES

(B) Caesar Cipher

(C) DES

(D) ** RSA

EXPLANATIONS BELOW

Concept note-1: -Explanation: The Diffie Hellman cipher uses asymmetric key cryptography.

Concept note-2: -RSA encryption is an asymmetric encryption algorithm named after its founders (Rivest, Shamir & Adleman) that uses block cipher methodology to encrypt data.

Concept note-3: -RSA is an algorithm used by systems to encrypt and decrypt messages. It is an asymmetric cryptographic algorithm.

Concept note-4: -RSA is the most famous example. The name “asymmetric” comes from the fact that different keys are used for encryption vs. decryption. In symmetric schemes like AES, the same key is used both for encryption and decryption.

Concept note-5: -The Caesar Cipher is a monoalphabetic rotation cipher used by Gaius Julius Caesar. Caesar rotated each letter of the plaintext forward three times to encrypt, so that A became D, B became E, etc., as shown in Table 4.6.
