TYPES OF COMPUTER SECURITY

[SOURCES]
A touchpad, joystick, and trackball are all examples of:

(A) ** Pointing device

(B) Keyboard

(C) Consoles

(D) Output device

EXPLANATIONS BELOW

Concept note-1: -Detailed Solution. The correct answer is pointing input. A pointing device is an input interface that allows a user to input spatial data to a computer.

Concept note-2: -A touchpad or trackpad is a flat surface that can detect finger contact. It is a stationary pointing device, commonly used on laptop computers. At least one physical button normally comes with the touchpad, but the user can also generate a mouse click by tapping on the pad.

Concept note-3: -A joystick is an input device that can be used for controlling the movement of the cursor or a pointer in a computer device. The pointer/cursor movement is controlled by maneuvering a lever on the joystick. The input device is mostly used for gaming applications and, sometimes, in graphics applications.

Concept note-4: -A trackball mouse is a pointing device consisting of a ball held by a socket containing sensors to detect a rotation of the ball about two axes-like an upside-down ball mouse with an exposed protruding ball.

Concept note-5: -Detailed Solution. Pointing devices: A pointing device is an input device.
