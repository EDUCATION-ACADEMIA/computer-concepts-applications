Which of the following port and IP address scanner famous among the users?

(A) Cain and Abel

(B) ** Angry IP Scanner

(C) Snort

(D) Ettercap

EXPLANATIONS BELOW

Concept note-1: -Explanation: Angry IP Scanner is a type of hacking tool that is usually used by both white hat and black hat types of hackers. It is very famous among the users because it helps to find the weaknesses in the network devices.

Concept note-2: -As you can see, I’m scanning for port 80 and port 443. If you want to scan a range of ports, then enter the port range like “1-1000.” After you have entered the port number, click on OK to save the changes. Once you are done configuring the Angry IP Scanner, you can continue to scan.

Concept note-3: -However, as it involves sending of ICMP (Internet Control Message Protocol) packets, it requires administrative (or root) privileges. If Angry IP Scanner runs without these privileges, this method can’t be used. Angry IP Scanner implements this using the Raw Sockets.

Concept note-4: -Port scanners usually scan TCP and sometimes UDP ports of a single host by sequentially probing each of them.

Concept note-5: -Run Angry IP from the applications menu or type ‘ipscan’ in a terminal, also you can run ipscan as root. Also, you can choose a random scanning or choose a text file with the IP addresses to scan. Write the ports in this field, also you can specify ranges.
