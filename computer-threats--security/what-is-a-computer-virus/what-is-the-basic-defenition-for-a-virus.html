WHAT IS A COMPUTER VIRUS

[SOURCES]
what is the basic defenition for a virus?

(A) a biological agent that reproduces inside the cells of living hosts.

(B) a term used to describe a malicious individual or group of individuals who scam users

(C) ** a malicious software program loaded onto pc without the user’s knowledge and performs malicious actions

(D) is when an attacker leaves a malware-infected physical device

EXPLANATIONS BELOW

Concept note-1: -Definition: A computer virus is a malicious software program loaded onto a user’s computer without the user’s knowledge and performs malicious actions. Description: The term ‘computer virus’ was first formally defined by Fred Cohen in 1983. Computer viruses never occur naturally. They are always induced by people.

Concept note-2: -Spyware is a type of malicious software–or malware–that is installed on a computing device without the end user’s knowledge.

Concept note-3: -A virus, on the other hand, is a type of malware. Its definition is limited only to programs or code that self-replicates or copies itself in order to spread to other devices or areas of the network. Based on the above definitions, the word malware can be used to refer to any type of virus.

Concept note-4: -A computer virus is a type of malware that attaches to another program (like a document), which can replicate and spread after a person first runs it on their system. For instance, you could receive an email with a malicious attachment, open the file unknowingly, and then the computer virus runs on your computer.
