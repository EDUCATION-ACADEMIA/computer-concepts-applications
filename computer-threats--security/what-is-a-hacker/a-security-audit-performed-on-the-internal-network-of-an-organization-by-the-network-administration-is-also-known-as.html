A security audit performed on the internal network of an organization by the network administration is also known as:

(A) Gray-box testing

(B) Black-box testing

(C) ** White-box testing

(D) Active testing

EXPLANATIONS BELOW

Concept note-1: -A security audit performed on the internal network of an organization by the network administration is also known as . Answer 2. Option C. Explanation: White-box testing is a security audit performed with internal knowledge of the systems.

Concept note-2: -The statement is TRUE because white-box testing is a security audit performed on the internal network of an organization by network administration. The opposite would be black-box testing, which is a security audit on a system that the hacker knows nothing about.

Concept note-3: -A penetration test is a type of audit that tests the security implementation and access controls in an organization by attempting to gain unauthorized access to systems, networks, or devices.

Concept note-4: -Which of the following statements best describes a white-hat hacker? Answer 1. Explanation: A white-hat hacker is a “good” guy who uses his skills for defensive purposes.

Concept note-5: -An ethical hacker (“white hat hacker”) is an information security professional who has the same skills and uses the same technologies as a malicious hacker (“black hat hacker”) to discover vulnerabilities and weaknesses in an organization’s systems.
