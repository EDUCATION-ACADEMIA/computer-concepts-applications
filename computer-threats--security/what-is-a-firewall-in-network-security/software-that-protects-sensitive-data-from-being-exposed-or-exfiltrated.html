WHAT IS A FIREWALL IN NETWORK SECURITY

[SOURCES]
Software that protects sensitive data from being exposed or exfiltrated

(A) ** DLP

(B) DAC

(C) DAL

(D) DAP

EXPLANATIONS BELOW

Concept note-1: -Data loss prevention (DLP) software, also known as data leak prevention software, is used to secure control and ensure compliance of sensitive business information. A key component of DLP solutions is distribution control, which ensures users do not send private information outside of corporate business networks.

Concept note-2: -Three types of data loss prevention are network DLP, endpoint DLP, and cloud DLP.

Concept note-3: -A DLP solution uses things like antivirus software, AI, and machine learning to detect suspicious activities. For example, it uses rule-based analysis and compares content to your DLP policy, which defines how your organization labels, shares, and protects data without exposing it to unauthorized users.
