Which of the following is an incorrect statement about packet filtering on firewalls____

(A) a. Packet filtering is a type of security technology that is used to regulate what packets are allowed to enter the system or network and what packets are blocked.

(B) a. Packet filtering is generally used to block suspicious traffic coming from suspicious IP addresses

(C) a. Packet filtering firewalls check on routers and switches for incoming data packets starting from the destination IP and origin, port number, type of data packet and the information carried and then forwarded to the destination address.

(D) ** This type of packet filtering firewall works at the TCP level

(E) Packet filtering is generally used to block suspicious traffic coming from, suspicious application protocol types

EXPLANATIONS BELOW

Concept note-1: -The correct answer is option 2. Concept: Option 1: A firewall acts as a packet filter inspecting all the packets entering the local network. True, A packet-filtering firewall examines each packet that crosses the firewall and tests the packet according to a set of rules that you set up.

Concept note-2: -Detailed Solution. The correct answer is It is unable to permit legitimate communication to pass.

Concept note-3: -Explanation: There are four types of firewall based on their working and characteristics. These are Packet Filtering Firewalls, Circuit Level Gateway Firewalls, Application level Gateway Firewalls, and Stateful Multilayer Inspection Firewalls.

Concept note-4: -Which of the following is / are the types of firewall? a)Packet Filtering Firewallb)Dual Homed Gateway Firewallc)Screen Host Firewalld)All of the mentionedCorrect answer is option ‘A’.

Concept note-5: -While circuit-level gateways provide a higher level of security than packet filtering firewalls, they should be used in conjunction with other systems. For example, circuit-level gateways are typically used alongside application-level gateways.
