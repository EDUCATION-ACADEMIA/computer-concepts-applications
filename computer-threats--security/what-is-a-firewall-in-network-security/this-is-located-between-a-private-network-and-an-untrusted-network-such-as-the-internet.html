WHAT IS A FIREWALL IN NETWORK SECURITY

[SOURCES]
This is located between a private network and an untrusted network, such as the internet.

(A) VPN

(B) ** Screened subnet

(C) VLAN

(D) Intranet

EXPLANATIONS BELOW

Concept note-1: -Since it adds an additional layer of security to the network, a screened subnet is suitable for many high-traffic sites. The screened subnetâ€™s main purpose is to isolate a demilitarized zone (DMZ) and its publicly accessible resources from an internal network to minimize possible attacks on the subnetwork.

Concept note-2: -A network DMZ sits between two firewalls, creating a semisafe buffer zone between the internet and the enterprise LAN. If better-prepared threat actors pass through the first firewall, they must then gain unauthorized access to the services in the DMZ before they can do any damage.

Concept note-3: -Publicly-accessible resources (servers) are placed inside the screened subnet. Examples of publicly-accessible resources include web, FTP, or email servers. Private resources that are not accessible from the internet are placed behind the DMZ (behind the inner firewall).

Concept note-4: -A Firewall manages the secure in-flow and out-flow of data in a device. It monitors the network traffic and acts as a barrier between the trusted and untrusted network.
