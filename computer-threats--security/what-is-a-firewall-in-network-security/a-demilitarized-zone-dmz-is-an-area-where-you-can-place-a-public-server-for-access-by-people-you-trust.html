WHAT IS A FIREWALL IN NETWORK SECURITY

[SOURCES]
A demilitarized zone (DMZ) is an area where you can place a public server for access by people you trust.

(A) True

(B) ** False

EXPLANATIONS BELOW

Concept note-1: -A demilitarized zone network, or DMZ, is a subnet that creates an extra layer of protection from external attack. Network administrators must balance access and security. Your employees must tap into data outside of the organization, and some visitors need to reach into data on your servers.

Concept note-2: -Purpose of a DMZ The DMZ network exists to protect the hosts most vulnerable to attack. These hosts usually involve services that extend to users outside of the local area network, the most common examples being email, web servers, and DNS servers.

Concept note-3: -DMZ in computer security is an area between the organizationâ€™s local network and the outside world. Similarly, a reverse proxy server is a proxy server placed at the Web server end instead of the client end, and it intervenes between the Internet users and the Web servers.

Concept note-4: -A demilitarized zone (DMZ or DZ) is an area in which treaties or agreements between nations, military powers or contending groups forbid military installations, activities, or personnel.
