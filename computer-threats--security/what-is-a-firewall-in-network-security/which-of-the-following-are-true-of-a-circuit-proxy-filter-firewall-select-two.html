WHAT IS A FIREWALL IN NETWORK SECURITY

[SOURCES]
Which of the following are true of a circuit proxy filter firewall? (Select two.)

(A) ** Operates at the Session layer.

(B) Operates at the Network and Transport layers.

(C) Operates at the Application layer.

(D) ** Verifies sequencing of session packets.

(E) Examines the entire message contents.

EXPLANATIONS BELOW

Concept note-1: -(A circuit proxy filter firewall operates at the session layer. It verifies the sequencing of session packets, breaks the connections, and acts as a proxy between the server and the client. An application layer firewall operates at the application layer, examines the entire message, and can act as a proxy to clients.

Concept note-2: -Which of the following are characteristics of a packet filtering firewall? (Select two.) A packet filtering firewall makes decisions about which network traffic to allow by examining information in the IP packet header such as source and destination addresses, ports, and service protocols.

Concept note-3: -The most common firewall types based on methods of operation are: Packet-filtering firewalls. Proxy firewalls.

Concept note-4: -Which of the following is true about a network-based firewall? A network-based firewall is installed at the edge of a private network or network segment.
