WHAT IS A COMPUTER KEYBOARD

[SOURCES]
This is used to move the cursor UP

(A) ** UP Arrow key

(B) Enter key

(C) Spacebar key

EXPLANATIONS BELOW

Concept note-1: -Answer: Arrow keys or cursor movement keys are used for moving the cursor left, right, up or down. Explanation: On a computer keyboard, arrow keys or cursor movement keys are buttons that are either programmed or designated to move the pointer in a particular direction.

Concept note-2: -Cursor control keys When using word processors, the ‘up arrow’ moves your cursor up one line of typed characters at a time, the ‘down’ arrow moves it down one line of typed characters at a time. The left and right arrows move the cursor one character to the left or right.

Concept note-3: -Moving With Arrow Keys You should be able to move the cursor freely about the screen by using combinations of the up, down, right, and left arrow keys.

Concept note-4: -Arrow keys or cursor movement keys are keys on a computer keyboard that are either programmed or designated to move the cursor in a specified direction.

Concept note-5: -Meaning of arrow key in English one of the four keys on a computer keyboard that you can use to move the cursor up, down, left, or right on the screen: Press your keyboard’s <ScrollLock> key, and then use the arrow keys on your keyboard to scroll around your page.
