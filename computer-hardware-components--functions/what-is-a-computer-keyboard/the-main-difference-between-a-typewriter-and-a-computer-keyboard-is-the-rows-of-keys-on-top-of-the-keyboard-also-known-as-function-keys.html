WHAT IS A COMPUTER KEYBOARD

[SOURCES]
The main difference between a typewriter and a computer keyboard is the rows of keys on top of the keyboard also known as function keys.

(A) ** True

(B) False

EXPLANATIONS BELOW

Concept note-1: -The main difference between a typewriter and a computer keyboard is the rows of keys on top of the keyboard also known as function keys. When the Number lock key of the numeric keypad is off, the 7 key works like the Page Up key. A computer keyboard is similar to a typewriter.

Concept note-2: -Keyboard is an input device. while type writer is a mechanical device . as we all know keyboard is a digital and input device but a typewriter is a fully mechanical machine. we cannot use a keyboard without monitor or any display but in typewriter we have to load a paper and it starts printing on it with ink.

Concept note-3: -QWERTY (/ˈkwɜːrti/) is a keyboard layout for Latin-script alphabets. The name comes from the order of the first six keys on the top left letter row of the keyboard ( Q W E R T Y ). The QWERTY design is based on a layout created for the Sholes and Glidden typewriter and sold to E. Remington and Sons in 1873.

Concept note-4: -The row above the top row keys is called the number row keys, and the row below is called the home row keys.

Concept note-5: -A function key refers to one of the twelve keys present on the top row of a common PC keyboard. Labeled from F1–F12, these function keys are responsible for performing shortcut tasks like printing or saving files. Different operating systems, for example Windows and Mac OS, have separate uses for function keys.
