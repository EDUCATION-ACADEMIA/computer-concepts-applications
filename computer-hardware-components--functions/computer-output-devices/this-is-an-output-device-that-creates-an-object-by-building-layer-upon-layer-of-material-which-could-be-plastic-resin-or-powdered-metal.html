COMPUTER OUTPUT DEVICES

[SOURCES]
This is an output device that creates an object by building layer upon layer of material which could be plastic, resin or powdered metal.

(A) ** 3D Printer

(B) 2D Cutter

(C) Actuator

(D) Inkjet Printer

EXPLANATIONS BELOW

Concept note-1: -Powder bed fusion (PBF) is a 3D printing process where a thermal energy source selectively metls powder particles (plastic, metal, or ceramic) inside a build area to create a solid object layer by layer.

Concept note-2: -3D Printer The computer model can be created by using a 3D scanner, or by hand using CAD modeling software. Using a method called additive manufacturing, 3D objects are created by layering a material, layer by layer, from the ground up until the object is completed.

Concept note-3: -A plotter is a printer designed for printing vector graphics. Instead of printing individual dots on the paper, plotters draw continuous lines. This makes plotters ideal for printing architectural blueprints, engineering designs, and other CAD drawings.

Concept note-4: -An output device is any piece of computer hardware converts information into a human-perceptible form or, historically, into a physical machine-readable form for use with other non-computerized equipment. It can be text, graphics, tactile, audio, or video.

Concept note-5: -Printer is an output device, which is used to print information on paper.
