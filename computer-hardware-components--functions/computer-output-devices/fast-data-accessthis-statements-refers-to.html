[SOURCES]
Fast data access.This statements refers to?

(A) ** Memory

(B) Storage

EXPLANATIONS BELOW

Concept note-1: -A memory cache is small, fast, expensive memory placed between the very fast CPU registers and the large slow main memory. Instead of directly interfacing with the main memory, the CPU will first check whether or not a requested item is resident in the cache.

Concept note-2: -(1) Memory access time is how long it takes for a character in RAM to be transferred to or from the CPU. Fast RAM chips have an access time of 10 nanoseconds (ns) or less.

Concept note-3: -Static random-access memory is a type of random-access memory that uses latching circuitry to store each bit. SRAM is a volatile memory; data is lost when power is removed. Hence, Registers provide the fastest access because these memory locations are situated inside the processor.

Concept note-4: -Cache memory, which also is a type of random access memory, does not need to be refreshed. It is built directly into the CPU to give the processor the fastest possible access to memory locations and provides nanosecond speed access time to frequently referenced instructions and data.
