Which of the following is NOT a true statement regarding NTFS?

(A) NTFS allows for file access restriction using permissions.

(B) NTFS uses journaling which allows for data recovery.

(C) NTFS manages data more efficiently than FAT or FAT 32 allowing more data to be stored on a volume.

(D) ** NTFS formatted volumes are limited to simple volumes across one disk.

EXPLANATIONS BELOW

Concept note-1: -NTFS can support volumes as large as 8 petabytes on Windows Server 2019 and newer and Windows 10, version 1709 and newer (older versions support up to 256 TB).

Concept note-2: -Volume mount points: They allow an empty folder in an NTFS-formatted file system to point to another partition or volume in the local computer. The partition connected via the volume mount point can be formatted with FAT, FAT32, or NTFS.
