WHAT IS A HARD DRIVE

[SOURCES]
A technician is attempting to create multiple partitions on a hard disk that supports a maximum partition size of 2TB. What is the maximum number of primary partitions allowed per hard drive?

(A) 1

(B) 32

(C) ** 4

(D) 68

EXPLANATIONS BELOW

Concept note-1: -In order for an operating system to fully support storage devices that have capacities that exceed 2 terabytes (2 TB, or 2 trillion bytes), the device must be initialized by using the GUID Partition Table (GPT) partitioning scheme. This scheme supports addressing of the full range of storage capacity.

Concept note-2: -Primary partition You can create up to four primary partitions on a basic disk. Each hard disk must have at least one primary partition where you can create a logical volume. You can set only one partition as an active partition. Primary partitions are assigned drive letters.

Concept note-3: -GPT disks support up to 128 primary partitions per disk. We assume that you will not take advantage of this because a single partition per disk is recommended for simplicity. GPT disks store multiple copies of the partition table and use cyclical redundancy check (CRC) to protect the partition table.

Concept note-4: -GPT disk provides much more storage space as compared to MBR. Users can create multiple partitions. GPT disk system can create as many as 128 partitions. GPT disk system is a breakthrough when we talk about the limitation of MBR where only 4 primary partitions can be created.
