WHAT IS A HARD DRIVE

[SOURCES]
Which volume type is contained entirely on a single disk drive?

(A) Striped volume

(B) ** Simple volume

(C) Spanned volume

(D) Mirrored volume

EXPLANATIONS BELOW

Concept note-1: -NTFS formatted volumes are limited to simple volumes across one disk.

Concept note-2: -A simple volume is a portion of a physical disk that functions as though it were a physically separate unit. A simple volume can consist of a single region on a disk or multiple regions of the same disk that are linked together.

Concept note-3: -Simple Volume VS Primary Partition The primary partition is a partition which can be used to boot an Operating System, can only be created on a basic disk with MBR or GPT partition table under all Windows systems. Hence, simple volumes are based on dynamic disk while primary partitions are based on basic disk.

Concept note-4: -A basic disk is a type of hard drive configuration, available with the Windows operating system. A basic disk makes use of primary partitions and optionally, an extended partition. The consist of up to four primary partitions or three primary partitions and one extended.

Concept note-5: -A basic disk using the GPT partition style can have up to 128 primary partitions, while dynamic disks will have a single LDM partition as with MBR partitioning. Because basic disks using GPT partitioning do not limit you to four partitions, you do not need to create extended partitions or logical drives.
