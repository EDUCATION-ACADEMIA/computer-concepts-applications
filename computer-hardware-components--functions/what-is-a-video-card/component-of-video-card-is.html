WHAT IS A VIDEO CARD

[SOURCES]
Component of Video Card is

(A) ** Video Memory

(B) Control unit

(C) Storage

EXPLANATIONS BELOW

Concept note-1: -The processing unit on a video card is referred to as a graphics processing unit (GPU). This is very similar to a CPU, but its design is optimized to work with images. The memory on a video card is very similar to the regular random-access memory (or RAM) on a motherboard.

Concept note-2: -Video memory is memory on a video card and in some cases on the motherboard, thatâ€™s accessible by the video and computer processor. With more video memory, the video card and computer are capable of handling more complex graphics at a faster rate.

Concept note-3: -It contains a graphics processing unit (GPU), which is a processor dedicated to creating images; a digital-to-analog converter; and memory chips that store display data.

Concept note-4: -VRAM is a type of volatile memory used in graphics cards. A given graphics card will have a gigabyte measurement listed in its specifications; this is how much VRAM that card has. So while they are related, they are not the same thing.
