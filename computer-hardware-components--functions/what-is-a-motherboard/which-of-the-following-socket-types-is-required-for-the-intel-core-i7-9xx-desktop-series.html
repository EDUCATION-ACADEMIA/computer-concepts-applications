Which of the following socket types is required for the Intel Core i7-9xx desktop series?

(A) ** LGA 1366

(B) LGA 1156

(C) LGA 1155

(D) LGA 775

EXPLANATIONS BELOW

Concept note-1: -Which of the following socket types is required for the Intel Core i7-9xx desktop series? The lower-end Core i7 desktop (non-mobile) processors call for the LGA 1156 socket, but the 9xx series requires the LGA 1366 socket.

Concept note-2: -The LGA1155 socket works with DDR3 memory. The LGA1155 socket was designed to replace the LGA1156 Socket. Does the Sandy Bridge chipset family use two chipset housings on the motherboard or a single chipset housing?

Concept note-3: -LGA 1155, also called Socket H2, is a zero insertion force flip-chip land grid array (LGA) CPU socket designed by Intel for their CPUs based on the Sandy Bridge (2nd Gen) and Ivy Bridge (3rd Gen) microarchitectures.

Concept note-4: -Much like SIMMs, DIMMs can be found in a variety of sizes, the most common being 72 pin DIMMs (36 pins per side, very small, used in laptops), 144 pin DIMMs (72 pins per side), and the now standard 168 pin DIMMs (84 pins per side, used in desktop systems).

Concept note-5: -B. Remember the 8:1 rule. Modules greater than but not including SDR SDRAM are named with a number eight times larger than the number used to name the chips on the module.
