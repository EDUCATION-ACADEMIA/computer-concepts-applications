WHAT IS A MOTHERBOARD

[SOURCES]
The most common RAM slots on consumer motherboards today are

(A) DROM

(B) ** DIMM

(C) SSD

(D) CMOS

EXPLANATIONS BELOW

Concept note-1: -DIMM is a module that contains one or several random access memory (RAM) chips on a small circuit board with pins that connect it to the computer motherboard. The DIMM stores each data bit in a separate memory cell. DIMMs use a 64-bit data path, since processors used in personal computers have a 64-bit data width.

Concept note-2: -DIMM (dual in-line memory module) slots are the place on your motherboard where the RAM goes. As such, you may also see DIMM slots referred to as “RAM slots.” The more DIMM slots your motherboard has, the more RAM you can install.

Concept note-3: -Yes, there were always motherboards with different types of slots, I even remember motherboards with both SIMM and DIMM slots.

Concept note-4: -Most motherboards will either have 2, 4, or 8 dimm slots for RAM memory. If your motherboard has 4 slots, you could buy four 32 GB memory modules for a total of 128 GB. The kit below is $580 on Newegg.com.
