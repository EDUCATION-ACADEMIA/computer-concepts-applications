How do we measure Processor Speed?

(A) ** Gigahertz

(B) Gigahurts

(C) Gigabytes

(D) Giga-burgers

EXPLANATIONS BELOW

Concept note-1: -The clock speed measures the number of cycles your CPU executes per second, measured in GHz (gigahertz). In this case, a “cycle” is the basic unit that measures a CPU’s speed. During each cycle, billions of transistors within the processor open and close .

Concept note-2: -If you’re wondering how to check your clock speed, click the Start menu (or click the Windows key) and type “System Information.” Your CPU’s model name and clock speed will be listed under “Processor.”

Concept note-3: -Ghz Meaning and Processor Speed A 2.6-Ghz processor, therefore, can run 2.6 billion instructions in a second, while a 2.3-Ghz processor can run 2.3 billion instructions per second. This isn’t that significant of a difference, but it’s something to consider, if you’re comparing two computers with those specifications.

Concept note-4: -So, that 2.5 GHz processor of yours can process 2.5 billion processor sub-steps each and every second. Wow, that’s pretty impressive.

Concept note-5: -What Does a 2.4 GHz Processor Mean? A hertz (Hz) means per second (the number of clock periods), while Giga (G) indicates a billion. Therefore, a 2.4 GHz processor runs 2.4 billion times per second to complete a timer cycle. This processing speed is less powerful than processors rated 3 GHz and above.
