Which packaging is used for DDR SDRAM memory?

(A) 168-pin DIMM

(B) 72-pin SIMM

(C) ** 184-pin DIMM

(D) 240-pin DIMM

EXPLANATIONS BELOW

Concept note-1: -DIMMs are 64-bit memory modules that are used as a package for SDRAM, DDR, and DDR2 . Dual means the function of the pins on one side of the module from the corresponding pins on the other side. With 84 pins per side, this makes 168 independent pins on each standard memory module.

Concept note-2: -Most DDR SDRAM uses a Thin Small Outline Package (TSOP). TSOP chips have fairly long contact pins on each side.

Concept note-3: -184-pin DIMMs are used to provide DDR SDRAM memory for desktop computers. Each 184-pin DIMM provides a 64-bit data path, so they are installed singly in 64-bit systems. 184-pin DIMMs are available in PC2100 DDR SDRAM or PC2700 DDR SDRAM.

Concept note-4: -184-pin DIMMs are for DDR SDRAM modules found in Pentium III and 4 and Athlon systems. Each 184-pin DIMM provides a 64-bit data path, so they are installed alone in 64-bit systems.

Concept note-5: -Most 168-pin DIMM modules use SDRAM memory.
