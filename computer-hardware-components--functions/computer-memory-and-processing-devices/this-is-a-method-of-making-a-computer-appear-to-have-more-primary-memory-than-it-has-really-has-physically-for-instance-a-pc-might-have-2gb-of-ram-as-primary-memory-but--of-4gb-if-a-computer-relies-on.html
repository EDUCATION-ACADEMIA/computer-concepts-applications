This is a method of making a computer appear to have more primary memory than it has really has physically. For instance a PC might have 2Gb of RAM as primary memory, but ____ of 4Gb. If a computer relies on this memory it will be slow.

(A) ** Virtual Memory

(B) Random Access Memory

(C) Read Only Memory

(D) Cache Memory

EXPLANATIONS BELOW

Concept note-1: -Virtual memory is a common technique used in a computer’s operating system (OS). Virtual memory uses both hardware and software to enable a computer to compensate for physical memory shortages, temporarily transferring data from random access memory (RAM) to disk storage.

Concept note-2: -RAM (Random Access Memory) is the hardware in a computing device where the operating system (OS), application programs and data in current use are kept so they can be quickly reached by the device’s processor. RAM is the main memory in a computer.

Concept note-3: -RAM. Random access memory (RAM) is the primary internal memory of the central processing unit (CPU). Your electronic device uses it for storing temporary data. It does this by providing applications with a place to store data you’re actively using so that it can quickly access the data.

Concept note-4: -The main difference between physical and virtual memory is that the physical memory refers to the actual RAM of the system attached to the motherboard, but the virtual memory is a memory management technique that allows the users to execute programs larger than the actual physical memory.
