COMPUTER MEMORY AND PROCESSING DEVICES

[SOURCES]
A computer has 1024 megabytes of RAM. That is the same as ____

(A) ** 1 Gb

(B) 1000 Gb

(C) 1 Kb

(D) 1 Tb

EXPLANATIONS BELOW

Concept note-1: -Detailed Solution. The size of information in the computer is measured in Kilobytes (KB), Megabytes (MB), Gigabytes (GB), and Terabytes (TB). One Megabyte (MB) is about 1 million bytes (or about 1024 Kilobytes (KB)). One Gigabyte (GB) is about 1 billion bytes, or (1024 Megabytes MB).

Concept note-2: -In this convention, one thousand and twenty-four megabytes (1024 MB) is equal to one gigabyte (1 GB), where 1 GB is 10243 bytes (i.e., 1 GiB).

Concept note-3: -A gigabyte is equivalent to the following standard measurements: 1, 000 megabytes (decimal) or 1, 024 megabytes (binary); 1, 000, 000 kilobytes (decimal) or 1, 048, 576 kilobytes (binary); and. 1, 000, 000, 000 bytes (decimal) or 1, 073, 741, 824 bytes (binary).

Concept note-4: -The SI unit prefix GIGA means x1000. Computers use binary, so the closest binary whole binary number to that is 1024. Long ago, it was decided to use 1024, because it was easy computationally.

Concept note-5: -A GB (gigabyte) is a way of measuring how much data you have on an electronic device. 1GB is approximately 1, 000MB (megabytes).
