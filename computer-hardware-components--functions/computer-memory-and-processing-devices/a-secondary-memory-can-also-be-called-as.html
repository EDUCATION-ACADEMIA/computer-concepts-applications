COMPUTER MEMORY AND PROCESSING DEVICES

[SOURCES]
A secondary memory can also be called as

(A) ** Backup Memory

(B) Main memory

(C) Internal memory

EXPLANATIONS BELOW

Concept note-1: -Secondary memory is known as a Backup memory or Additional memory or Auxiliary memory. Hard Drive, SSD, Flash, Optical Drive, USD Drive are some examples of secondary memories in computer.

Concept note-2: -Secondary memory is known as a Backup memory or Additional memory or Auxiliary memory. Data is directly accessed by the processing unit. Data cannot be accessed directly by the processor. It is first copied from secondary memory to primary memory.

Concept note-3: -Programs and data are stored in secondary memory. Auxiliary memory is another name for it. It differs from primary memory in that it is non-volatile and not directly accessible by the CPU.

Concept note-4: -In secondary memory, programs and data are stored. It is also called auxiliary memory.

Concept note-5: -Organizations typically use secondary storage for backup and disaster recovery (DR) data, archival data, or noncritical active data. Secondary storage is also referred to as auxiliary storage.
