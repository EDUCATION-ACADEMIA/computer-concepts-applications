COMPUTER SYSTEM COMPONENTS

[SOURCES]
____ is nonvolitile and refers to memory chips storing permanent data and instructions.

(A) RAM

(B) ** read-only memory (ROM)

(C) flash memory

(D) firmware

EXPLANATIONS BELOW

Concept note-1: -Read-only memory, or ROM, is a type of computer storage containing non-volatile, permanent data that, normally, can only be read, not written to. ROM contains the programming that allows a computer to start up or regenerate each time it is turned on.

Concept note-2: -ROM (Read-only memory) refers to memory chips storing permanent data and instructions. The data cannot be modified on most ROM chips and is nonvolatile. ROM chips called firmware contain permanently written data, instructions, or information.

Concept note-3: -Non-volatile memory is memory that retains its values even when power is removed. Earlier forms of non-volatile memory included various forms of read-only memory (ROM).

Concept note-4: -Non-volatile memory, on the other hand, is static. It remains in a computer even after a user switches it off. HDD and ROM are types of non-volatile memory.

Concept note-5: -Volatile Memory is used to store computer programs and data that CPU needs in real time and is erased once computer is switched off. RAM and Cache memory are volatile memory. Where as Non-volatile memory is static and remains in the computer even if computer is switched off. ROM and HDD are non-volatile memory.
