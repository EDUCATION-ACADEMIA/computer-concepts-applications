Why is it that Hard Disk Drive cannot give CPU data?

(A) They cannot understand each other.

(B) ** Hard Disk drives work much more slowly than CPU and it wastes a lot of time of the CPU waiting for the data to arrive.

(C) Hard disk Drive has other roles that keeps it very busy.

(D) Hard Disk Drive is not in the computer.

EXPLANATIONS BELOW

Concept note-1: -This is because using a HDD as the primary memory would cause a computer to perform much slower (a HDD or SSD is not directly accessible to the CPU, and isn’t as fast as RAM).

Concept note-2: -A slow computer is often caused by too many programs running simultaneously, taking up processing power and reducing the PC’s performance. Some programs will continue running in the background even after you have closed them or will start automatically when you boot up your computer.

Concept note-3: -The CPU can only access its registers and main memory. It cannot, for example, make direct access to the hard drive, so any data stored there must first be transferred into the main memory chips before the CPU can work with it.

Concept note-4: -The main reasons computers slow down over time are: large amounts of data stored on the hard disk drive, unnecessary software and obsolete data, lack of sufficient RAM, constantly updating software, and viruses.

Concept note-5: -Computers do tend to slow down as the hard drive fills up. Some of this is unrelated to the hard drive; as they age, operating systems get bogged down with extra programs and files that slow down the computer. However, hard drives do need empty space for virtual memory.
