In PHP assigning value to a variable may look like this:

(A) $ ____ myname = “Vanness”

(B) myname = “Vanness”

(C) ** $myname = “Vanness”

(D) var myname = “Vanness”

EXPLANATIONS BELOW

Concept note-1: -Rules for PHP variables: A variable starts with the $ sign, followed by the name of the variable. A variable name must start with a letter or the underscore character. A variable name cannot start with a number.

Concept note-2: -The $var (single dollar) is a normal variable with the name var that stores any value like string, integer, float, etc.

Concept note-3: -Definition and Usage The var keyword creates a property in a class. Since PHP 5, it is equivalent to the public keyword. Note: The var keyword is only used for compatibility reasons. Since PHP 5, the keywords private, protected and public should be used instead.

Concept note-4: -It means pass the variable by reference, rather than passing the value of the variable.
