Match the followings between column A and column B.Column A Column B i) TCP M) Physical Layer ii) NIC N) Data Link layer iii) Repeaters O) Network Layer iv) Router P) Transport Layer v) SMTP Q) Application

(A) i, M; ii, N; iii, O; iv, P; v, Q

(B) ** i, P; ii, N; iii, Q; iv, O; v, Q

(C) i, N; ii, Q; iii, M; iv, O, v, P

(D) i, O; ii, Q; iii, M; iv, N; v, P

EXPLANATIONS BELOW

Concept note-1: -Layer 4 of the OSI Model: Transport Layer provides transparent transfer of data between end users, providing reliable data transfer services to the upper layers. The transport layer controls the reliability of a given link through flow control, segmentation and desegmentation, and error control.

Concept note-2: -The physical layer of TCP/IP describes hardware standards such as IEEE 802.3, the specification for Ethernet network media, and RS-232, the specification for standard pin connectors.

Concept note-3: -Network layer: Handles the routing and sending of data between different networks. The most important protocols at this layer are IP and ICMP. 2. Data link layer: Handles communications between devices on the same network.

Concept note-4: -The Internet Layer of the TCP/IP model aligns with the Layer 3 (Network) layer of the OSI model. This is where IP addresses and routing live. When data is transmitted from a node on one LAN to a node on a different LAN, the Internet Layer is used.
