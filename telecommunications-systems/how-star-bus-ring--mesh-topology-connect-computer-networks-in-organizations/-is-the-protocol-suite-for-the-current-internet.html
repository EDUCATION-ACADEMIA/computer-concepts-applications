____ is the protocol suite for the current Internet.

(A) ** TCP/IP

(B) NCP

(C) UNIX

(D) ACM

EXPLANATIONS BELOW

Concept note-1: -TCP/IP stands for Transmission Control Protocol/Internet Protocol and is a suite of communication protocols used to interconnect network devices on the internet. TCP/IP is also used as a communications protocol in a private computer network (an intranet or extranet).

Concept note-2: -TCP/IP, also known as Internet Protocol Suite, is a collection of networking protocols that work in tandem to transfer a data packet from one computer to another using computer networks.

Concept note-3: -A minimal implementation of TCP/IP includes the following: Internet Protocol (IP), Address Resolution Protocol (ARP), Internet Control Message Protocol (ICMP), Transmission Control Protocol (TCP), User Datagram Protocol (UDP), and Internet Group Management Protocol (IGMP).

Concept note-4: -The TCP/IP protocol suite is a collection of protocols that are used on the Internet. It is named after two of the main protocols (TCP and IP) and uses a 4 layer networking model.
