A modem contained within a computer system unit is called a/an ____ modem.

(A) External

(B) ** Internal

(C) Wireless

(D) Wired

EXPLANATIONS BELOW

Concept note-1: -A modem that is contained within the system unit is called a(n) modem. a) external b) internal c) wireless Page 7 d) WiFI Answer: B Difficulty level: Easy Page: 240 Response: A modem contained within the system unit is an internal modem.

Concept note-2: -A modulator-demodulator or modem is a computer hardware device that converts data from a digital format into a format suitable for an analog transmission medium such as telephone or radio.

Concept note-3: -The main internal components of a computer system are: Processor. Main memory. Input/output (I/O) controllers.

Concept note-4: -A Modem is considered an input and output device because it sends data (upload/output) and receives data (download/input).
