HOW STAR BUS RING MESH TOPOLOGY CONNECT COMPUTER NETWORKS IN ORGANIZATIONS

[SOURCES]
Data is transmitted in both directions, but not at the same time.

(A) Full Duplex

(B) Simplex

(C) ** Half Duplex

(D) Half Synchronous

EXPLANATIONS BELOW

Concept note-1: -Half-duplex data transmission allows data to flow in both directions but only in one direction at a time. Semi-Duplex is another name for it. In other words, each station can transmit and receive data at the same time, but not simultaneously.

Concept note-2: -Half-Duplex In other words, each station can both transmit and receive the data but not at the same time. When one device is sending the other can only receive and vice-versa. In this type of transmission mode, the entire capacity of the channel can be utilized for each direction.

Concept note-3: -Half-duplex means that between two devices, only one device is capable of transmitting and receiving data on a network or bus at one time. Half-duplex is bi-directional, meaning that data can be sent between devices both ways, but the data can only go in one direction at a time.

Concept note-4: -Half duplex mode In this mode, the connected devices can transmit or receive the data but not simultaneously. In half duplex mode, the direction of communication can be reversed as the radio stations can receive as well as transmit the data.

Concept note-5: -A full duplex communication channel is able to transmit data in both directions on a signal carrier at the same time.
