Which network can deny requests from users if they do not have the right access level?

(A) ** Client server network

(B) Peer to peer network

(C) Both client server and peer to peer

EXPLANATIONS BELOW

Concept note-1: -This policy setting prohibits users from connecting to a computer from across the network, which would allow users to access and potentially modify data remotely. In high security environments, there should be no need for remote users to access data on a computer.

Concept note-2: -An access control list (ACL) is a list of rules that specifies which users or systems are granted or denied access to a particular object or system resource. Access control lists are also installed in routers or switches, where they act as filters, managing which traffic can access the network.

Concept note-3: -Physical access control is utilizing physical barriers which can help prevent unauthorized users from accessing systems.

Concept note-4: -The process of preventing unauthorized users and devices from gaining access to a corporate or private network is known as Network Access Control (NAC). NAC ensures that only authenticated users and devices that are authorized and in compliance with security policies can access the network.
