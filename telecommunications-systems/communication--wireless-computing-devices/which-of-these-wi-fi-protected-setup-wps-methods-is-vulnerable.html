Which of these Wi-Fi Protected Setup (WPS) methods is vulnerable?

(A) ** PIN method

(B) push-button method

(C) piconet method

(D) NFC method

EXPLANATIONS BELOW

Concept note-1: -Overview. Wi-Fi Protected Setup (WPS) provides simplified mechanisms to configure secure wireless networks. The external registrar PIN exchange mechanism is susceptible to brute-force attacks that could allow an attacker to gain access to an encrypted Wi-Fi network.

Concept note-2: -The PIN (Personal Identification Number) Method is one of the connection methods developed by the Wi-Fi Alliance. By inputting a PIN which is created by an Enrollee (your machine) to the Registrar (a device that manages the wireless LAN), you can setup the wireless network and security settings.

Concept note-3: -WPS can be implemented in modes: push button configuration (PBC), Pin input configuration (PIN), and AP PIN. PBC: Press a WPS button to achieve a secure WPS connection. PIN: Enter an STA PIN on an ONT to achieve a secure WPS connection.
