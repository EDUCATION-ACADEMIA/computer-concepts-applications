With reference to an IP address indicate which of the following are not true.

(A) A public IP Address is one given to your router by your ISP.

(B) A private IP Address is one given by your router to individual devices connected to it.

(C) ** The IP Address can be stored in 3 Bytes

(D) IP Address are usually dynamic-they change each time a device is disconnected and then reconnected to a network.

EXPLANATIONS BELOW

Concept note-1: -1 Answer. To elaborate: It need not be that a single hostname will correspond to a ip address.

Concept note-2: -An IPv4 address is typically written in decimal digits, formatted as four 8-bit fields that are separated by periods. Each 8-bit field represents a byte of the IPv4 address. This form of representing the bytes of an IPv4 address is often referred to as the dotted-decimal format.

Concept note-3: -An IPv4 address is 32 bits. An IP Address is shown as 4 decimal numbers representing 4 bytes: d.d.d.d where d = decimal number (0-255). High order bits are the network identifier and lower order bits are the host identifier.

Concept note-4: -An IP address has two parts: the network ID, comprising the first three numbers of the address, and a host ID, the fourth number in the address. So on your home network-192.168.1.1, for example â€“ 192.168.1 is the network ID, and the final number is the host ID.
