As you read your email, you have some that have important information you will need to access later. Which of these can you do to make that email easier to locate at a later time?

(A) Create a list

(B) Sort your messages by name

(C) ** Set a flag / Flag the message

(D) Download the email and print it

EXPLANATIONS BELOW

Concept note-1: -Click File > Options > Advanced. Under Outlook panes, click Reading Pane. Select the Mark items as read when viewed in the Reading Pane check box.

Concept note-2: -Quickly create an email about a meeting/appointment, task, or note by clicking on an item, and dragging it onto your inbox or mail tab: This creates a new message with the item’s title as the subject, and the item’s details as the body.

Concept note-3: -The inbox is where you’ll view and manage emails you receive. Emails are listed with the name of the sender, the subject of the message, and the date received.

Concept note-4: -To quickly create a rule that moves all email from a specific sender or a set of senders to a folder, right-click a message in your message list that you want to create a rule for, and select Create rule. Choose the folder where you want all messages from that sender or set of senders to be moved, and then select OK.
