What is the purpose for the World Wide Web Consortium (W3C)?

(A) ** To develop standards for the World Wide Web.

(B) To prosecute copyright violations.

(C) They host the entire Internet on a single web server.

(D) They decide which websites should be included in the World Wide Web.

EXPLANATIONS BELOW

Concept note-1: -The W3C (World Wide Web Consortium) is an international organization that creates standards for the World Wide Web. The WC3 is committed to improving the web by setting and promoting web-based standards. The W3Câ€™s goal is to create technical standards and guidelines for web technologies worldwide.

Concept note-2: -The World Wide Web Consortium (W3C) is an international community where Member organizations, a full-time staff, and the public work together to develop Web standards.

Concept note-3: -The World Wide Web Consortium was created in October 1994 to lead the World Wide Web to its full potential by developing common protocols that promote its evolution and ensure its interoperability.

Concept note-4: -The standards set out but the W3C are invaluable because they ensure a fair and accessible web. Regardless of who you are, or where you are, if you create a web application using the standards, you know that it will work with all others that are W3C compliant, and you know that everyone can access it.

Concept note-5: -The W3C works to develop and maintain standards for key technologies used on the Web, including HTML (Hypertext Markup Language), CSS (Cascading Style Sheets), and JavaScript.
