THE INTERNET

[SOURCES]
____ is a sequence of numbers that uniquely identifies the location of each computer or device connected to the Internet.

(A) web address

(B) computer address

(C) ** IP address

(D) network address

EXPLANATIONS BELOW

Concept note-1: -IP address, in fullInternet Protocol address, number that uniquely identifies each computer on the Internet. A computer’s IP address may be permanently assigned or supplied each time that it connects to the Internet by an Internet service provider.

Concept note-2: -IP address: An IP address is a unique number assigned to every device connected to a network that uses the Internet Protocol for communication. Each IP address identifies the device’s host network and the location of the device on the host network.

Concept note-3: -The IPv4 address or the Internet Protocol Address is the fourth version of the Internet Protocol. IPv4 addresses are 32-bit addresses that are unique to every host or device on the internet. The IP address allows the host to be connected to other devices on the internet to communicate with them.

Concept note-4: -IP address stands for internet protocol address; it is an identifying number that is associated with a specific computer or computer network. When connected to the internet, the IP address allows the computers to send and receive information.

Concept note-5: -A server is a computer program or device that provides a service to another computer program and its user, also known as the client.
