(A) ** An IP address

(B) Someones phone number

EXPLANATIONS BELOW

Concept note-1: -An IP address is a string of numbers separated by periods. IP addresses are expressed as a set of four numbers-an example address might be 192.158.1.38. Each number in the set can range from 0 to 255. So, the full IP addressing range goes from 0.0.0.0 to 255.255.255.255.

Concept note-2: -An internet protocol (IP) address allows computers to send and receive information. There are four types of IP addresses: public, private, static, and dynamic.

Concept note-3: -An Internet Protocol address (IP address) is a numerical label such as 192.0.2.1 that is connected to a computer network that uses the Internet Protocol for communication. An IP address serves two main functions: network interface identification and location addressing.

Concept note-4: -Reserved Addresses 0.0” and “255.255. 255.255.” Another number, “127.0. 0.1” is called the “localhost; ” every computer on a network refers to itself as this address. Because these numbers have special meanings, the network does not assign them to PCs; such addresses would be invalid.

Concept note-5: -An Internet Protocol (IP) address is a unique sequence of values separated by periods that is used to identify a device on a network. IP addresses are used globally by the Internet, and locally by the devices on your network to ensure that network data is sent to its intended recipient.
