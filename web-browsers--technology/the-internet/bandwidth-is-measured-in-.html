(A) Megabytes

(B) ** Bitrate

(C) Binary

(D) Routers

EXPLANATIONS BELOW

Concept note-1: -We use the term bandwidth to describe the maximum bit rate of a system. If a network connection has a bandwidth of 100 Mbps, that means it canâ€™t transfer more than 100 megabits per second.

Concept note-2: -Network bandwidth is commonly measured in bits per second (bps). In practice, organizations and internet service providers (ISPs) measure bandwidth in megabits per second (Mbps) or gigabits per second (Gbps).

Concept note-3: -Bandwidth can be measured in bits per second (bps) megabits per second (Mbps) and gigabits per second (Gbps).

Concept note-4: -While bandwidth is traditionally expressed in bits per second (bps), modern network links now have far greater capacity, which is why bandwidth is now more often expressed as Mbps or Gbps.

Concept note-5: -The bit rate is expressed in the unit bit per second (symbol: bit/s), often in conjunction with an SI prefix such as kilo (1 kbit/s = 1, 000 bit/s), mega (1 Mbit/s = 1, 000 kbit/s), giga (1 Gbit/s = 1, 000 Mbit/s) or tera (1 Tbit/s = 1, 000 Gbit/s).
