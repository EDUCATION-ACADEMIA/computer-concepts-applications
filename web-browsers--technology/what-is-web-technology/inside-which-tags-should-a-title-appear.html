Inside which tags should a <title> appear?

(A) <html> </html>

(B) ** <head> </head>

(C) <p> </p>

(D) <h1> </h1>

EXPLANATIONS BELOW

Concept note-1: -The h1 element appears on the page when loaded in the browser-generally this should be used once per page, to mark up the title of your page content (the story title, or news headline, or whatever is appropriate to your usage.)

Concept note-2: -The <head> element is a container for metadata (data about data) and is placed between the <html> tag and the <body> tag. Metadata is data about the HTML document. Metadata is not displayed. Metadata typically define the document title, character set, styles, scripts, and other meta information.

Concept note-3: -Yes the title tag should only be used in the head .
