What are the two main ways in which AI operates?

(A) ** Using symbolic and machine learning

(B) Using facial recognition and symbolic learning

(C) Using machine learning and large data sets

(D) ** Using large data sets and facial recognition

EXPLANATIONS BELOW

Concept note-1: -Building AI face recognition systems is possible in two ways: Use of ready-made pre-trained face recognition deep learning models. Models such as DeepFace, FaceNet, and others are specially designed for face recognition tasks. Custom model development.

Concept note-2: -The main facial recognition methods are feature analysis, neural network, eigen faces, and automatic face processing. Although facial recognition technology has come a long way, there is still a need for enhancements to prove accuracy and reliability.

Concept note-3: -The most common type of machine learning algorithm used for facial recognition is a deep learning Convolutional Neural Network (CNN). CNNs are a type of artificial neural network that are well-suited for image classification tasks.

Concept note-4: -It works by identifying and measuring facial features in an image. Facial recognition can identify human faces in images or videos, determine if the face in two images belongs to the same person, or search for a face among a large collection of existing images.
