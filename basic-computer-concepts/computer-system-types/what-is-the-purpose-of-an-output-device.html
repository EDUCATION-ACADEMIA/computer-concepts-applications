What is the purpose of an output device?

(A) A device that allows users to do a specific task

(B) A device that allows memory to store data

(C) ** A device that can display information to people

(D) A device that helps the printer to print the output

EXPLANATIONS BELOW

Concept note-1: -Output devices convert the computer data to human understandable form. We give input to the computer using input devices and the computer performs operations on the data and displays the output to the user using the output device.

Concept note-2: -An output device allows data to be transmitted by the computer in a human-friendly form, for example, sound being played through a speaker. Monitor-The main output device of a computer. It forms images by converting electrical energy into light in the form of tiny dots on the screen called pixels.

Concept note-3: -A standard output device is the computer monitor, which displays text, images, and user interface elements by lighting up thousands of pixels with different colors.

Concept note-4: -Examples include monitors, printers, speakers, headphones, projectors, GPS devices, optical mark readers, and braille readers.
