What are some uses for a Print Server (3 answers)

(A) Allows for remote storage of data rather than local

(B) ** Provides access to printers on a network

(C) ** Allow for greater manageability of devices

(D) Used for the retrieval and delivery of emails

(E) ** Admins can pause, delete, resume and view jobs

EXPLANATIONS BELOW

Concept note-1: -The server connects with computers over a network and allows them to communicate with available printers. The print server helps printers avoid overloads. It queues print jobs and manages their distribution to devices to ensure orderly operation and to avoid overloading printer hardware.

Concept note-2: -Answer: A print server can provide client computers access to print resources, manage print jobs by storing them in a queue until the print device is ready for them, and provide print job status messages to users.
