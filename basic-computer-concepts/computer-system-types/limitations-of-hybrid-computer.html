Concept note-1: -Mostly hybrid computers are way too expensive because it is hybrid and there are different networks, hubs and it requires a lot of cables, expensive circuits which makes it costly for any individual.

Concept note-2: -Without an operating system computer hardware is only an inactive electronic machine, which is inconvenient to user for execution of programs. As the computer hardware or machine understands only the machine language. It is difficult to develop each and every program in machine language in order to execute it.

Concept note-3: -Hybrid computers are well known for their ability to blend analog and digital features of computers. Advantages include the availability of both analog and digital computations within a single unit and an efficient processing speed.

Concept note-4: -Working at a computer can cause back, neck and shoulder pains, headache, eyestrain and overuse injuries of the arms and hands. You can help avoid computer-related injuries with proper furniture, better posture and good working habits.
