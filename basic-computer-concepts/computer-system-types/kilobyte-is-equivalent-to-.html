COMPUTER SYSTEM TYPES

[SOURCES]
kilobyte is equivalent to ____

(A) ** 1000 bytes

(B) 1000 megabytes

(C) 1000 kilobytes

(D) 1000 gigabytes

EXPLANATIONS BELOW

Concept note-1: -One kilobyte (KB) is a collection of about 1000 bytes.

Concept note-2: -According to the International System of Units (SI), kilo prefix is specified as 1000 (103). Accordingly, 1 kilobyte is equal to 1000 bytes.

Concept note-3: -Remember: Computers can only work with binary code. However computers do like the number 1024 because its binary code is 10000000000. That’s why there are 1024 Bytes in a KB, 1024 KB in a MB and so on…

Concept note-4: -The International System of Units defines the prefix kilo as 1000. A kilobyte file is 1024 bytes in size. One kilobyte is equal to 8000 bits.

Concept note-5: -A kilobyte (KB) is 1, 024 bytes, not one thousand bytes as might be expected, because computers use binary (base two) math, instead of a decimal (base ten) system. Computer storage and memory is often measured in megabytes (MB) and gigabytes (GB). A medium-sized novel contains about 1 MB of information.
