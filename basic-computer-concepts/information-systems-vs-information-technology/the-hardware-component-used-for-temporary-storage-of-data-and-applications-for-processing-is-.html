INFORMATION SYSTEMS VS INFORMATION TECHNOLOGY

[SOURCES]
The hardware component used for temporary storage of data and applications for processing is ____

(A) Hard disk

(B) Processor

(C) ** Random access memory

(D) Memory card

EXPLANATIONS BELOW

Concept note-1: -RAM (Random Access Memory) is the hardware in a computing device where the operating system (OS), application programs and data in current use are kept so they can be quickly reached by the device’s processor. RAM is the main memory in a computer.

Concept note-2: -RAM. RAM–or dynamic RAM–is temporary memory storage that makes information immediately accessible to programs; RAM is volatile memory, so stored data is cleared when the computer powers off. Hard drive.

Concept note-3: -The hardware component used for temporary storage of data and applications for processing is: hard-disk.

Concept note-4: -RAM. Random access memory (RAM) is the primary internal memory of the central processing unit (CPU). Your electronic device uses it for storing temporary data. It does this by providing applications with a place to store data you’re actively using so that it can quickly access the data.

Concept note-5: -A hard disk drive or solid state drive holds all of the data; files, photos, programs, music, and movies, that the user wants to keep. Removable, external media storage devices such as flash drives and read/writeable CDs and DVDs are also secondary storage.
