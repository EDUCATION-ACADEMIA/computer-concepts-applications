What utilities are used in recovering from system failure?

(A) ** Windows 8 Startup Settings

(B) Diagnostic Tools

(C) ** System Restore

(D) ** Automated System Recovery

EXPLANATIONS BELOW

Concept note-1: -The RECOVER utility recovers data to the current state or to a previous point in time by restoring a copy and then applying log records. The RECOVER utility can also recover data to a previous point in time by backing out committed work.

Concept note-2: -The System Recovery Options menu contains several tools, such as Startup Repair, that can help you recover Windows from a serious error. This set of tools is on your computer’s hard disk and on the Windows installation disc. Note: You can also create a system repair disc that contains the System Recovery Options menu.

Concept note-3: -ASR involves two main processes: ASR backup The process of creating an ASR set, which consists of a 1.44 MB floppy diskette and a linked backup media containing ASR-created backup data. These two components are necessary for performing an ASR restore and must be kept together.

Concept note-4: -Automated system recovery (ASR) is a feature of the Windows XP operating system that can be used to simplify recovery of a computer’s system or boot volumes. ASR consists of two parts: an automated backup, and an automated restore.
