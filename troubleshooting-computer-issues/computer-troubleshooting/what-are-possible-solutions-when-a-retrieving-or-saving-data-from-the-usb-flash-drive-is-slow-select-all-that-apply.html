COMPUTER TROUBLESHOOTING

[SOURCES]
What are possible solutions when a Retrieving or saving data from the USB flash drive is slow? (Select all that apply).

(A) ** Replace the motherboard with a USB 3.0 capable motherboard

(B) ** Add a USB 3.0 expansion card

(C) ** Set the port speed in the BIOS settings to high speed

(D) Reconfigure the media reader in the BIOS settings

(E) Replace the removable external drive

EXPLANATIONS BELOW

Concept note-1: -Update USB Drive Driver or Its Chipset Drivers. If you want to want to solve the problem of “USB transfer speed slow", you can try to update the USB driver or the chipset driver. There is a utility to help you. You can use EaseUS DriverHandy to update your USD drive driver.

Concept note-2: -Set the file system as NTFS, click start, check the Quick Format box and click OK to format your USB. Once you’ve done this your USB device will be much faster, and you can even apply the recovered files back into the flash drive if you want to.

Concept note-3: -Most likely if you are experiencing slow speeds, it is set to the default Quick removal. Simply switch the setting to Better performance and select OK. You will probably be prompted to restart the computer, and after you do, you should start to see substantially faster transfer speeds!
