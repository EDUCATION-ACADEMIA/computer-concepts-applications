USING MICROSOFT WORD

FORMATTING TEXT IN MICROSOFT WORD

[SOURCES]
You must always save your work to your ____

(A) template (T:)

(B) ** p drive (P:)

EXPLANATIONS BELOW

Concept note-1: -Windows 10 Right-click on the Documents option and left-click Properties from the pop-up menu. Left-click on the Location tab, enter the path of your P: drive into Target Location. Click OK to save the new configuration.

Concept note-2: -The P drive is a folder on a file share that is used for secure file storge. The P drive is: Private: each individual will have their own folder that is only accessible to that user.

Concept note-3: -The “P” or “Projects” drive is a place where all project related documents should be saved. To access the P drive, click on Start then Computer. Look for “Projects” or (P:) under the Network locations section.

Concept note-4: -Next to the C drive, the D drive is the first storage drive available for files and data management. Hence, if asked for system files, the C drive is the best option, while the D drive is the best option for personal files.
