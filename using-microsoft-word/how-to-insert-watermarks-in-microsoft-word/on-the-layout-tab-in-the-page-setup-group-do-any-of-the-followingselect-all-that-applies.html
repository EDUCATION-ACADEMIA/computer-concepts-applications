USING MICROSOFT WORD

HOW TO INSERT WATERMARKS IN MICROSOFT WORD

[SOURCES]
On the Layout tab, in the Page Setup group, do any of the following:*select all that applies*

(A) ** Click the Margins button, and then click Custom Margins.

(B) ** Click the Size button, and then click More Paper Sizes.

(C) A and B

(D) ** Click the Page Setup dialog box launcher.

EXPLANATIONS BELOW

Concept note-1: -The Page Layout Tab holds all the options that allow you to arrange your document pages just the way you want them. You can set margins, apply themes, control of page orientation and size, add sections and line breaks, display line numbers, and set paragraph indentation and lines.

Concept note-2: -Layout tab. The Layout tab includes a selection tool for cells, rows, columns, and the entire table; tools to delete or insert rows and columns; tools to merge or divide cells; cell alignment and margin settings; the typography controls for Open Type fonts.

Concept note-3: -The “Page Setup” group on the “Page Layout” tab of the Ribbon contains buttons that let you make changes to the page setup of the document. In addition to these buttons, you can also click the “Page Setup” dialog box button in the lower right corner of the “Page Setup” group to open the “Page Setup” dialog box.

Concept note-4: -The Page Layout Tab consists of five major groups: Themes, Page Setup, Scale to Fit, Sheet Options, and Arrange.
