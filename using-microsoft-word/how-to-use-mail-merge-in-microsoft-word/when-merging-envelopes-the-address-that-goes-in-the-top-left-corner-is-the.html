HOW TO USE MAIL MERGE IN MICROSOFT WORD

[SOURCES]
When merging envelopes, the address that goes in the top left corner is the:

(A) ** Return Address

(B) Letter Address

EXPLANATIONS BELOW

Concept note-1: -The return address should be written in the upper left-hand corner of the envelope. Some classes or types of mail do require a valid return address.

Concept note-2: -Business Mail 101 The Postal Service encourages mailers to use return addresses because if the piece is undeliverable we can return it. The return address has the same elements as the Delivery Address and must be placed in the upper left corner of the address side or in the upper left of the addressing area.

Concept note-3: -A return address tells the USPS where the sender wants the mail returned if it is undeliverable and is required on certain types of mail.
