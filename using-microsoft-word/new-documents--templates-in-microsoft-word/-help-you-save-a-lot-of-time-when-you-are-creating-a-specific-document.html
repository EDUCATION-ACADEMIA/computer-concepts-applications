USING MICROSOFT WORD

NEW DOCUMENTS TEMPLATES IN MICROSOFT WORD

[SOURCES]
____ help you save a lot of time when you are creating a specific document.

(A) Tell me feature

(B) ** Templates

(C) smart lookup

(D) Co-authoring

EXPLANATIONS BELOW

Concept note-1: -Well thought-out templates mean authors are able to concentrate on content rather than document format and layout. And it means they’re not confronted with a blank page which gives them no idea of where to start. It means your valuable resources save valuable time. And they’ll be far less stressed in the process.

Concept note-2: -Save a document as a template On the File menu, click Save As. On the Format pop-up menu, click Word Template (.dotx). In the Save As box, type the name that you want to use for the new template, and then click Save.

Concept note-3: -A template is a document type that creates a copy of itself when you open it. For example, a business plan is a common document that is written in Word. Instead of creating the structure of the business plan from scratch, you can use a template with predefined page layout, fonts, margins, and styles.

Concept note-4: -They help you save time If you have items that you use on a consistent basis, you should create a template for them. For example, if you have a document that you send to all new clients or email(s) that you send regularly, you should create a template to prevent you from taking the time to recreate them each time.
