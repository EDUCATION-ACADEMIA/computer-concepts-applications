USING MICROSOFT WORD

NEW DOCUMENTS TEMPLATES IN MICROSOFT WORD

[SOURCES]
Use the ____ to close backstage view and return to Word

(A) ** Arrow

(B) Close

(C) Options

(D) Export

EXPLANATIONS BELOW

Concept note-1: -If you want to exit Backstage, and return to the document you were working on, click the Back arrow that is at the top left of the navigation pane, or just press the Escape key on your keyboard.

Concept note-2: -The Backstage view (File tab) is the first view you will see when you start the application. It allows you to work with databases and application-level settings.

Concept note-3: -The Backstage view can be accessed by just clicking the File tab. By clicking on any tab, you can escape this view (including the File tab again). You can also use the keyboard’s ‘Esc’ button. There are three parts or panes in the backstage view.

Concept note-4: -The Backstage view. The Backstage view occupies the File tab on the ribbon and contains many commands. The Backstage view also contains other commands that apply to an entire database file. When you open Access but do not open a database (e.g., you open Access from the Windows Start menu), you see the Backstage view.

Concept note-5: -Click the File tab to access Backstage view, choose Export, then select Change File Type. Select a file type, then click Save As. The Save As dialog box will appear.
