USING MICROSOFT WORD

NEW DOCUMENTS TEMPLATES IN MICROSOFT WORD

[SOURCES]
It’s good practice to save your work and to create a document name which is relevant to the content.

(A) ** True

(B) False

EXPLANATIONS BELOW

Concept note-1: -Save your document Click FILE > Save, pick or browse to a folder, type a name for your document in the File name box, and click Save. Save your work as you go-hit Ctrl+S often.

Concept note-2: -Saving a file under a new name If you want to create an exact copy of an original document for editing or revising purposes, you should perform a Save As on the file and save it under a new name. This will guarantee that you always have a saved original copy.

Concept note-3: -It’s important to save your document whenever you start a new project or make changes to an existing one. Saving early and often can prevent your work from being lost. You’ll also need to pay close attention to where you save the document so it will be easy to find later.

Concept note-4: -Detailed Solution. The correct answer is Save As. Save and Save As are the two options for saving a file in Word.
