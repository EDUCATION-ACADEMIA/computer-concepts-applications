Word tabs, such as the FILE and HOME tabs, are organized in groups on the ____

(A) ** Ribbon

(B) title bar

(C) status bar

(D) Quick Access Toolbar

EXPLANATIONS BELOW

Concept note-1: -The ribbon is a command bar that organizes a programâ€™s features into a series of tabs at the top of the screen. Ribbon tabs are composed of groups of closely related commands, designed to help users quickly find desired commands. Each ribbon is a bar (line) across the page.

Concept note-2: -It has five groups of related commands; Clipboard, Font, Paragraph, Styles and Editing.

Concept note-3: -The Ribbon is located close to the top of the Word window. The Ribbon is organized into a set of task-oriented tabs, and each tab on the Ribbon contains groups of commands. The Home Tab contains the most frequently used commands in Word. To get to another tab on the Ribbon click that particular tab.

Concept note-4: -The File tab is a colored tab, for example, a blue tab in Word, located in the upper-left corner. When you click the File tab, you see many of the same basic commands that you saw on the File menu in earlier releases of Office, such as Open, Save, and Print.
