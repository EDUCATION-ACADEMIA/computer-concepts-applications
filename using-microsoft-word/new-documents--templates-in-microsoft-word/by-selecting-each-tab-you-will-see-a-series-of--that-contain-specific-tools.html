NEW DOCUMENTS TEMPLATES IN MICROSOFT WORD

[SOURCES]
By selecting each tab, you will see a series of ____ that contain specific tools.

(A) Commands

(B) Data

(C) ** Subgroups

(D) Options

EXPLANATIONS BELOW

Concept note-1: -Select one subset of data as you desire to include in a group and then click on the Group command in the Data tab of Excel Ribbon. To create another group with another subset of data, select the rows and then either click on the Group command or use this shortcut key Shift + Alt + Right Arrow.

Concept note-2: -On the Data tab, in the Outline group, click Subtotal. The Subtotal dialog box is displayed. In the At each change in box, click the nested subtotal column.

Concept note-3: -Select the Data tab, then click the Subtotal command. The Subtotal dialog box will appear. Click the drop-down arrow for the At each change in: field to select the column you want to subtotal.

Concept note-4: -From the Data tab, click the What-If Analysis command, then select Goal Seek from the drop-down menu. A dialog box will appear with three fields. The first field, Set cell:, will contain the desired result. In our example, cell B5 is already selected.
