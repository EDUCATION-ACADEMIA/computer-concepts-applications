Concept note-1: -The Bottom Line SaaS or Software as a Service uses cloud computing to provide users with access to a program via the Internet. Without having to install software in-house, SaaS allows each user to access programs typically through a subscription service.

Concept note-2: -With Word on your PC, Mac, or mobile device, you can: Create documents from scratch, or a template.

Concept note-3: -Word processors Word processor – a program that allows users to create, save, edit, format, print, and retrieve documents. Word processing programs can be used to create all types of text-based documents, such as: Assignments.
