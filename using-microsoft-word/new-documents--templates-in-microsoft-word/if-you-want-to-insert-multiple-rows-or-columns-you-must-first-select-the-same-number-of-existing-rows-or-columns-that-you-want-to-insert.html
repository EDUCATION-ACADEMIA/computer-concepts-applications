USING MICROSOFT WORD

NEW DOCUMENTS TEMPLATES IN MICROSOFT WORD

[SOURCES]
If you want to insert multiple rows or columns, you must first select the same number of existing rows or columns that you want to insert.

(A) ** True

(B) False

EXPLANATIONS BELOW

Concept note-1: -Tip: To insert more than one row (or column) at the same time, select as many rows or columns as you want to add before you click the insert control. For example, to insert two rows above a row, first select two rows in your table and then click Insert Above.

Concept note-2: -When inserting rows and columns, make sure you select the entire row or column by clicking the heading. If you select only a cell in the row or column, the Insert command will only insert a new cell.
