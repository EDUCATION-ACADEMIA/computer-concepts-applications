AutoCorrect fixes which of the following user mistakes?

(A) Common punctuation errors

(B) Capitalizes proper names and dates if the user does not

(C) Capitalizes the first letter in a sentence if the user does not

(D) ** All of these options

EXPLANATIONS BELOW

Concept note-1: -You can use the AutoCorrect feature to correct typos, capitalization errors, and misspelled words, as well as automatically insert symbols and other pieces of text. By default, AutoCorrect uses a standard list of typical misspellings and symbols, but you can modify the entries in this list.

Concept note-2: -Autocorrect is a word processing feature that identifies misspelled words, and uses algorithms to identify the words most likely to have been intended, and edits the text accordingly.

Concept note-3: -Go to File > Options > Proofing and select AutoCorrect Options. On the AutoCorrect tab, select or clear Replace text as you type.

Concept note-4: -Go to the AutoCorrect tab. In the Replace box, type a word or phrase that you often misspell. In the With box, type the correct spelling of the word.
