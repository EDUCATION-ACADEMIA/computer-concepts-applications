USING MICROSOFT WORD

CREATING HEADERS FOOTERS IN MICROSOFT WORD

[SOURCES]
As documents that exceed one page are typed, Word inserts ____ when it determines the text has filled one page according to paper size, margin settings, line spacing, and other settings.

(A) ** automatic page breaks / soft page breaks

(B) manual page breaks / hard page breaks

EXPLANATIONS BELOW

Concept note-1: -As you type documents that exceed one page, Word automatically inserts page breaks, called automatic page breaks or soft page breaks, when it determines the text has filled one page according to paper size, margin settings, line spacing, and other settings.

Concept note-2: -When you fill a page with text or graphics, Word inserts an automatic page break and starts a new page. To force a page break at a specific location, you can insert a manual page break. In the example below: An automatic page break is indicated by a dotted line.

Concept note-3: -The default line spacing in Word is 1.15. By default, paragraphs are followed by a blank line and headings have a space above them.

Concept note-4: -A header is text that is placed at the top of a page, while a footer is placed at the bottom, or foot, of a page.
