HOW TO REVIEW TRACK CHANGES IN MICROSOFT WORD

[SOURCES]
Which group on the Insert tab contains the commands for adding Hyperlinks & Bookmarks?

(A) Illustrations

(B) Media

(C) ** LInks

(D) Text

EXPLANATIONS BELOW

Concept note-1: -The Language & Links group has commands to set the language of the whole document or a section of text, here you can also add bookmarks and hyperlinks to the document.

Concept note-2: -On the Insert tab, click Hyperlink. The Insert Hyperlink dialog appears. 1-Text to Display: The linked text in the document. 2-ScreenTip: Text shown when you point your mouse at the linked text on the slide.

Concept note-3: -The Insert tab has seven groups of related commands; Pages, Tables, Illustrations, Links, Header & Footer, Text and Symbols.

Concept note-4: -Press Ctrl+K to open the Insert Hyperlink dialog box. The focus is on the Address field. Type the web address of the destination site, and then press Alt+T to move to the Text to display field. Type the link text you want to display in your document.
