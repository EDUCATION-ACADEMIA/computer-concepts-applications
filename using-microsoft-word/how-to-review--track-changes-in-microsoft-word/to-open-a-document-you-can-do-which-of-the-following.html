USING MICROSOFT WORD

HOW TO REVIEW TRACK CHANGES IN MICROSOFT WORD

[SOURCES]
To open a document, you can do which of the following?

(A) In the Options dialog box, select Open document.

(B) ** On the Quick Access toolbar, select the open command, and then navigate to your file.

(C) Double Click

(D) Select it

EXPLANATIONS BELOW

Concept note-1: -On the Quick Access Toolbar, select the Open command, and then navigate to your file. With Word open, on the Windows taskbar, right-click or access the context menu of Word, and then select your recent document from the list. In the Options dialog box, select Open document.

Concept note-2: -Press Shift+F10 or the Windows Menu key to open the context menu. To open the Quick Access Toolbar tab in the app options, press C.

Concept note-3: -The Quick Access Toolbar (QAT) is a Windows Office feature that contains a set of predefined or commonly used commands that may be used and generated across all environments. By default, the QAT is located in the upper left-hand corner of an Office application and may be displayed either above or below the ribbon.

Concept note-4: -If the Quick Access Toolbar is hidden, you can show it by right-clicking in the upper right of the window above the ribbon, and from the list select Show Quick Access Toolbar.
