HOW TO REVIEW TRACK CHANGES IN MICROSOFT WORD

Which of the following opens when the Envelopes icon is clicked?

(A) Envelopes menu

(B) Labels menu

(C) ** Envelopes & Labels dialog box & the Envelopes tab

(D) Envelopes & Labels dialog box & the Labels tab

EXPLANATIONS BELOW

Concept note-1: -By using the Envelopes and Labels dialog boxâ€™s Envelopes tab you can quickly create an envelope with the desired delivery and return addresses. Once you type the addresses into the appropriate areas, you can define the size of the envelope by choosing Options.

Concept note-2: -MS Word provides a feature to add an Envelope directly into the Word Document, and it is ready to get printed with a click. It also provides the Label feature, in which we can provide the address/information we want to get on the label, select its quantity, and good to go for printing on a click.

Concept note-3: -Word allows users to begin typing almost anywhere on a page using a feature called Click and Type. This means you do not have to begin typing at the beginning of the document or from existing text.

Concept note-4: -The Insert tab is used to insert or add extra features to the document, such as pictures, shapes, pages, symbols, etc. Icons and text are used to identify these options. Most of these options have a drop down menu for additional features.
