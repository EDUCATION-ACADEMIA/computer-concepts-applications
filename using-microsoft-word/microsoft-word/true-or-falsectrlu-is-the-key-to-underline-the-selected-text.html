True or False.Ctrl+U is the key to underline the selected text.

(A) ** True

(B) False

EXPLANATIONS BELOW

Concept note-1: -The quickest way to underline text is to press Ctrl+U and start typing. When you want to stop underlining, press Ctrl+U again. You can also underline text and spaces in several other ways.

Concept note-2: -Alternatively referred to as Control U and C-u, Ctrl+U is a shortcut key most often used to underline text.

Concept note-3: -Ctrl+U Underline the selected text.

Concept note-4: -Alternatively called Control+U and C-u, ^u, Ctrl+U is a keyboard shortcut often used to underline text.

Concept note-5: -The Control key is one of the combination keys on a computerâ€™s keyboard, similar to the Shift key. The user must press it down in combination with other keys. By doing so, certain commands are activated and actions are performed. The Control key thus serves as a shortcut for various commands.
