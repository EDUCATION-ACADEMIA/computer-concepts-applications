USING MICROSOFT WORD

MICROSOFT WORD

[SOURCES]
This tab has options like font color, font style, line spacing, etc.

(A) Review

(B) Insert

(C) View

(D) ** Home

EXPLANATIONS BELOW

Concept note-1: -Home tab: The Home tab is the default tab in Microsoft Word. It has five groups of related commands; Clipboard, Font, Paragraph, Styles and Editing. It helps you change document settings like font size, adding bullets, adjusting styles and many other common features.

Concept note-2: -On the Home tab, click the Font Dialog Box Launcher, and then click the Advanced tab. Note: If youâ€™re using Word 2007 the tab is called Character Spacing. In the Spacing box, click Expanded or Condensed, and then specify how much space you want in the By box.

Concept note-3: -On the Home tab, in the Font group, choose the arrow next to Font Color, and then select a color. You can also use the formatting options on the Mini toolbar to quickly format text.

Concept note-4: -The Home Tab contains the most commonly used document controls; actions in the Home Tab can change the font and size of the text, paragraph, and line spacing, copy and paste, change the organizational structure of the document. The Home Tab is grouped into four distinct areas: Clipboard, Font, Paragraph, and Styles.

Concept note-5: -Font group within Home Tab contains font size and font color buttons. In older versions of Microsoft Office tools, the font size and font color buttons were available on Formatting Toolbar.
