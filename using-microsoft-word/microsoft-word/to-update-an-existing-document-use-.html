USING MICROSOFT WORD

MICROSOFT WORD

[SOURCES]
To update an existing document use ____

(A) ** Save

(B) Save as

EXPLANATIONS BELOW

Concept note-1: -Click FILE > Save, pick or browse to a folder, type a name for your document in the File name box, and click Save. Save your work as you go-hit Ctrl+S often.

Concept note-2: -To save your file: Press CTRL+S or select File > Save.

Concept note-3: -Use Save when you’re editing an existing document and you want to preserve your changes to it as you work.

Concept note-4: -1 Use Save As to save your document for the first time. 2 Use Save to save changes to an existing document. 3 Click the disk icon in the toolbar to quickly save your changes. 4 Use Control+s or ⌘ Command+s to quickly save.
