What are the Maximum font size of MS Word?

(A) ** Maximum 1638.

(B) Maximum 1983.

(C) Maximum 1836.

(D) Maximum 1683.

EXPLANATIONS BELOW

Concept note-1: -The correct answer is 8 and 72. You can access the Fonts dialog box or use the tools in the Home tab in MS Word. The list contains font size in points 8, 9, 10, 11, 12, 14, 16, 18, 20, 22, 24, 26, 28, 36, 48 and 72. Word supports font size between 1 and 1638.

Concept note-2: -Available font sizes range from . 5 point to 999.9 points.

Concept note-3: -Detailed Solution. Key Points: The maximum font-size available in Microsoft Word 2010 from the dropdown list is 72; however, the font size can be set up to 1638 by typing the size manually for the font.

Concept note-4: -1638 is the maximum font size you can apply for any character.

Concept note-5: -Notes: The maximum font size available in MS Word from the dropdown list is 72. However, a greater size value for a font can be added manually.
