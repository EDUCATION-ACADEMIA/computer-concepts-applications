USING MICROSOFT WORD

MICROSOFT WORD

[SOURCES]
In a Microsoft Word, a file is saved with an extension

(A) .doc

(B) ** .docx

(C) .pptx

(D) .jpg

EXPLANATIONS BELOW

Concept note-1: -For example, the docx extension tells your computer that Microsoft Word can open the file and to display a Word icon when you view it in File Explorer. Most file name extensions are compatible with more than one program that you have installed.

Concept note-2: -Select the folder where you want to save your document. The dialog box will open > Select “Save as” > In the “Save as type” menu > Select the option “Word document (. docx)” > Click on the “Save as” button and a copy of your file will be saved in Docx format.
