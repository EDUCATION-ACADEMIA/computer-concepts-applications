USING MICROSOFT WORD

MICROSOFT WORD

[SOURCES]
This activity describes how to enter a text into your document. Except?

(A) Type the Enter key to insert blank line within the two paragraphs.

(B) While you are entering data using the keyboard, use the Enter key only for inserting paragraphs or to insert a new line.

(C) Use the Space key to insert spaces between two words.

(D) ** Use the Space Key only for double spaces such as separating two words.

EXPLANATIONS BELOW

Concept note-1: -You can double-space all or part of a Word document. To double-space the whole document, go to Design > Paragraph Spacing, and choose Double. Tip: To double-space only part of the document, select the paragraphs you want to change, go to Home > Line and Paragraph Spacing, and choose 2.0.

Concept note-2: -Go to Insert > Text Box. Click in your file where you’d like to insert the text box, hold your mouse button down, then drag to draw the text box the size that you want. After you’ve drawn the text box click inside it to add text.

Concept note-3: -Double space using a keyboard shortcut Press Ctrl + 2 to double space.

Concept note-4: -If you double-space something you are writing or typing, you include a full line of space between each line of writing. Double-space the entire list. double-spaced adjective.
