USING MICROSOFT WORD

MICROSOFT WORD

[SOURCES]
Suggests text insert into a document

(A) ** AutoComplete

(B) Autocorrect

(C) Horizontal ruler

(D) Zoom slider

EXPLANATIONS BELOW

Concept note-1: -Description. Autocomplete or word completion works so that when the writer writes the first letter or letters of a word, the program predicts one or more possible words as choices. If the word he intends to write is included in the list he can select it, for example by using the number keys.

Concept note-2: -Go to File > Options > Proofing, and select AutoCorrect Options. On the AutoCorrect tab, select the Replace text as you type check box, if itâ€™s not already checked. Under Replace, type the characters that you want to trigger the automatic text. The text that you selected in your document should appear under With.

Concept note-3: -Editor anticipates your next words and suggests words or phrases as you type. To accept the suggested text, press the Tab or Right-arrow key on your keyboard, and keep typing. To ignore a suggestion, just keep typing or press Esc, and the suggestion will disappear.

Concept note-4: -A text box is an object you can add to your document that lets you put and type text anywhere in your file.
