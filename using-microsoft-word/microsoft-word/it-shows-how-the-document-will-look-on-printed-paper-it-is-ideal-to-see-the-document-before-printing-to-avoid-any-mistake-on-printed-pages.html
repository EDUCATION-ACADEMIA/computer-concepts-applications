USING MICROSOFT WORD

MICROSOFT WORD

[SOURCES]
It shows how the document will look on printed paper. It is ideal to see the document before printing to avoid any mistake on printed pages.

(A) Split

(B) ** Print Preview

(C) Zoom

(D) Page Breaks

EXPLANATIONS BELOW

Concept note-1: -Print preview is a feature that displays on the screen what a hard copy would look like when printed. Using print preview, you can find any errors that may exist or fix the layout before printing, which can save ink or toner and paper by not having to print more than once.

Concept note-2: -Print layout will help to show what the document will look like when it is actually printed. In the print layout, we can see the page breaks and the other features as well as the basic paragraph formatting and the basic character.

Concept note-3: -In some programs, like Excel and Word, Print Preview appears on the main Print screen. In other programs, like Access and OneNote, you click the Print Preview button on the main Print screen to access Print Preview.

Concept note-4: -If the report is open, right-click the document tab for the report and click Print Preview. On the Print Preview tab, in the Zoom group, click Two Pages to preview your report two pages at a time. In the Zoom group, click More Pages, and then select an option.

Concept note-5: -This option is used to view the page or make adjustments before any document gets printed. By using print preview, you can discover any errors that may exist in the document or fix the layout before printing, which can save ink/toner and paper.
