How can you sort alphabetically, from A to Z, thecontents of a table column?

(A) ** Click in the column, use Ascending option inthe Sort dialog box.

(B) Select the column, use Descending option inthe Sort dialog box.

(C) Select the column and then click the Sort buttonin the paragraph group.

(D) Select the column and double click on theselection.

EXPLANATIONS BELOW

Concept note-1: -Select the list you want to sort. Go to Home > Sort. Set Sort by to Paragraphs and Text. Choose Ascending (A to Z) or Descending (Z to A).

Concept note-2: -Quick Sort With Sort Buttons Follow these steps to sort with the Quick Sort command buttons: Select one cell in the column you want to sort. On the Excel Ribbon, click the Data tab. Click Sort A to Z (smallest to largest) or Sort Z to A (largest to smallest)
