CREATING MODIFYING TABLES IN MICROSOFT WORD

[SOURCES]
Read Mode

(A) Formatting, such as a specific font color, bolding, or italics

(B) ** Changes the page layout of the document with a large font for easier reading

EXPLANATIONS BELOW

Concept note-1: -If you’re reading a document, not writing or major editing, click or tap View > Read Mode to hide the writing tools and menus and to leave more room for the pages themselves. Read Mode automatically fits the page layout to your device, using columns and larger font sizes, both of which you can adjust.

Concept note-2: -This behavior is by design. In Read mode, text is resized according to the zoom settings. However, shapes, pictures, and text boxes are resized so that you can read the text more easily. Additionally, some text wrapping options affect the layout of objects in Read mode.

Concept note-3: -The read mode also hides most of the program toolbars and menus from view, forcing you to switch to print mode if you want to edit the document. If you prefer to open documents in print layout mode instead of the read mode, you will need to change your program preferences.

Concept note-4: -Mobile view You can optimize your document’s display to make it easier to read, edit text, and add comments on your phone’s small screen. When you first open your document, it appears in Print Layout view.
