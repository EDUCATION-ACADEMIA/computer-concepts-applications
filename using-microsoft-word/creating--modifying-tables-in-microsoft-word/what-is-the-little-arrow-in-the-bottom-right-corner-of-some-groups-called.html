What is the little arrow in the bottom right corner of some groups called?

(A) ** Dialog Box Launcher

(B) Expand

(C) More

(D) Advanced Options

EXPLANATIONS BELOW

Concept note-1: -Dialog launcher is a small arrow in the lower-right corner of a group that brings up more related commands. Dialog launchers appear in groups that contain more commands than available space. Command button is the button you click to perform a particular action.

Concept note-2: –The Ribbon At the bottom right-hand corner of some groups, there is a diagonal arrow called a Dialog Box Launcher (See Figure 4). Clicking this button opens a dialog box for that group containing further option selections for the group.

Concept note-3: -A dialog box launcher is a small icon that appears in a group. Users click this icon to open related dialog boxes or task panes that provide more options that relate to the group.

Concept note-4: -Before You Begin. A Ribbon Group can show a small button called a DialogBoxLauncher on the right side of the Ribbon Group’s caption. As the name suggests, this button is primarily used to show a dialog or task pane related to the application tasks in that Ribbon Group.

Concept note-5: -Expert-Verified Answer "The given statement is true. There is a tiny arrow found in the right bottom corner of each ribbon in the toolbar. Those tiny arrows are called Dialogue Box Launcher. On clicking these launchers, we get the dialogue box of that ribbon.
