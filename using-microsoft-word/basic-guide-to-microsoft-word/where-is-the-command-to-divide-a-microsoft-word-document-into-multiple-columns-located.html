Where is the command to divide a Microsoft Word document into multiple columns located?

(A) Home ribbon, paragraph group

(B) Home ribbon, styles group

(C) Insert ribbon, text group

(D) ** Page layout ribbon, page setup group

EXPLANATIONS BELOW

Concept note-1: -To use multiple columns, go to Page Layout â†’ Page Setup â†’ Columns, and then click one of the following options: â–¸ One. Whether you know it or not, every page in Word has a column layout.

Concept note-2: -On the Page Layout tab, click Columns, then click the layout you want. To apply columns to only part of your document, with your cursor, select the text that you want to format. On the Page Layout tab, click Columns, then click More Columns.

Concept note-3: -Select Layout > Split Cells. Enter the number of columns or rows that you want to split the selected cell into, and then select OK.
