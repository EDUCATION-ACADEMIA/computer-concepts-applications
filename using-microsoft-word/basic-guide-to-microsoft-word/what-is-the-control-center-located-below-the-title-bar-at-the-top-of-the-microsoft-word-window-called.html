USING MICROSOFT WORD

BASIC GUIDE TO MICROSOFT WORD

[SOURCES]
What is the control center located below the title bar at the top of the Microsoft Word window called?

(A) Task Pane

(B) Tab

(C) ** Ribbon

(D) Groups

EXPLANATIONS BELOW

Concept note-1: -Below the upper ribbon is the lower ribbon, also known as the Toolbar. The lower ribbon changes according to which tab is selected in the top ribbon – it is directly associated with a specific tab in the top ribbon.

Concept note-2: -The Menu bar is directly below the Title bar and it displays the menu. The menu begins with the word File and continues with Edit, View, Insert, Format, Tools, Table, Window, and Help. You use the menu to give instructions to the software.

Concept note-3: -The title bar at the top of a window displays an application-defined icon and line of text.

Concept note-4: -The ribbon is a set of toolbars at the top of the window in Office programs designed to help you quickly find the commands that you need to complete a task.

Concept note-5: -The three parts of the Ribbon are tabs, groups, and commands. There are three basic components to the Ribbon. It’s good to know what each one is called so that you understand how to use it.
