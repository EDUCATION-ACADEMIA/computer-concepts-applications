APPLICATION SOFTWARE FOR PERSONAL ENTERPRISE WORKGROUP OBJECTIVES

[SOURCES]
Ciara is looking at purchasing new spreadsheet software for her business. Which of the following are compatibility issues that might affect her choice?

(A) ** It may not work with the operating systems used by employees computers.

(B) Employees may not have the experience with the new software.

(C) It may not work with and communicate with other software applications employees are using.

(D) It may not have the features & functionality her employees need.

EXPLANATIONS BELOW

Concept note-1: -Ciara is looking at purchasing new spreadsheet software for her business. Which two of the following are compatibility issues that might affect her choice? It may not work with the operating systems used by employees computers. Employees may not have the experience with the new software.

Concept note-2: -job control is not included in basic functions of operating system. we can use job scheduling, memory management and data management. operating systems help people interface with computers and provide network. job control is not a function opearting systemcapabilities.

Concept note-3: -Multitasking describes a computer operating system that is capable of running multiple applications at the same time. Multithreading describes an operating system that can support two or more CPUs.

Concept note-4: -Application software programs are created to help with a wide range of tasks.

Concept note-5: -An operating system (OS) is the program that, after being initially loaded into the computer by a boot program, manages all of the other application programs in a computer. The application programs make use of the operating system by making requests for services through a defined application program interface (API).
