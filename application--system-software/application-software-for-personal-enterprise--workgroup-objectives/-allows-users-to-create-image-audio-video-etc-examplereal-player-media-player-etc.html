APPLICATION SOFTWARE FOR PERSONAL ENTERPRISE WORKGROUP OBJECTIVES

[SOURCES]
____ :Allows users to create image, audio, video etc. Example:Real Player, Media Player etc.

(A) Presentation Graphic Software

(B) Database Software

(C) Spreadsheet Software

(D) Word Processing Software

(E) ** Multimedia Software

EXPLANATIONS BELOW

Concept note-1: -Answer: Multimedia software is tool that allows the user to create audios and graphics with the help of Media players and real players. Explanation: The definition of a multimedia software has been changing time by time.

Concept note-2: -Examples. Well known examples of media player software include Windows Media Player, VLC media player, iTunes, Winamp, IPTV Smarters, Media Player Classic, MediaMonkey, foobar2000 and AIMP.

Concept note-3: -Video-The term video refers to the moving picture, accompanied by sound such as a picture in television. Video element of multimedia application gives a lot of information in small duration of time.

Concept note-4: -Multimedia combines different media for text, sound, and images into one presentation to create a more enriched and entertaining message. Multimedia technologies include the machines and systems used to create and transmit these messages.

Concept note-5: -Multimedia software is defined as the combination of text, audio, images, animation, or video to produce interactive content. Learn about media players, file formats, and how to work with audio and video software. Updated: 09/28/2021.
