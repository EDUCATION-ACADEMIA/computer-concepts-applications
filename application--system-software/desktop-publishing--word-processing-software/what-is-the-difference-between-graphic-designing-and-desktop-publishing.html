What is the difference between Graphic designing and Desktop Publishing?

(A) Desktop Publisher involves using arrangements for visually communicating a message while Graphic Design is more production-oriented

(B) ** Graphic design involves using arrangements for visually communicating a message while desktop publishing is more production-oriented than design-oriented.

(C) One is a Editing tool to create, edit, and convert multimedia elements while the other isn’t

EXPLANATIONS BELOW

Concept note-1: -DTP mainly focuses on creating things consisting of leaflets, sales brochures, books, websites and a lot more. On the other hand, Graphic design is a name given to processes which are used for creating graphics for product packaging, website designing, branding to social media network avatars and a lot more.

Concept note-2: -DTP centers on creating things that would once have been traditionally published, including flyers, brochures, books, websites and more from a computer. Graphic design, on the other hand, is the name given to processes used to create graphics, from product packaging to website design to social network avatars and more.

Concept note-3: -It’s Graphic Design: the process of visual communication and problem-solving using typography, photography and illustration. Graphics design is the specific task under which a Graphic Artist/Designer would create supporting images, charts, tables, or graphs, but it’s not an overly common term.

Concept note-4: -Graphic designers use desktop publishing software and techniques to create the print materials they envision. The computer and desktop publishing software also aids in the creative process by allowing the designer to easily try out various page layouts, fonts, colors, and other elements.

Concept note-5: -While graphic design focuses mostly on static designs, digital design involves movement, such as animation, interactive pages, and 2D or 3D modeling. Digital designers create images and elements that will end up on a screen, whether that’s a computer screen, a phone screen, a dashboard, or any other digital formats.
