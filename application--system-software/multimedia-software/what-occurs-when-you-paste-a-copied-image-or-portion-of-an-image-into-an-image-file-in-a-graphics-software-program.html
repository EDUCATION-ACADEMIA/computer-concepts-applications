What occurs when you paste a copied image or portion of an image into an image file in a graphics software program?

(A) A copy of the image file is created.

(B) A duplicate layer is created.

(C) A new image file is opened.

(D) ** A new layer is created.

EXPLANATIONS BELOW

Concept note-1: -What occurs when you paste a copied image or portion of an image into an image file? A new layer is created.

Concept note-2: -Layers aren’t a photo editor effect, nor are they a tool, exactly. More accurately, layering images is a method. To layer images simply means to organize the arrangement of objects-photos, text, graphics, background color-that you’ve placed in a design or photo so that you can manipulate them more easily.

Concept note-3: -Clone Stamp is a tool in Photoshop that lets you copy pixels from one part of an image and transfer them onto another. It functions much as the Brush tool does, except it’s used for painting pixels. It’s a great way to remove an unwanted background object without a trace.

Concept note-4: -To copy part of an image from one area to another, use the Clone tool. Just like the other retouching tools, the Clone tool is brush-based, so you choose a clone source and then use the Clone brush to paint the source area in a new part of your image.
