Which of the following is a raster graphics format?

(A) SVG

(B) ** JPG

(C) PPI

(D) DPI

EXPLANATIONS BELOW

Concept note-1: -Popular types of raster files include JPEG, PNG, and GIF images. However, because their pixel number is fixed, raster images can become distorted or blurry when resized to fill a bigger or smaller space.

Concept note-2: -Raster File Images The smaller and closer the pixel cells, the better the quality of the image (i.e. higher resolution). Raster images are files with the following file extensions: TIFF, JPEG, CMP, BMP and some PDFs.

Concept note-3: -JPEG is a lossy raster format that stands for Joint Photographic Experts Group, the technical team that developed it. This is one of the most widely used formats online, typically for photos, email graphics and large web images like banner ads.

Concept note-4: -Popular raster file formats include GIF (graphics interchange format) and JPEG (joint photographic experts group), which are lossy formats, and BMP (Windows bitmap) and TIFF (tagged image file format), which are lossless.

Concept note-5: -A PNG is a raster-that is, pixel-based-image format. If you scale up a raster image too far, it becomes grainy and pixelated. Similarly, if the image is too small, it can become indistinct. While PNGs are capable of handling very high resolutions, theyâ€™re not infinitely expandable.
