What term is used to describe how clear or see-through an object appears?

(A) ** Transparency

(B) Background

(C) Resolution

(D) Clarity

EXPLANATIONS BELOW

Concept note-1: -In the field of optics, transparency (also called pellucidity or diaphaneity) is the physical property of allowing light to pass through the material without appreciable scattering of light.

Concept note-2: -Objects that are transparent look clear, like eye glasses or water. Light passes through transparent objects, so you can see through them. Some objects are translucent meaning they only let some light through. For example, most paper is translucent.

Concept note-3: -Transparent, translucent agree in describing material that light rays can pass through.

Concept note-4: -Material which allows light to completely pass through it without getting scattered is called transparent material. Example: clear glass. Material that does not allow light to pass through it is called opaque. Those that allow light to pass only partially are called translucent.
