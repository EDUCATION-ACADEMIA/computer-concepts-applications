Creating a folder within a folder is called?

(A) ** Nesting

(B) Copying

(C) Pasting

(D) Encrypting

EXPLANATIONS BELOW

Concept note-1: -A folder stored within another folder. Technically, the nested folder is a “subfolder, ” and subfolders can also contain subfolders and so on up to a maximum level.

Concept note-2: -Folders can also be stored within other folders, called subfolders, which in turn can store further subfolders. Windows Explorer displays your drives, folders and files in this hierarchical structure, making it easy to navigate your files.

Concept note-3: -Tip: You can also right-click any folder in the Folder Pane and click New Folder.

Concept note-4: -A folder within a folder is usually called a subfolder. You can create any number of subfolders, and each can hold any number of files and additional subfolders.

Concept note-5: -Ans-A folder is a container that holds one or more files and folder. A folder stored inside another folder is called subfolder.
