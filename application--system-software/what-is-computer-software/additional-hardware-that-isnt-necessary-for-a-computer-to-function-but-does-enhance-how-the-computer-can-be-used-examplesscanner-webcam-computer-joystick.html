APPLICATION SYSTEM SOFTWARE

WHAT IS COMPUTER SOFTWARE

[SOURCES]
Additional hardware that isn’t necessary for a computer to function, but does enhance how the computer can be used. Examples:Scanner, Webcam, Computer joystick

(A) Input device

(B) Output device

(C) ** Peripheral device

EXPLANATIONS BELOW

Concept note-1: -There are three kinds of peripherals: input, input/output, and output devices. Some common computer peripherals include keyboards, mice, tablet pens, joysticks, scanners, monitors, speakers, printers, external hard drives, and media card readers.

Concept note-2: -Connecting Peripheral Devices They can be hardwired to the computer via ports on the motherboard itself, such as a hard drive and a video card. In addition to ports on the motherboard, other ports on the computer that peripherals can connect to are USB ports, HDMI ports, DVI ports, VGA ports, and Ethernet ports.

Concept note-3: -The most common peripherals are a printer, scanner, keyboard, mouse, tape device, microphone, and external modem that are externally connected to the computer.

Concept note-4: -A peripheral device is an internal or external device that connects directly to a computer or other digital device but does not contribute to the computer’s primary function, such as computing. It helps end users access and use the functionalities of a computer.
