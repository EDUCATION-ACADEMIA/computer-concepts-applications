Concept note-2: -Without an operating system computer hardware is only an inactive electronic machine, which is inconvenient to user for execution of programs. As the computer hardware or machine understands only the machine language. It is difficult to develop each and every program in machine language in order to execute it.

Concept note-3: -Unlike operating systems, such as Windows, that are designed for single users to control one computer, network operating systems (NOS) coordinate the activities of multiple computers across a network. The network operating system acts as a director to keep the network running smoothly.
