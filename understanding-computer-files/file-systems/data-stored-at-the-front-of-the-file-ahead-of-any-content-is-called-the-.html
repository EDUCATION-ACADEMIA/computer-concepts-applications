Data stored at the front of the file ahead of any content is called the ____

(A) Prologue

(B) ** Header

(C) Extension

(D) Log

EXPLANATIONS BELOW

Concept note-1: -The metadata contained in a file header are usually stored at the start of the file, but might be present in other areas too, often including the end, depending on the file format or the type of data contained.

Concept note-2: -The types of files recognized by the system are either regular, directory, or special. However, the operating system uses many variations of these basic types. All file types recognized by the system fall into one of these categories. However, the operating system uses many variations of these basic types.
