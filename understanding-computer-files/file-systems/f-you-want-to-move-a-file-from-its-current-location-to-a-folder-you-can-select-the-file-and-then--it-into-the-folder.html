UNDERSTANDING COMPUTER FILES

FILE SYSTEMS

[SOURCES]
f you want to move a file from its current location to a folder, you can select the file and then ____ it into the folder.

(A) Cut

(B) Copy

(C) ** Drag

(D) Paste

EXPLANATIONS BELOW

Concept note-1: -You can move a file or folder from one folder to another by dragging it from its current location and dropping it into the destination folder, just as you would with a file on your desktop.

Concept note-2: -How do I Drag and Drop? By default, if you left-click and HOLD the left mouse or touchpad button while moving your mouse pointer to a different folder location on the same drive, when you release the left mouse button the file will be moved to the new location where you released the mouse button.

Concept note-3: -Use the mv command to move files and directories from one directory to another or to rename a file or directory. If you move a file or directory to a new directory without specifying a new name, it retains its original name. Attention: The mv command can overwrite many existing files unless you specify the-i flag.

Concept note-4: -Cut and paste files to move them Select the file you want to move by clicking on it once. Right-click and pick Cut, or press Ctrl + X . Navigate to another folder, where you want to move the file. Click the menu button in the toolbar and pick Paste to finish moving the file, or press Ctrl + V .
