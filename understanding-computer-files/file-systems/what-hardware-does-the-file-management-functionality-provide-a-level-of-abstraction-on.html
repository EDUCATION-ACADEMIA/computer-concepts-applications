UNDERSTANDING COMPUTER FILES

FILE SYSTEMS

[SOURCES]
What hardware does the file management functionality provide a level of abstraction on?

(A) printer

(B) ** secondary storage

(C) input/output

(D) processor

EXPLANATIONS BELOW

Concept note-1: -As (integral or separate) part of an operating system, the file system manages this information on secondary storage. Uses of secondary storage include storing various forms of programs (source, object, executable) and temporary storage of virtual memory pages (paging device or swap space).

Concept note-2: -The main activity that is performed in secondary storage management is disk scheduling.

Concept note-3: -Functions of the File Management System Store, arrange, or accessing files on a disk or other storage locations. Creating new files. Displaying the old files. Adding and editing the data in files.
