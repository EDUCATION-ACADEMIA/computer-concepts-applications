UNDERSTANDING COMPUTER FILES

FILE SYSTEMS

[SOURCES]
To properly access stored data, we need to use a Directory File at the start of the storage device. This directory file contains ____

(A) ** An index of where each file is

(B) ** Identification of each file’s type

(C) ** Metadata about each file

(D) The contents of each file

(E) a system of folders and subfolders

EXPLANATIONS BELOW

Concept note-1: -Most media file types can store metadata directly in the file itself. This is typically done in the file header, which is a block of storage that is separate from the actual media payload.

Concept note-2: -What is file metadata? A great way to think of file metadata is as extra information about a file, carried along with that file, that makes it easier to use and find. So it’s not the actual document or photo itself, it’s information about it-like the file’s name, thumbnail image, or creation date.

Concept note-3: -The metadata contained in a file header are usually stored at the start of the file, but might be present in other areas too, often including the end, depending on the file format or the type of data contained.

Concept note-4: -Folder’s Metadata Each folder has a set of standard metadata you can access and modify(if you have the proper permissions) in the Properties tab.
