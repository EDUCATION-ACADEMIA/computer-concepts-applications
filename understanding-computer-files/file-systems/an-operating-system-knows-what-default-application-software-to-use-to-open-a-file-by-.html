UNDERSTANDING COMPUTER FILES

FILE SYSTEMS

[SOURCES]
An operating system knows what default application software to use to open a file by ____

(A) examining the full contents of the file

(B) sending messages to all the installed applications to request permission

(C) ** looking at the file extension & its list of default software for each extension

EXPLANATIONS BELOW

Concept note-1: -In almost every high-level language, the function that opens a file is a wrapper around the corresponding kernel system call. It may do other fancy stuff as well, but in contemporary operating systems, opening a file must always go through the kernel.

Concept note-2: -operating systems and applications both may use file contents or file attributes (such as name) to determine how to treat a file. Microsoft Windows uses file suffix (such as .exe ) pervasively, but even for that, there are different binary formats for “.exe” files (corresponding to the resources they need).

Concept note-3: -The default file extension for all Word documents is ‘. docx’.
