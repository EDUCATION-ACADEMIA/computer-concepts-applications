UNDERSTANDING COMPUTER FILES

FILE SYSTEMS

[SOURCES]
A unique name which identifies users to Windows

(A) Password

(B) Identifier

(C) Administrative Code

(D) ** User Name

EXPLANATIONS BELOW

Concept note-1: -userPrincipalName-the logon name for the user. objectGUID-the unique identifier of a user.

Concept note-2: -The SID (Security IDentifier) is a unique ID number that a computer or domain controller uses to identify you. It is a string of alphanumeric characters assigned to each user on a Windows computer, or to each user, group, and computer on a domain-controlled network such as Indiana University’s Active Directory.

Concept note-3: -The SID is stored in a security database. The system generates the SID that identifies a particular account or group at the time the account or group is created. When a SID has been used as the unique identifier for a user or group, it can never be used again to identify another user or group.

Concept note-4: -Click on “Task Manager.” 4. In the new menu, select the “Users” tab. Your username will be listed here.

Concept note-5: -In Active Directory user properties, SAMAccountname is the “User Logon Name (Pre Windows 2000) property. UPN is the User Logon Name property. With Active Directory user source, when a user logs into ZENworks, the typed in username is passed to CASA server.
