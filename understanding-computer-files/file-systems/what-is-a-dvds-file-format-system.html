What is a DVDs file format system?

(A) FAT32

(B) ** CDFS

(C) HFS

(D) NTFS

EXPLANATIONS BELOW

Concept note-1: -CDfs is a virtual file system for Unix-like operating systems; it provides access to data and audio tracks on Compact Discs. When the CDfs driver mounts a Compact Disc, it represents each track as a file. This is consistent with the Unix convention â€œeverything is a file".

Concept note-2: -CDFS stands for Compact Disc File System.

Concept note-3: -Cdfs is a read-only file system that addresses various issues such as accessing data on disk, interacting with the cache manager, and handling various I/O operations such as opening files, performing reads on a file, retrieving information on a file, and performing various control operations on the file system.

Concept note-4: -The Compact Disc File System (CDFS) is a file system for read-only and write-once CD-ROMs developed by Simson Garfinkel and J. Spencer Love at the MIT Media Lab between 1985 and 1986. The file system provided for the creation, modification, renaming and deletion of files and directories on a write-once media.

Concept note-5: -CDF files can be opened using the Wolfram CDF Player, which is a standalone application and a Web browser plug-in provided freely by Wolfram Research. The player supports Mac, Windows, Linux, and mobile platforms.
