UNDERSTANDING COMPUTER FILES

COMPUTER FILES

[SOURCES]
This is an object on a computer that stores data and information.

(A) ** Files

(B) Folders

(C) Computer

EXPLANATIONS BELOW

Concept note-1: -The hard drive is where your software, documents, and other files are stored. The hard drive is long-term storage, which means the data is still saved even if you turn the computer off or unplug it. When you run a program or open a file, the computer copies some of the data from the hard drive onto the RAM.

Concept note-2: -disk. noun. computing a flat circular object such as a hard disk, DVD etc on which information from a computer can be stored.

Concept note-3: -CSV is one of the most common file formats for storing textual data. These files can be opened using a wide variety of programs including Notepad. The reason behind using this format over others is its ability to store complex data in a simple and readable way.

Concept note-4: -A collection of related data is called Database. A database is a data structure that stores organized information.

Concept note-5: -Object storage, also known as object-based storage, is a computer data storage architecture designed to handle large amounts of unstructured data. Unlike other architectures, it designates data as distinct units, bundled with metadata and a unique identifier that can be used to locate and access each data unit.
