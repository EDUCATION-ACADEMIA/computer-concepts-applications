What does csv stand for?

(A) common standard version

(B) comma selected value

(C) ** comma separated value

(D) common special value

EXPLANATIONS BELOW

Concept note-1: -A CSV (comma-separated values) file is a text file that has a specific format which allows data to be saved in a table structured format.

Concept note-2: -A CSV file contains a set of records separated by a carriage return/line feed (CR/LF) pair ( ˚), or by a line feed (LF) character. Each record contains a set of fields separated by a comma. If the field contains either a comma or a CR/LF, the comma must be escaped with double quotation marks as the delimiter.

Concept note-3: -A CSV is a comma-separated values file, which allows data to be saved in a tabular format. CSVs look like a garden-variety spreadsheet but with a . csv extension. CSV files can be used with most any spreadsheet program, such as Microsoft Excel or Google Spreadsheets.

Concept note-4: -A CSV file is a list of data separated by commas. For instance, it may look like the following: Name, email, phone number, address. Example, exampleexample.com, 555-555-5555, Example Address.

Concept note-5: -Comma separated values (CSV) is a term used to refer to a computer file containing tabular data that is presented in plain text. It’s organised so the data can be imported into tables at a later date, e.g. in Excel. As such it’s often considered a basic form of spreadsheet.
