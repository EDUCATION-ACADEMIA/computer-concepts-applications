FILE EXTENSIONS AND FILE TYPES

[SOURCES]
If you don’t have the program to open a file?

(A) ** The file won’t be able to open correctly

(B) The file will get damaged

(C) The file will open correctly

EXPLANATIONS BELOW

Concept note-1: -Step 1: Launch Microsoft Excel on your PC. Step 2: Head to the “files” section, and click “Export.” Step 3: In the section, select “change file type” and click on the file with the error. Step 4: Change the file extension and save the file.

Concept note-2: -The most common reason for this problem: Excel won’t open a file because the file is corrupted. This is especially common if Excel crashes while saving the file, or if a problematic macro prevents the file from being saved correctly.

Concept note-3: -"This workbook couldn’t be opened because the file format may not be matching the file extension.” This error is caused when the default saving method in Microsoft Excel is set to a format different to the default . xlsx format of the FYI for Office add-in.
